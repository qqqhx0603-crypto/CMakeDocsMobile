FindSelfPackers¶

查找 UPX，可执行文件终极压缩器

find_package(SelfPackers [...])

代码说明

  • find_package(SelfPackers [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

此模块搜索可执行文件打包工具，可以将可执行文件或共享库压缩为即时解压的自解压版本。目前它支持 UPX。

结果变量¶

此模块定义了以下变量

SelfPackers_FOUND

版本 4.2 中添加。

布尔值，指示是否找到了打包工具。

缓存变量¶

以下缓存变量也可以设置

SELF_PACKER_FOR_EXECUTABLE

用于压缩可执行文件的打包工具的路径。

SELF_PACKER_FOR_SHARED_LIB

用于压缩共享库的打包工具的路径。

SELF_PACKER_FOR_EXECUTABLE_FLAGS

压缩可执行文件时使用的命令行选项。

SELF_PACKER_FOR_SHARED_LIB_FLAGS

压缩共享库时使用的命令行选项。

示例¶

查找 UPX

find_package(SelfPackers)
