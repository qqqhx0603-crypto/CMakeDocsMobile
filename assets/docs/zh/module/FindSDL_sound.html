FindSDL_sound¶

查找 SDL_sound 库，这是一个用于 SDL（Simple DirectMedia Layer）应用程序的抽象声音文件解码器。

find_package(SDL_sound [<version>] [...])

代码说明

  • find_package(SDL_sound [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

注意

本模块专门用于 SDL_sound 1.x 版本。从 2.0.2 版本开始，SDL_sound 在使用 CMake 构建时会提供 CMake 包配置文件，应通过 find_package(SDL2_sound) 进行查找。这些较新的版本提供了 导入目标 (Imported Targets)，封装了使用需求。有关更多信息，请参考上游的 SDL_sound 文档。

注意

本模块依赖于 SDL 的查找结果，必须在 find_package(SDL) 之后调用。

根据 SDL_sound 库的构建方式，它可能需要找到额外的依赖库才能使本模块成功运行。这些依赖项可能包括 MikMod、ModPlug、Ogg、Vorbis、SMPEG、FLAC 和 Speex。

结果变量¶

此模块定义了以下变量

SDL_sound_FOUND

3.3 版本中新增。

布尔值，指示是否找到了（所请求版本的）SDL_sound 库。

SDL_sound_VERSION

版本 4.2 中添加。

包含所找到的 SDL_sound 版本的人类可读字符串。

SDL_SOUND_LIBRARIES

使用 SDL_sound 库时需要链接的库。

缓存变量¶

以下缓存变量也可以设置

SDL_SOUND_INCLUDE_DIR

包含 SDL_sound.h 及使用 SDL_sound 库所需的其他头文件的目录。

SDL_SOUND_LIBRARY

你要链接的 SDL_sound 库的文件名。请使用 SDL_SOUND_LIBRARIES 来获取链接指令，而不是使用此变量。

MIKMOD_LIBRARY

依赖库 MikMod 的路径。

MODPLUG_LIBRARY

依赖库 ModPlug (libmodplug) 的路径。

OGG_LIBRARY

依赖库 Ogg 的路径。

VORBIS_LIBRARY

依赖库 Vorbis 的路径。

SMPEG_LIBRARY

依赖库 SMPEG 的路径。

FLAC_LIBRARY

依赖库 FLAC 的路径。

SPEEX_LIBRARY

依赖库 Speex 的路径。

提示¶

此模块接受以下变量

SDLDIR

环境变量，可用于帮助定位安装在自定义位置的 SDL 库。它应指向配置、构建和安装 SDL 库时使用的安装目标：./configure --prefix=$SDLDIR。

在 macOS（苹果桌面系统） 上，设置此变量将优先选择 Framework 版本（如果找到）而非其他版本。在这种情况下，需要手动更改 SDL_LIBRARY 的缓存值以覆盖此选择，或者设置 CMAKE_INCLUDE_PATH 变量来修改搜索路径。

SDLSOUNDDIR

环境变量，其功能与 SDLDIR 相同。

SDL_SOUND_EXTRAS

这是一个可选的缓存变量，可用于添加额外的标志，这些标志会被预置到 SDL_SOUND_LIBRARIES 结果变量中。这主要用于本模块未预料到且必须添加额外标志的情况。

已弃用变量¶

以下变量提供用于向后兼容性

SDL_SOUND_FOUND

4.2 版本后已弃用：请使用 SDL_sound_FOUND，其值相同。

布尔值，指示是否找到了（所请求版本的）SDL_sound 库。

SDL_SOUND_VERSION_STRING

4.2 版本后已弃用：请使用 SDL_sound_VERSION，其值相同。

包含所找到的 SDL_sound 版本的人类可读字符串。

示例¶

查找 SDL_sound 库并创建用于将其链接到项目目标的导入接口目标

find_package(SDL)
find_package(SDL_sound)

if(SDL_sound_FOUND AND NOT TARGET SDL::SDL_sound)
  add_library(SDL::SDL_sound INTERFACE IMPORTED)
  set_target_properties(
    SDL::SDL_sound
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${SDL_SOUND_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${SDL_SOUND_LIBRARIES}"
  )

  # Append the SDL dependency as imported target to be transitively linked:
  set_property(
    TARGET SDL::SDL_sound
    APPEND
    PROPERTY INTERFACE_LINK_LIBRARIES SDL::SDL
  )
endif()

target_link_libraries(project_target PRIVATE SDL::SDL_sound)

代码说明

  • find_package(SDL) find_package(SDL_sound) if(SDL_sound_FOUND AND NOT TARGET SDL::SDL_sound) add_library(SDL::SDL_sound INTERFACE IMPORTED) set_target_properties( SDL::SDL_sound PROPERTIES INTERFACE_INCLUDE_DIRECTORIES "${SDL_SOUND_INCLUDE_DIR}" INTERFACE_LINK_LIBRARIES "${SDL_SOUND_LIBRARIES}" ) # Append the SDL dependency as imported target to be transitively linked: set_property( TARGET SDL::SDL_sound APPEND PROPERTY INTERFACE_LINK_LIBRARIES SDL::SDL ) endif() target_link_libraries(project_target PRIVATE SDL::SDL_sound)：调用 `find_package()` 命令，查找并加载外部依赖包。

在使用 SDL_sound 2.x 版本时，上游包直接提供了 SDL2_sound::SDL2_sound 导入目标。可以在不使用此模块的情况下在项目中使用它。

find_package(SDL2_sound)
target_link_libraries(project_target PRIVATE SDL2_sound::SDL2_sound)

代码说明

  • find_package(SDL2_sound) target_link_libraries(project_target PRIVATE SDL2_sound::SDL2_sound)：调用 `find_package()` 命令，查找并加载外部依赖包。
