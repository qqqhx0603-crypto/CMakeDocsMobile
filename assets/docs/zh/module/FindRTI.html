FindRTI¶

查找 HLA RTI 标准库及其包含目录

find_package(RTI [...])

代码说明

  • find_package(RTI [...])：调用 find_package() 命令，查找并加载外部依赖包。

RTI (运行时基础设施) 是 IEEE 和 SISO 标准化的仿真基础设施，在实现 HLA (高级架构) 时是必需的。它提供了一个定义良好的 C++ API，确保 M&S (建模与仿真) 应用程序与特定的 RTI 实现保持独立。

结果变量¶

此模块定义了以下变量

RTI_FOUND

布尔值，指示是否找到了 HLA RTI。

RTI_LIBRARIES

用于使用 RTI 的库。

RTI_DEFINITIONS

使用 RTI 的编译定义。默认值为 -DRTI_USES_STD_FSTREAM。

缓存变量¶

以下缓存变量也可以设置

RTI_INCLUDE_DIR

找到 RTI 包含文件的目录。

示例¶

查找 RTI 并创建导入接口目标，以便将其链接到项目目标

find_package(RTI)

if(RTI_FOUND AND NOT TARGET RTI::RTI)
  add_library(RTI::RTI INTERFACE IMPORTED)
  set_target_properties(
    RTI::RTI
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${RTI_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${RTI_LIBRARIES}"
      INTERFACE_COMPILE_DEFINITIONS "${RTI_DEFINITIONS}"
  )
endif()

target_link_libraries(example PRIVATE RTI::RTI)
