FindOpenSceneGraph¶

查找 OpenSceneGraph (OSG)，一个 3D 图形应用程序编程接口

find_package(OpenSceneGraph [<version>] [COMPONENTS <components>...] [...])

注意

OpenSceneGraph 的开发已大部分过渡到其后继项目 VulkanSceneGraph，新代码应优先使用该项目。有关将 VulkanSceneGraph 与 CMake 结合使用的指导，请参考上游文档。

该模块用于搜索 OpenSceneGraph 核心 osg 库、其依赖项 OpenThreads，以及其他 OpenSceneGraph 库（其中一些也被称为 NodeKits，如果已指定的话）。

使用 OpenSceneGraph 时，其核心库头文件应包含在 C++ 项目源代码中，例如：

example.cxx¶
#include <osg/PositionAttitudeTransform>

OpenSceneGraph 库和 NodeKits 的头文件遵循类似的包含结构，例如：

example.cxx¶
#include <osgAnimation/Animation>
#include <osgDB/DatabasePager>
#include <osgFX/BumpMapping>
// ...

组件¶

OpenSceneGraph 工具包由核心库 osg 和其他附加库组成，这些附加库可以作为组件通过 find_package() 命令进行选择性指定。

find_package(OpenSceneGraph [COMPONENTS <components>...])

支持的组件包括

osg

查找使用 OpenSceneGraph 所必需的核心 osg 库 (libosg)。该组件始终会被自动包含。

OpenThreads

通过 FindOpenThreads 模块查找依赖的 OpenThreads 库 (libOpenThreads)。该组件始终会被自动包含，因为它是使用 OpenSceneGraph 所必需的。

osgAnimation

查找 osgAnimation 库，该库提供用于动画的通用实用类。

osgDB

查找用于读写场景图支持的 osgDB 库。

osgFX

查找 osgFX NodeKit，该库提供实现特效的框架。

osgGA

查找 osgGA (GUI Abstraction) 库，该库提供与各种窗口系统交互的工具。

osgIntrospection

查找 osgIntrospection 库，该库提供反射框架，用于在运行时访问和调用类属性及方法，而无需修改类本身。

注意

自 OpenSceneGraph 3.0 版本起，osgIntrospection 库已从 OpenSceneGraph 工具包中移除。

osgManipulator

查找 osgManipulator NodeKit，该库提供对 3D 交互式操纵器的支持。

osgParticle

查找 osgParticle NodeKit，该库提供对粒子特效的支持。

osgPresentation

查找 osgPresentation NodeKit，该库提供对基于 3D 场景图的演示文稿的支持。

注意

此 NodeKit 已在 OpenSceneGraph 3.0.0 中添加。

osgProducer

查找 osgProducer 实用库，该库提供窗口管理和事件处理功能。

注意

osgProducer 已在 OpenSceneGraph 1.x 工具包的早期版本中移除，并已被 osgViewer 库取代。

osgQt

查找 osgQt 实用库，该库提供各种辅助 Qt 集成的类。

注意

自 OpenSceneGraph 3.6 版本起，该库已迁移至其独立仓库。

osgShadow

查找 osgShadow NodeKit，该库提供对多种阴影技术的支持。

osgSim

查找 osgSim NodeKit，该库增加了对导航灯和 OpenFlight 风格移动控制等模拟特性的支持。

osgTerrain

查找 osgTerrain NodeKit，该库提供地理特有地形渲染支持。

osgText

查找 osgText NodeKit，该库提供高质量文本支持。

osgUtil

查找 osgUtil 库，该库提供更新、剔除和绘制遍历等通用工具，以及诸如优化、三角形带化 (triangle stripping) 和细分 (tessellation) 等场景图工具。

osgViewer

查找 osgViewer 库，该库提供高级查看器功能。

osgVolume

查找 osgVolume NodeKit，该库提供体渲染支持。

osgWidget

查找 osgWidget NodeKit，该库提供对 2D 和 3D GUI 小部件集的支持。

如果未指定组件，该模块默认搜索 osg 和 OpenThreads 组件。

结果变量¶

此模块定义了以下变量

OpenSceneGraph_FOUND

3.3 版本中新增。

布尔值，指示是否找到了（所请求版本的）带有所有指定组件的 OpenSceneGraph。

OpenSceneGraph_VERSION

版本 4.2 中添加。

找到的 OpenSceneGraph 版本。

OPENSCENEGRAPH_INCLUDE_DIRS

包含使用 OpenSceneGraph 所需头文件的目录。

OPENSCENEGRAPH_LIBRARIES

使用 OpenSceneGraph 进行链接所需的库。

提示¶

此模块接受以下变量

OpenSceneGraph_DEBUG

将此变量设为布尔值 true 以启用该模块的调试输出。

OpenSceneGraph_MARK_AS_ADVANCED

将此变量设为布尔值 true，可自动将该模块的缓存变量标记为高级。

为了帮助此模块找到安装在自定义位置的 OpenSceneGraph 及其各种组件，可以使用 CMAKE_PREFIX_PATH 变量。此外，以下变量也会被考虑：

<COMPONENT>_DIR

可设置为 OSG 通用安装根目录的环境变量或 CMake 变量，其中 <COMPONENT> 是上述列出组件的大写形式。例如，使用 OSGVOLUME_DIR 来查找 osgVolume 组件。

OSG_DIR

可设置为影响整体 OpenSceneGraph 安装根目录检测的环境变量或 CMake 变量。

OSGDIR

环境变量，处理方式与 OSG_DIR 相同。

OSG_ROOT

环境变量，处理方式与 OSG_DIR 相同。

已弃用变量¶

以下变量提供用于向后兼容性

OPENSCENEGRAPH_FOUND

自 4.2 版本弃用：使用具有相同值的 OpenSceneGraph_FOUND。

布尔值，指示是否找到了（所请求版本的）带有所有指定组件的 OpenSceneGraph。

OPENSCENEGRAPH_VERSION

自 4.2 版本弃用：被 OpenSceneGraph_VERSION 取代。

找到的 OpenSceneGraph 版本。

示例¶

查找指定了 osgDB 和 osgUtil 组件的 OpenSceneGraph，并创建一个接口 导入目标 (imported target)，该目标封装了其链接到项目目标所需的使用要求。

find_package(OpenSceneGraph 2.0.0 REQUIRED COMPONENTS osgDB osgUtil)

if(OpenSceneGraph_FOUND AND NOT TARGET OpenSceneGraph::OpenSceneGraph)
  add_library(OpenSceneGraph::OpenSceneGraph INTERFACE IMPORTED)
  set_target_properties(
    OpenSceneGraph::OpenSceneGraph
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${OPENSCENEGRAPH_INCLUDE_DIRS}"
      INTERFACE_LINK_LIBRARIES "${OPENSCENEGRAPH_LIBRARIES}"
  )
endif()

add_executable(example example.cxx)

target_link_libraries(example PRIVATE OpenSceneGraph::OpenSceneGraph)
