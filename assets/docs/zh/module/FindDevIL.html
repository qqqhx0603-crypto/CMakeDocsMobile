FindDevIL¶

查找开发者图像库 DevIL

find_package(DevIL [<version>] [...])

代码说明

  • find_package(DevIL [<version>] [...])：调用 find_package() 命令，查找并加载外部依赖包。

4.2 版本新增：支持在 find_package() 调用中使用 <version> 参数。版本也可以指定为范围。

DevIL 软件包内部包含以下库，它们均作为同一版本发布的一部分：

  • 核心图像库 (IL)

    使用 DevIL 时必须包含此库，因为它提供了主要的图像加载和处理功能。

  • 图像库工具 (ILU)

    此库依赖于 IL，提供图像滤镜和特效。仅当应用程序使用这些扩展功能时才需要。

  • 图像库工具包 (ILUT)

    此库同时依赖于 IL 和 ILU，并额外提供了 OpenGL（开放图形库） 接口。仅当应用程序将 DevIL 与 OpenGL（开放图形库） 结合使用时才需要。

导入的目标¶

此模块提供以下 导入目标

DevIL::IL

3.21 版本新增。

封装了核心图像库 (IL) 使用需求的导入目标，在找到 DevIL 软件包时可用。

DevIL::ILU

3.21 版本新增。

封装了图像库工具 (ILU) 使用需求的导入目标，在找到 DevIL 软件包时可用。为方便起见，此目标还会链接到 DevIL::IL，因为 ILU 依赖于核心 IL 库。

DevIL::ILUT

3.21 版本新增。

封装了图像库工具包 (ILUT) 使用需求的导入目标，在找到 DevIL 软件包及其 ILUT 库时可用。此目标还会链接到 DevIL::ILU 并传递性地链接到 DevIL::IL，因为 ILUT 同时依赖于这两者。

结果变量¶

此模块定义了以下变量

DevIL_FOUND

版本 3.8 新增。

布尔值，表示是否找到了（指定版本的）DevIL 软件包，包括 IL 和 ILU 库。

DevIL_VERSION

版本 4.2 中添加。

找到的 DevIL 版本。

DevIL_ILUT_FOUND

3.21 版本新增。

布尔值，表示是否找到了 ILUT 库。在大多数系统上，当同时提供 IL 和 ILU 时，即可找到 ILUT。

缓存变量¶

以下缓存变量也可以设置

IL_INCLUDE_DIR

包含 il.h、ilu.h 和 ilut.h 头文件的目录。

IL_LIBRARIES

核心图像库 (IL) 的完整路径。

ILU_LIBRARIES

ILU 库的完整路径。

ILUT_LIBRARIES

ILUT 库的完整路径。

示例¶

查找 DevIL 软件包并链接核心图像库 (IL)

find_package(DevIL)
target_link_libraries(app PRIVATE DevIL::IL)

代码说明

  • find_package(DevIL)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(app PRIVATE DevIL::IL)：调用 target_link_libraries() 命令，为目标链接依赖库。

链接图像库工具 (ILU)

find_package(DevIL)
target_link_libraries(app PRIVATE DevIL::ILU)

代码说明

  • find_package(DevIL)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(app PRIVATE DevIL::ILU)：调用 target_link_libraries() 命令，为目标链接依赖库。

链接图像库工具包 (ILUT)

find_package(DevIL)
target_link_libraries(app PRIVATE DevIL::ILUT)

代码说明

  • find_package(DevIL)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(app PRIVATE DevIL::ILUT)：调用 target_link_libraries() 命令，为目标链接依赖库。
