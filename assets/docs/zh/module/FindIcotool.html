FindIcotool¶

查找 icotool，用于转换和创建 Win32 图标和光标文件的命令行程序

find_package(Icotool [<version>] [...])

结果变量¶

此模块定义了以下变量

Icotool_FOUND

3.3 版本中新增。

布尔值，指示是否找到（请求版本的）icotool。

Icotool_VERSION

版本 4.2 中添加。

找到的 icotool 的版本。

缓存变量¶

以下缓存变量也可以设置

ICOTOOL_EXECUTABLE

指向 icotool 工具的完整路径。

已弃用变量¶

以下变量提供用于向后兼容性

ICOTOOL_FOUND

自版本 4.2 起已弃用：使用 Icotool_FOUND，其值相同。

布尔值，指示是否找到（请求版本的）icotool。

ICOTOOL_VERSION_STRING

自版本 4.2 起已弃用：使用 Icotool_VERSION，其值相同。

找到的 icotool 的版本。

示例¶

查找 icotool 并在进程中执行它，以从当前源目录中位于的源 .png 图像创建 .ico 图标

find_package(Icotool)
if(Icotool_FOUND)
  execute_process(
    COMMAND
      ${ICOTOOL_EXECUTABLE} -c -o ${CMAKE_CURRENT_BINARY_DIR}/img.ico img.png
  )
endif()
