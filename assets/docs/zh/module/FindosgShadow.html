FindosgShadow¶

查找 OpenSceneGraph 工具包中的 osgShadow NodeKit。

注意

在大多数情况下，建议改用 FindOpenSceneGraph 模块并将 osgShadow 列为组件。这将自动处理 OpenThreads 和核心 osg 库等依赖项。

find_package(OpenSceneGraph COMPONENTS osgShadow)

代码说明

  • find_package(OpenSceneGraph COMPONENTS osgShadow)：调用 `find_package()` 命令，查找并加载外部依赖包。

此模块被 FindOpenSceneGraph 内部使用，用于查找 osgShadow NodeKit。在常规使用 find_package() 命令时，不建议直接包含此模块。不过，对于需要更细致的检测控制等高级用例，它也可以作为独立模块使用。例如，显式查找 osgShadow 或绕过自动组件检测。

find_package(osgShadow)

代码说明

  • find_package(osgShadow)：调用 `find_package()` 命令，查找并加载外部依赖包。

OpenSceneGraph 和 osgShadow 头文件旨在以下列方式包含在 C++ 项目源代码中：

example.cxx¶
#include <osg/PositionAttitudeTransform>
#include <osgShadow/ShadowTexture>
// ...

代码说明

  • #include <osg/PositionAttitudeTransform> #include <osgShadow/ShadowTexture> // ...：这一行引入头文件，使当前源文件可以使用其中声明的类型、函数或宏。

使用 OpenSceneGraph 工具包时，可能还需要其他库，例如 OpenGL（开放图形库）。

结果变量¶

此模块定义了以下变量

osgShadow_FOUND

3.3 版本中新增。

指示是否找到 OpenSceneGraph 工具包的 osgShadow NodeKit 的布尔值。

OSGSHADOW_LIBRARIES

使用 osgShadow 进行链接时所需的库。

OSGSHADOW_LIBRARY

一个结果变量，其值与 OSGSHADOW_LIBRARIES 变量相同。

缓存变量¶

以下缓存变量也可以设置

OSGSHADOW_INCLUDE_DIR

包含使用 osgShadow 所需头文件的包含目录。

OSGSHADOW_LIBRARY_DEBUG

osgShadow 调试库的路径。

提示¶

此模块接受以下变量

OSGDIR

当 OpenSceneGraph 工具包（包括其 osgShadow NodeKit）安装在自定义位置时，可以设置此环境变量以帮助定位它。它应指向配置、构建和安装时所使用的 OpenSceneGraph 安装前缀：./configure --prefix=$OSGDIR。

已弃用变量¶

以下变量提供用于向后兼容性

OSGSHADOW_FOUND

自 4.2 版本起弃用：请使用 osgShadow_FOUND，其值相同。

指示是否找到 OpenSceneGraph 工具包的 osgShadow NodeKit 的布尔值。

示例¶

使用此模块显式查找 osgShadow，并创建一个 导入目标，该目标封装了将其链接到项目目标时的使用要求。

find_package(osgShadow)

if(osgShadow_FOUND AND NOT TARGET osgShadow::osgShadow)
  add_library(osgShadow::osgShadow INTERFACE IMPORTED)
  set_target_properties(
    osgShadow::osgShadow
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${OSGSHADOW_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${OSGSHADOW_LIBRARIES}"
  )
endif()

target_link_libraries(example PRIVATE osgShadow::osgShadow)

代码说明

  • find_package(osgShadow) if(osgShadow_FOUND AND NOT TARGET osgShadow::osgShadow) add_library(osgShadow::osgShadow INTERFACE IMPORTED) set_target_properties( osgShadow::osgShadow PROPERTIES INTERFACE_INCLUDE_DIRECTORIES "${OSGSHADOW_INCLUDE_DIR}" INTERFACE_LINK_LIBRARIES "${OSGSHADOW_LIBRARIES}" ) endif() target_link_libraries(example PRIVATE osgShadow::osgShadow)：调用 `find_package()` 命令，查找并加载外部依赖包。
