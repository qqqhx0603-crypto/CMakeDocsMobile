FindBullet¶

查找 Bullet 物理引擎

find_package(Bullet [...])

代码说明

  • find_package(Bullet [...])：调用 find_package() 命令，查找并加载外部依赖包。

结果变量¶

此模块定义了以下变量

Bullet_FOUND

3.3 版本中新增。

布尔值，指示是否找到 Bullet。

BULLET_INCLUDE_DIRS

Bullet 的包含目录。

BULLET_LIBRARIES

链接到 Bullet 所需的库。默认情况下，所有 Bullet 组件（动力学、碰撞、线性代数和柔体）都会被添加。

提示¶

此模块接受以下变量

BULLET_ROOT

可以设置为 Bullet 安装路径或 Windows（微软桌面系统） 构建路径，以指定查找 Bullet 的位置。

已弃用变量¶

以下变量提供用于向后兼容性

BULLET_FOUND

自版本 4.2 起已弃用：使用 Bullet_FOUND，其值相同。

布尔值，指示是否找到 Bullet。

示例¶

查找 Bullet 并有条件地创建一个 导入目标，该目标封装了其用于链接到项目目标的使用要求

find_package(Bullet)

if(Bullet_FOUND AND NOT TARGET Bullet::Bullet)
  add_library(Bullet::Bullet INTERFACE IMPORTED)
  set_target_properties(
    Bullet::Bullet
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${BULLET_INCLUDE_DIRS}"
      INTERFACE_LINK_LIBRARIES "${BULLET_LIBRARIES}"
  )
endif()

target_link_libraries(example PRIVATE Bullet::Bullet)

代码说明

  • find_package(Bullet)：调用 find_package() 命令，查找并加载外部依赖包。

  • if(Bullet_FOUND AND NOT TARGET Bullet::Bullet)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句；条件运算符：NOT 表示逻辑非：反转后续条件的真假值；AND 表示逻辑与：两侧条件都为真时结果为真；TARGET 表示判断给定构建目标是否已定义。

  • add_library(Bullet::Bullet INTERFACE IMPORTED)：调用 add_library() 命令，定义一个库目标。

  • set_target_properties(：这一行继续补充当前命令或表达式的内容。

  • Bullet::Bullet：这一行继续补充当前命令或表达式的内容。

  • PROPERTIES：这一行继续补充当前命令或表达式的内容。

  • INTERFACE_INCLUDE_DIRECTORIES "${BULLET_INCLUDE_DIRS}"：这一行继续补充当前命令或表达式的内容；变量说明：BULLET_INCLUDE_DIRS：包含目录相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • INTERFACE_LINK_LIBRARIES "${BULLET_LIBRARIES}"：这一行继续补充当前命令或表达式的内容；变量说明：BULLET_LIBRARIES：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • )：这一行继续补充当前命令或表达式的内容。

  • endif()：这一行结束或切换上一段控制结构的分支。
