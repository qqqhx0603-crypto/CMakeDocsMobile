Use_wxWindows（微软桌面系统）¶

自版本 2.8.10 起已弃用： 此模块不应再使用。请使用 find_package(wxWidgets) 代替。

此模块充当查找 wxWidgets 库（以前称为 wxWindows（微软桌面系统））的便利包装器，并将其实用需求（例如库、包含目录和编译器标志）传播到当前目录范围，供目标使用。

在 CMake 项目中加载此模块，使用

include(Use_wxWindows)

代码说明

  • include(Use_wxWindows)：调用 `include()` 命令，加载并执行另一个 CMake 脚本。

CMakeLists.txt¶
include(Use_wxWindows)

代码说明

  • include(Use_wxWindows)：调用 `include()` 命令，加载并执行另一个 CMake 脚本。

要请求 OpenGL（开放图形库） 支持，可以在包含此模块之前设置 WXWINDOWS_USE_GL 变量

CMakeLists.txt¶
set(WXWINDOWS_USE_GL ON)
include(Use_wxWindows)

add_library(example example.cxx)

代码说明

  • set(WXWINDOWS_USE_GL ON) include(Use_wxWindows) add_library(example example.cxx)：调用 `set()` 命令，把变量 `WXWINDOWS_USE_GL` 设为给定的值。

从 CMake 3.0 开始，可以使用 FindwxWidgets 模块查找 wxWidgets，该模块从 CMake 3.27 开始使用结果变量或导入目标提供 wxWidgets 的使用需求

CMakeLists.txt¶
find_package(wxWidgets)

add_library(example example.cxx)

target_link_libraries(example PRIVATE wxWidgets::wxWidgets)

代码说明

  • find_package(wxWidgets) add_library(example example.cxx) target_link_libraries(example PRIVATE wxWidgets::wxWidgets)：调用 `find_package()` 命令，查找并加载外部依赖包。
