FindFLTK¶

查找 Fast Light Toolkit (FLTK)，这是一个用于 GUI 开发的跨平台工具包。

find_package(FLTK [...])

代码说明

  • find_package(FLTK [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

FLTK 使用基于 CMake 的构建系统，并为项目查找自身提供了软件包配置文件。从 1.4.0 版本开始，它还提供了 导入目标 (Imported Targets)，用以封装使用需求。例如 fltk::fltk，它可以链接到需要 FLTK 的项目目标中。此模块考虑了这一点，并首先尝试以“配置模式 (config mode)”查找 FLTK。如果配置文件不可用，它将回退到“模块模式 (module mode)”并搜索标准位置。有关如何将 FLTK 与 CMake 一起使用的更多信息，请参阅 FLTK 官方文档。

3.11 版本新增： 调试（Debug）和发布（Release，即优化版）库变体将被分开查找，并使用按配置 (per-configuration)变量。

结果变量¶

此模块定义了以下变量

FLTK_FOUND

布尔值，指示是否找到了 FLTK。

FLTK_LIBRARIES

使用 FLTK 时需要链接的库。

FLTK_WRAP_UI

布尔值，指示是否找到了 fluid 可执行文件。此变量仅在以“模块模式”找到 FLTK 时可用，例如，可用于在需要且可用时有条件地调用 fltk_wrap_ui() 命令。

缓存变量¶

以下缓存变量也可用于设置或使用

FLTK_FLUID_EXECUTABLE

fluid 二进制工具的路径。

FLTK_INCLUDE_DIR

使用 FLTK 所需的包含头文件的目录。

FLTK_BASE_LIBRARY_RELEASE

3.11 版本新增。

发布（优化版）FLTK 基础库的路径。

FLTK_BASE_LIBRARY_DEBUG

3.11 版本新增。

调试版 FLTK 基础库的路径。

FLTK_GL_LIBRARY_RELEASE

3.11 版本新增。

发布（优化版）FLTK GL 库的路径。

FLTK_GL_LIBRARY_DEBUG

3.11 版本新增。

调试版 FLTK GL 库的路径。

FLTK_FORMS_LIBRARY_RELEASE

3.11 版本新增。

发布（优化版）FLTK Forms 库的路径。

FLTK_FORMS_LIBRARY_DEBUG

3.11 版本新增。

调试版 FLTK Forms 库的路径。

FLTK_IMAGES_LIBRARY_RELEASE

3.11 版本新增。

发布（优化版）FLTK Images 库的路径。

FLTK_IMAGES_LIBRARY_DEBUG

3.11 版本新增。

调试版 FLTK Images 库的路径。

输入变量¶

默认情况下，此模块会搜索所有 FLTK 库及其 fluid 可执行文件。在调用 find_package(FLTK) 之前，可以设置以下变量来指定哪些元素对于成功配置是可选的：

FLTK_SKIP_FLUID

设置为布尔值 true，将 fluid 可执行文件标记为可选。

FLTK_SKIP_FORMS

设置为布尔值 true，将 FLTK Forms 库标记为可选；因此它不会被包含在 FLTK_LIBRARIES 结果变量中。

FLTK_SKIP_IMAGES

设置为布尔值 true，将 FLTK Image 库标记为可选；因此它不会被包含在 FLTK_LIBRARIES 结果变量中。

FLTK_SKIP_OPENGL

设置为布尔值 true，将 FLTK OpenGL（开放图形库） 库标记为可选；因此它不会被包含在 FLTK_LIBRARIES 结果变量中。

示例¶

查找 FLTK 并有条件地创建一个 fltk::fltk 导入接口目标（如果上游 FLTK 软件包未提供该目标）。导入目标随后可以链接到项目目标。

find_package(FLTK)

if(FLTK_FOUND AND NOT TARGET fltk::fltk)
  add_library(fltk::fltk INTERFACE IMPORTED)
  set_target_properties(
    fltk::fltk
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${FLTK_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${FLTK_LIBRARIES}"
  )
endif()

target_link_libraries(project_target PRIVATE fltk::fltk)

代码说明

  • find_package(FLTK) if(FLTK_FOUND AND NOT TARGET fltk::fltk) add_library(fltk::fltk INTERFACE IMPORTED) set_target_properties( fltk::fltk PROPERTIES INTERFACE_INCLUDE_DIRECTORIES "${FLTK_INCLUDE_DIR}" INTERFACE_LINK_LIBRARIES "${FLTK_LIBRARIES}" ) endif() target_link_libraries(project_target PRIVATE fltk::fltk)：调用 `find_package()` 命令，查找并加载外部依赖包。
