FindPackageMessage¶

此模块提供了一个用于打印查找结果消息的命令，旨在用于实现查找模块中的find_package(<PackageName>) 调用。

在 CMake 查找模块中加载此模块，使用

FindFoo.cmake¶
include(FindPackageMessage)

代码说明

  • include(FindPackageMessage)：调用 include() 命令，加载并执行另一个 CMake 脚本。

命令¶

此模块提供以下命令

find_package_message¶

为每个唯一的查找结果打印一次消息，以告知用户找到了哪个包以及在哪里找到的

find_package_message(<PackageName> <message> <details>)

代码说明

  • find_package_message(<PackageName> <message> <details>)：调用 find_package_message() 命令，具体行为由这一行传入的参数决定。

<PackageName>

包的名称（例如，在Find<PackageName>.cmake模块文件名中使用）。

<message>

要显示的消息字符串。

<details>

用于跟踪消息显示的唯一标识符。<message> 对于每个不同的<details> 值仅打印一次。如果在后续配置阶段中<details> 字符串发生变化，则消息将再次显示。

如果调用find_package() 时使用了QUIET 选项，则不会打印<message>。

示例¶

在自定义查找模块中打印结果消息

FindFoo.cmake¶
find_library(Foo_LIBRARY foo)
find_path(Foo_INCLUDE_DIR foo.h)

include(FindPackageMessage)

if(Foo_LIBRARY AND Foo_INCLUDE_DIR)
  find_package_message(
    Foo
    "Found Foo: ${Foo_LIBRARY}"
    "[${Foo_LIBRARY}][${Foo_INCLUDE_DIR}]"
  )
else()
  message(STATUS "Could NOT find Foo")
endif()

代码说明

  • find_library(Foo_LIBRARY foo)：调用 find_library() 命令，查找库文件。

  • find_path(Foo_INCLUDE_DIR foo.h)：调用 find_path() 命令，查找头文件或目录路径。

  • include(FindPackageMessage)：调用 include() 命令，加载并执行另一个 CMake 脚本。

  • if(Foo_LIBRARY AND Foo_INCLUDE_DIR)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句；条件运算符：AND 表示逻辑与：两侧条件都为真时结果为真。

  • find_package_message(：这一行继续补充当前命令或表达式的内容。

  • Foo：这一行继续补充当前命令或表达式的内容。

  • "Found Foo: ${Foo_LIBRARY}"：这一行继续补充当前命令或表达式的内容；变量说明：Foo_LIBRARY：库相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • "[${Foo_LIBRARY}][${Foo_INCLUDE_DIR}]"：这一行继续补充当前命令或表达式的内容；变量说明：Foo_LIBRARY：库相关变量，其精确取值和作用域以当前页面或脚本上下文为准；Foo_INCLUDE_DIR：包含目录、目录相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • )：这一行继续补充当前命令或表达式的内容。

  • else()：这一行结束或切换上一段控制结构的分支。

  • message(STATUS "Could NOT find Foo")：调用 message() 输出提示信息，方便观察脚本执行结果。

  • endif()：这一行结束或切换上一段控制结构的分支。

在编写标准查找模块时，请使用FindPackageHandleStandardArgs模块及其find_package_handle_standard_args()命令，该命令会根据是否找到包来自动打印查找结果消息

FindFoo.cmake¶
# ...

include(FindPackageHandleStandardArgs)

find_package_handle_standard_args(
  Foo
  REQUIRED_VARS Foo_LIBRARY Foo_INCLUDE_DIR
)

代码说明

  • # ...：这是 CMake 注释。此处省略了与当前要点无关的代码。

  • include(FindPackageHandleStandardArgs)：调用 include() 命令，加载并执行另一个 CMake 脚本。

  • find_package_handle_standard_args(：这一行继续补充当前命令或表达式的内容。

  • Foo：这一行继续补充当前命令或表达式的内容。

  • REQUIRED_VARS Foo_LIBRARY Foo_INCLUDE_DIR：这一行继续补充当前命令或表达式的内容。

  • )：这一行继续补充当前命令或表达式的内容。
