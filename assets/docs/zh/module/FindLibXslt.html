FindLibXslt¶

查找 XSL 转换，可扩展样式表语言转换 (XSLT) 库 (libxslt)

find_package(LibXslt [<version>] [...])

代码说明

  • find_package(LibXslt [<version>] [...])：调用 find_package() 命令，查找并加载外部依赖包。

导入的目标¶

在 3.18 版本中新增。

此模块提供以下 导入目标

LibXslt::LibXslt

封装 libxslt 库使用需求的 target。只有找到 libxslt 时，此 target 才可用。

LibXslt::LibExslt

封装 libexslt 库使用需求的 target。作为 libxslt 包的一部分，libexslt 在 libxslt 之上提供对 XSLT 的可选扩展。只有找到主 libxslt 库时，此 target 才可用。

LibXslt::xsltproc

封装命令行 XSLT 处理器 (xsltproc)。此工具是 libxslt 包的一部分，作为命令行替代方案将 XSLT 样式表应用于 XML 文档，而不是 libxslt 库。只有找到 xsltproc 可执行文件时，此 target 才可用。

结果变量¶

此模块定义了以下变量

LibXslt_FOUND

3.3 版本中新增。

布尔值，指示是否找到 (请求版本的) libxslt。

LibXslt_VERSION

版本 4.2 中添加。

找到的 libxslt 版本。

LIBXSLT_LIBRARIES

链接到 libxslt 所需的库。

LIBXSLT_DEFINITIONS

使用 libxslt 所需的编译器开关。

LIBXSLT_EXSLT_LIBRARIES

链接到 exslt 库时所需的库。这些仅在使用了 exslt 库时可用和需要。

缓存变量¶

以下缓存变量也可以设置

LIBXSLT_INCLUDE_DIR

包含 libxslt/xslt.h 和其他 libxslt 头文件的目录。

LIBXSLT_EXSLT_INCLUDE_DIR

在 3.18 版本中新增。

包含 libexslt/exslt.h 和其他 exslt 相关头文件的目录。这些仅在使用 exslt (XSLT 的扩展) 时需要。

LIBXSLT_XSLTPROC_EXECUTABLE

如果找到，则 XSLT 处理器可执行文件 xsltproc 的完整路径。此路径是可选的。

已弃用变量¶

以下变量提供用于向后兼容性

LIBXSLT_FOUND

自版本 4.2 起已弃用：使用 LibXslt_FOUND，其具有相同的值。

布尔值，指示是否找到 (请求版本的) libxslt。

LIBXSLT_VERSION_STRING

自版本 4.2 起已弃用：被 LibXslt_VERSION 取代。

找到的 libxslt 版本。

示例¶

查找 libxslt 库并将其链接到项目 target

find_package(LibXslt)
target_link_libraries(foo PRIVATE LibXslt::LibXslt)

代码说明

  • find_package(LibXslt)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(foo PRIVATE LibXslt::LibXslt)：调用 target_link_libraries() 命令，为目标链接依赖库。

当项目还需要 XSLT 的扩展 (exslt) 库时，应链接这两个 target

find_package(LibXslt)
target_link_libraries(foo PRIVATE LibXslt::LibXslt LibXslt::LibExslt)

代码说明

  • find_package(LibXslt)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(foo PRIVATE LibXslt::LibXslt LibXslt::LibExslt)：调用 target_link_libraries() 命令，为目标链接依赖库。

示例，如何在自定义命令构建规则中使用 XSLT 处理器

find_package(LibXslt)

if(TARGET LibXslt::xsltproc)
  # Executed when some build rule depends on example.html.
  add_custom_command(
    OUTPUT example.html
    COMMAND LibXslt::xsltproc -o example.html transform.xslt example.xml
  )
endif()

代码说明

  • find_package(LibXslt)：调用 find_package() 命令，查找并加载外部依赖包。

  • if(TARGET LibXslt::xsltproc)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句；条件运算符：TARGET 表示判断给定构建目标是否已定义。

  • # Executed when some build rule depends on example.html.：这是 CMake 注释。这是代码作者保留的说明，描述当前步骤的目的、输入输出或使用限制；英文原文保留在代码中。

  • add_custom_command(：这一行继续补充当前命令或表达式的内容。

  • OUTPUT example.html：这一行继续补充当前命令或表达式的内容。

  • COMMAND LibXslt::xsltproc -o example.html transform.xslt example.xml：这一行继续补充当前命令或表达式的内容；条件运算符：COMMAND 表示判断给定名称是否为可调用命令。

  • )：这一行继续补充当前命令或表达式的内容。

  • endif()：这一行结束或切换上一段控制结构的分支。
