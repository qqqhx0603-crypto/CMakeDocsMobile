FindGIF¶

查找图形交换格式 (GIF) 库 (giflib)

find_package(GIF [<version>] [...])

代码说明

  • find_package(GIF [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

导入的目标¶

此模块提供以下 导入目标

GIF::GIF

3.14 版新增。

一个目标，封装了使用 GIF 库的要求，在找到库时可用。

结果变量¶

此模块定义了以下变量

GIF_FOUND

一个布尔值，指示是否找到了（请求版本的）GIF 库。

GIF_VERSION

找到的 GIF 库的版本字符串（例如，5.1.4）。对于 GIF 库版本早于 4.1.6，版本字符串将仅设置为 3 或 4，因为这些版本未在其头文件中提供版本信息。

GIF_INCLUDE_DIRS

使用 GIF 库所需的包含目录。

GIF_LIBRARIES

链接到 GIF 库所需的库。

缓存变量¶

以下缓存变量也可以设置

GIF_INCLUDE_DIR

包含 gif_lib.h 和其他 GIF 库头文件的目录。

GIF_LIBRARY

GIF 库的路径。

提示¶

此模块接受以下变量

GIF_DIR

环境变量，可以设置为帮助定位安装在自定义位置的 GIF 库。它应指向配置、构建和安装 GIF 库时使用的安装目标：./configure --prefix=$GIF_DIR。

示例¶

查找 GIF 库并将其链接到项目目标

find_package(GIF)
target_link_libraries(project_target PRIVATE GIF::GIF)

代码说明

  • find_package(GIF) target_link_libraries(project_target PRIVATE GIF::GIF)：调用 `find_package()` 命令，查找并加载外部依赖包。
