FindICU¶

3.7 版本中新增。

查找国际化组件 Unicode (ICU) 库和程序

find_package(ICU [<version>] COMPONENTS <components>... [...])

代码说明

  • find_package(ICU [<version>] COMPONENTS <components>... [...])：调用 find_package() 命令，查找并加载外部依赖包。

3.11 版本新增：支持 Windows（微软桌面系统） 上的静态库。

组件¶

此模块支持组件，可以通过以下方式指定

find_package(ICU COMPONENTS <components>...)

代码说明

  • find_package(ICU COMPONENTS <components>...)：调用 find_package() 命令，查找并加载外部依赖包。

支持以下组件

data

查找 ICU 数据库。在 Windows（微软桌面系统） 上，该库组件命名为 dt，否则可使用上述任何组件名称，系统将自动选择相应的特定平台库名称。

i18n

查找 ICU 国际化库。在 Windows（微软桌面系统） 上，该库组件命名为 in，否则可使用上述任何组件名称，系统将自动选择相应的特定平台库名称。

io

查找 ICU 流与 I/O (Unicode stdio) 库。

le

查找已弃用的 ICU 布局引擎 (Layout Engine) 库，该库已在 ICU 58 版本中移除。

lx

查找用于段落布局的 ICU 布局扩展引擎 (Layout Extensions Engine) 库。

测试

查找 ICU 测试套件。

tu

查找 ICU 工具实用程序 (Tool Utility) 库。

uc

查找 ICU 基础通用库 (Common) 和数据库 (Data)。所有其他 ICU 库均依赖此库，建议在使用 ICU 组件时包含此库。

为了让该模块成功找到 ICU，至少应指定一个组件。

导入的目标¶

此模块提供以下 导入目标

ICU::<component>

封装了指定 ICU 组件使用要求的目标，仅在找到该组件时可用。<component> 应按上述列表以小写形式编写。例如，对于国际化库，使用 ICU::i18n。

结果变量¶

此模块定义了以下变量

ICU_FOUND

布尔值，指示是否找到了（所请求版本的）主要 ICU 程序和库。

ICU_VERSION

找到的 ICU 版本号。

ICU_INCLUDE_DIRS

包含 ICU 头文件的目录。

ICU_LIBRARIES

需要链接的组件库。

ICU 程序定义在以下变量中

ICU_GENCNVAL_EXECUTABLE

gencnval 可执行文件的路径。

ICU_ICUINFO_EXECUTABLE

icuinfo 可执行文件的路径。

ICU_GENBRK_EXECUTABLE

genbrk 可执行文件的路径。

ICU_ICU-CONFIG_EXECUTABLE

icu-config 可执行文件的路径。

ICU_GENRB_EXECUTABLE

genrb 可执行文件的路径。

ICU_GENDICT_EXECUTABLE

gendict 可执行文件的路径。

ICU_DERB_EXECUTABLE

derb 可执行文件的路径。

ICU_PKGDATA_EXECUTABLE

pkgdata 可执行文件的路径。

ICU_UCONV_EXECUTABLE

uconv 可执行文件的路径。

ICU_GENCFU_EXECUTABLE

gencfu 可执行文件的路径。

ICU_MAKECONV_EXECUTABLE

makeconv 可执行文件的路径。

ICU_GENNORM2_EXECUTABLE

gennorm2 可执行文件的路径。

ICU_GENCCODE_EXECUTABLE

genccode 可执行文件的路径。

ICU_GENSPREP_EXECUTABLE

gensprep 可执行文件的路径。

ICU_ICUPKG_EXECUTABLE

icupkg 可执行文件的路径。

ICU_GENCMN_EXECUTABLE

gencmn 可执行文件的路径。

ICU 组件库定义在以下变量中

ICU_<COMPONENT>_FOUND

布尔值，指示是否找到了 ICU 组件；<COMPONENT> 应以大写形式编写。

ICU_<COMPONENT>_LIBRARIES

组件库；<COMPONENT> 应以大写形式编写。

ICU 数据文件定义在以下变量中

ICU_MAKEFILE_INC

Makefile.inc 文件的路径。

ICU_PKGDATA_INC

pkgdata.inc 文件的路径。

缓存变量¶

以下缓存变量也可以设置

ICU_<PROGRAM>_EXECUTABLE

可执行文件 <PROGRAM> 的路径；<PROGRAM> 应以大写形式编写。这些变量对应于上面列出的 ICU 程序结果变量。

ICU_INCLUDE_DIR

包含 ICU 头文件的目录。

ICU_<COMPONENT>_LIBRARY

ICU 组件的库。<COMPONENT> 应以大写形式编写。

提示¶

该模块读取有关搜索结果的提示信息，来源为

ICU_ROOT

ICU 安装的根目录。也可以使用环境变量 ICU_ROOT；ICU_ROOT 变量具有最高优先级。

注意

在大多数情况下，无需设置上述任何变量，除非存在多个版本的 ICU 且需要特定版本。

示例¶

查找 ICU 组件并将其链接到项目目标

find_package(ICU COMPONENTS i18n io uc)
target_link_libraries(project_target PRIVATE ICU::i18n ICU::io ICU::uc)

代码说明

  • find_package(ICU COMPONENTS i18n io uc)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(project_target PRIVATE ICU::i18n ICU::io ICU::uc)：调用 target_link_libraries() 命令，为目标链接依赖库。
