FindProducer¶

注意

Producer（也称为 Open Producer）库起源于 OpenSceneGraph 工具包早期版本中的 osgProducer 实用程序库，后来发展成为一个独立的库。osgProducer 最终被 osgViewer 库取代，而独立的 Producer 库也已过时，不再进行维护。有关 OpenSceneGraph 使用的详细信息，请参阅 FindOpenSceneGraph 模块。

查找 Producer 库，这是一个主要为实时图形应用程序设计的窗口管理和事件处理库。

find_package(Producer [...])

代码说明

  • find_package(Producer [...])：调用 find_package() 命令，查找并加载外部依赖包。

Producer 库头文件应按如下方式包含在 C++ 项目源代码中：

example.cxx¶
#include <Producer/CameraGroup>

代码说明

  • #include <Producer/CameraGroup>：这一行引入头文件，使当前源文件可以使用其中声明的类型、函数或宏。

结果变量¶

此模块定义了以下变量

Producer_FOUND

3.3 版本中新增。

布尔值，指示是否找到了 Producer。

缓存变量¶

以下缓存变量也可以设置

PRODUCER_INCLUDE_DIR

包含使用 Producer 所需头文件的目录。

PRODUCER_LIBRARY

使用时链接 Producer 库所需的路径。

提示¶

此模块接受以下变量

PRODUCER_DIR

可以设置该环境变量以帮助定位 Producer 库的自定义安装。它应指向安装 Producer 库的根目录。这应该与配置和构建 Producer 时使用的安装前缀相匹配，例如 ./configure --prefix=$PRODUCER_DIR。

由于 Producer 在历史上与 OpenSceneGraph 紧密集成，该模块为了方便起见，也接受以下环境变量作为 PRODUCER_DIR 的等效项，以便一次性指定多个 OpenSceneGraph 相关库的公共安装根目录：

OSGDIR

环境变量，其处理方式与 PRODUCER_DIR 相同。

OSG_DIR

环境变量，其处理方式与 PRODUCER_DIR 相同。

已弃用变量¶

以下变量提供用于向后兼容性

PRODUCER_FOUND

4.2 版本弃用：请使用 Producer_FOUND，其值相同。

布尔值，指示是否找到了 Producer。

示例¶

查找 Producer 库并创建一个 导入目标 (imported target)，该目标封装了将其链接到项目目标所需的使用要求。

find_package(Producer)

if(Producer_FOUND AND NOT TARGET Producer::Producer)
  add_library(Producer::Producer INTERFACE IMPORTED)
  set_target_properties(
    Producer::Producer
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${PRODUCER_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${PRODUCER_LIBRARY}"
  )
endif()

target_link_libraries(example PRIVATE Producer::Producer)

代码说明

  • find_package(Producer)：调用 find_package() 命令，查找并加载外部依赖包。

  • if(Producer_FOUND AND NOT TARGET Producer::Producer)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句；条件运算符：NOT 表示逻辑非：反转后续条件的真假值；AND 表示逻辑与：两侧条件都为真时结果为真；TARGET 表示判断给定构建目标是否已定义。

  • add_library(Producer::Producer INTERFACE IMPORTED)：调用 add_library() 命令，定义一个库目标。

  • set_target_properties(：这一行继续补充当前命令或表达式的内容。

  • Producer::Producer：这一行继续补充当前命令或表达式的内容。

  • PROPERTIES：这一行继续补充当前命令或表达式的内容。

  • INTERFACE_INCLUDE_DIRECTORIES "${PRODUCER_INCLUDE_DIR}"：这一行继续补充当前命令或表达式的内容；变量说明：PRODUCER_INCLUDE_DIR：包含目录、目录相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • INTERFACE_LINK_LIBRARIES "${PRODUCER_LIBRARY}"：这一行继续补充当前命令或表达式的内容；变量说明：PRODUCER_LIBRARY：库相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • )：这一行继续补充当前命令或表达式的内容。

  • endif()：这一行结束或切换上一段控制结构的分支。

  • target_link_libraries(example PRIVATE Producer::Producer)：调用 target_link_libraries() 命令，为目标链接依赖库。
