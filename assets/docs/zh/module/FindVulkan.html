FindVulkan¶

3.7 版本中新增。

查找 Vulkan（一种低开销、跨平台的 3D 图形和计算 API）以及通常由 Vulkan SDK 提供的相关开发工具。

find_package(Vulkan [<version>] [COMPONENTS <components>...] [...])

代码说明

  • find_package(Vulkan [<version>] [COMPONENTS <components>...] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

Vulkan 包含常用的实用程序，例如着色器编译器和 SPIR-V 工具（如 DXC、glslc、glslang 等），这些工具支持基于 Vulkan 的开发工作流。

组件¶

在 3.24 版本中添加。

该模块支持多个可选组件，可以通过 find_package() 命令指定这些组件。

find_package(Vulkan [COMPONENTS <components>...])

代码说明

  • find_package(Vulkan [COMPONENTS <components>...])：调用 `find_package()` 命令，查找并加载外部依赖包。

每个组件都提供对应的导入目标 (imported target)。支持的组件包括：

glslc

在 3.24 版本中添加。

查找 SPIR-V 编译器。为了向后兼容，即使未明确请求，此可选组件也会自动包含。

glslangValidator

在 3.24 版本中添加。

查找用于将 GLSL 和 HLSL 着色器编译为 SPIR-V 的 glslangValidator 工具。为了向后兼容，即使未明确请求，此可选组件也会自动包含。

glslang

在 3.24 版本中添加。

查找 Khronos 参考前端着色器解析器和 SPIR-V 代码生成库（glslang）。

shaderc_combined

在 3.24 版本中添加。

查找用于 Vulkan 着色器编译的 Google 静态库。

SPIRV-Tools

在 3.24 版本中添加。

查找用于分析和转换 SPIR-V 模块的 Khronos 库。

MoltenVK

在 3.24 版本中添加。

查找 Khronos MoltenVK 库。该库在 macOS（苹果桌面系统） 上可用，并在 Apple Metal 图形框架之上实现了 Vulkan API 的一个子集。

dxc

在 3.25 版本中新增。

查找 DirectX 着色器编译器 (DXC)，包括库和命令行工具。请注意，Visual Studio（集成开发环境） 也提供 DXC 工具，但通常需要 Vulkan SDK 中包含的版本以进行 Vulkan 开发，因为它启用了 Vulkan 功能。

volk

在 3.25 版本中新增。

查找 Vulkan 元加载器 volk 库，这是一种向量优化后的内核库。

导入的目标¶

此模块提供以下 导入目标

Vulkan::Vulkan

封装了主要 Vulkan 库使用需求的目标，如果找到了 Vulkan，则可用。

Vulkan::glslc

3.19 版本新增。

封装了 GLSLC SPIR-V 编译器使用需求的导入可执行目标，如果找到了 glslc，则可用。

Vulkan::Headers

3.21 版本新增。

封装了包含 Vulkan 头文件所需的使用需求的目标。它仅提供包含目录，不链接任何库。这对于在运行时动态加载 Vulkan 库的应用程序非常有用。如果找到了 Vulkan，此目标可用。

Vulkan::glslangValidator

3.21 版本新增。

封装了 glslangValidator 使用需求的导入可执行目标，如果找到了此工具，则可用。

Vulkan::glslang

在 3.24 版本中添加。

封装了 glslang 库使用需求的目标，如果 SDK 中找到了 glslang，则可用。

Vulkan::shaderc_combined

在 3.24 版本中添加。

封装了 shaderc_combined 库使用需求的目标，如果 SDK 中找到了此库，则可用。

Vulkan::SPIRV-Tools

在 3.24 版本中添加。

封装了 SPIRV-Tools 库使用需求的目标，如果 SDK 中找到了此库，则可用。

Vulkan::MoltenVK

在 3.24 版本中添加。

封装了 MoltenVK 库使用需求的目标，如果 SDK 中找到了此库，则可用。

Vulkan::volk

在 3.25 版本中新增。

封装了 volk 库使用需求的目标，如果 SDK 中找到了 volk，则可用。

Vulkan::dxc_lib

在 3.25 版本中新增。

封装了 DirectX 着色器编译器库使用需求的目标，如果 SDK 中找到了 DXC 库，则可用。

Vulkan::dxc_exe

在 3.25 版本中新增。

提供 DirectX 着色器编译器 CLI 工具使用需求的导入可执行目标，如果 SDK 中包含此工具，则可用。

结果变量¶

此模块定义了以下变量

Vulkan_FOUND

布尔值，表示是否找到了（指定版本的）Vulkan 和所有必需的组件。

Vulkan_VERSION

在版本 3.23 中添加。

找到的 Vulkan 版本。该值从 vulkan/vulkan_core.h 中检索。

Vulkan_INCLUDE_DIRS

使用主要 Vulkan 库所需的包含目录。

Vulkan_LIBRARIES

使用主要 Vulkan 库时需要链接的库。

Vulkan_glslc_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 glslc 可执行文件。

Vulkan_glslangValidator_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 glslangValidator 可执行文件。

Vulkan_glslang_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 glslang 库。

Vulkan_shaderc_combined_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 shaderc_combined 库。

Vulkan_SPIRV-Tools_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 SPIRV-Tools 库。

Vulkan_MoltenVK_FOUND

在 3.24 版本中添加。

布尔值，表示 SDK 是否提供了 MoltenVK 库。

Vulkan_volk_FOUND

在 3.25 版本中新增。

布尔值，表示 SDK 是否提供了 volk 库。

Vulkan_dxc_lib_FOUND

在 3.25 版本中新增。

布尔值，表示 SDK 是否提供了 DirectX 着色器编译器库。

Vulkan_dxc_exe_FOUND

在 3.25 版本中新增。

布尔值，表示 SDK 是否提供了 DirectX 着色器编译器 CLI 工具。

提示¶

此模块接受以下变量

VULKAN_SDK

可选地设置此环境变量以指定特定架构的 Vulkan SDK 根目录位置。它通常通过在 Shell 环境中执行 Vulkan SDK 目录下的顶层 setup-env.sh 脚本来设置。

示例¶

查找 Vulkan 库并将其链接到项目目标

find_package(Vulkan)
target_link_libraries(project_target PRIVATE Vulkan::Vulkan)

代码说明

  • find_package(Vulkan) target_link_libraries(project_target PRIVATE Vulkan::Vulkan)：调用 `find_package()` 命令，查找并加载外部依赖包。

查找 Vulkan 库以及附加组件

find_package(Vulkan COMPONENTS volk)
target_link_libraries(project_target PRIVATE Vulkan::Vulkan Vulkan::volk)

代码说明

  • find_package(Vulkan COMPONENTS volk) target_link_libraries(project_target PRIVATE Vulkan::Vulkan Vulkan::volk)：调用 `find_package()` 命令，查找并加载外部依赖包。
