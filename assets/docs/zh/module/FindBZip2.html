FindBZip2¶

查找 BZip2 数据压缩库 (libbz2)

find_package(BZip2 [<version>] [...])

代码说明

  • find_package(BZip2 [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

导入的目标¶

此模块提供以下 导入目标

BZip2::BZip2

3.12 版本新增。

封装 BZip2 库使用需求的 target。只有在找到 BZip2 时才可用此 target。

结果变量¶

此模块定义了以下变量

BZip2_FOUND

3.3 版本中新增。

布尔值，指示是否找到了（请求版本的）BZip2 库。

BZip2_VERSION

版本 4.2 中添加。

找到的 BZip2 版本。

BZIP2_INCLUDE_DIRS

3.12 版本新增。

使用 BZip2 库所需的包含目录。

BZIP2_LIBRARIES

链接使用 BZip2 所需的库。

缓存变量¶

以下缓存变量也可以设置

BZIP2_INCLUDE_DIR

包含 BZip2 头文件的目录。

BZIP2_LIBRARY_RELEASE

用于发布配置的 BZip2 库的路径。

BZIP2_LIBRARY_DEBUG

用于调试配置的 BZip2 库的路径。

BZIP2_NEED_PREFIX

布尔值，指示 BZip2 函数是否以 BZ2_ 前缀（例如 BZ2_bzCompressInit()）。BZip2 1.0.0 之前的版本使用未加前缀的函数名（例如 bzCompressInit()）。

已弃用变量¶

以下变量提供用于向后兼容性

BZIP2_FOUND

自版本 4.2 起已弃用：使用 BZip2_FOUND，其值相同。

布尔值，指示是否找到了（请求版本的）BZip2 库。

BZIP2_VERSION_STRING

自版本 3.26 起已弃用：被 BZIP2_VERSION（和 BZip2_VERSION）取代。

找到的 BZip2 版本。

BZIP2_VERSION

3.26 版新增。

自版本 4.2 起已弃用：被 BZip2_VERSION 取代。

找到的 BZip2 版本。

示例¶

查找 BZip2 库并将其链接到项目 target

find_package(BZip2)
target_link_libraries(project_target PRIVATE BZip2::BZip2)

代码说明

  • find_package(BZip2) target_link_libraries(project_target PRIVATE BZip2::BZip2)：调用 `find_package()` 命令，查找并加载外部依赖包。
