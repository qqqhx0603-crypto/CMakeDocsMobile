FindGLUT¶

查找 OpenGL Utility Toolkit (GLUT) 库，它为创建窗口、处理输入和管理 OpenGL 应用程序中的事件提供了一个简单的 API

find_package(GLUT [...])

导入的目标¶

此模块提供以下 导入目标

GLUT::GLUT

版本 3.1 中新增。

目标封装了 GLUT 的使用要求，如果找到 GLUT 则可用。

结果变量¶

此模块定义了以下变量

GLUT_FOUND

布尔值，指示是否找到了 GLUT。

GLUT_INCLUDE_DIRS

在版本 3.23 中添加。

使用 GLUT 所需的包含目录。从 CMake 3.23 开始，此变量旨在用于目标使用要求，而不是缓存变量 GLUT_INCLUDE_DIR，后者旨在用于查找 GLUT。

GLUT_LIBRARIES

使用 GLUT 需要链接的库列表。

缓存变量¶

此模块可能会根据平台设置以下缓存变量。这些变量可以可选地设置为帮助此模块找到正确的文件，但不应将其用作结果变量

GLUT_INCLUDE_DIR

包含 GL/glut.h 的目录的完整路径（不包括 GL/）。

GLUT_glut_LIBRARY

glut 库的完整路径。

GLUT_Xi_LIBRARY

某些系统上依赖的 Xi (X Input Device Extension) 库的完整路径。

GLUT_Xmu_LIBRARY

某些系统上依赖的 Xmu (X Miscellaneous Utilities) 库的完整路径。

示例¶

查找 GLUT 并将其链接到项目目标

find_package(GLUT)
target_link_libraries(project_target PRIVATE GLUT::GLUT)
