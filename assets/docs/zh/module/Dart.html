Dart¶

自版本 3.27 起已弃用：此模块仅在策略 CMP0145 未设置为 NEW 时可用。请勿在新的代码中使用它。请改用 CTest 模块。

配置项目以使用 CTest 或旧的 Dart Tcl 客户端进行测试

此文件是 CTest 模块的向后兼容版本。它支持使用旧的 Dart 1 Tcl 客户端来驱动仪表板提交，以及使用 CTest 进行测试。此模块应包含在 CMakeLists.txt 文件的项目顶部。典型用法

include(Dart)
if(BUILD_TESTING)
  # ... testing related CMake code ...
endif()

代码说明

  • include(Dart) if(BUILD_TESTING) # ... testing related CMake code ... endif()：调用 `include()` 命令，加载并执行另一个 CMake 脚本。

BUILD_TESTING 选项由 Dart 模块创建，用于确定是否应启用测试支持。默认值为 ON。
