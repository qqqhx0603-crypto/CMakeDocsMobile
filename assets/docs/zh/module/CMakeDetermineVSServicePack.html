CMake确定VSServicePack¶

版本 4.1 中更改：只有当策略 CMP0196 未设置为 NEW 时，此模块才可用。

版本 3.0 起已弃用：不应再使用此模块。此模块的功能已被 CMAKE_<LANG>_COMPILER_VERSION 变量取代，该变量包含编译器版本号。

此模块提供了一个命令，用于确定 Visual Studio 2012 及更早版本中安装的 Visual Studio 服务包版本。

在 CMake 项目中加载此模块，使用

include(CMakeDetermineVSServicePack)

命令¶

此模块提供以下命令

DetermineVSServicePack¶

确定正在使用的 cl 编译器的 Visual Studio 服务包版本

DetermineVSServicePack(<variable>)

结果存储在指定的内部缓存变量 <variable> 中，该变量设置为以下值之一，或者如果无法确定服务包，则设置为一个空字符串

  • vc80, vc80sp1

  • vc90, vc90sp1

  • vc100, vc100sp1

  • vc110, vc110sp1, vc110sp2, vc110sp3, vc110sp4

示例¶

在项目中确定 Visual Studio 服务包版本

if(MSVC)
  include(CMakeDetermineVSServicePack)
  DetermineVSServicePack(my_service_pack)
  if(my_service_pack)
    message(STATUS "Detected: ${my_service_pack}")
  endif()
endif()
