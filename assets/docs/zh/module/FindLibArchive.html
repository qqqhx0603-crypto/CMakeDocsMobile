FindLibArchive¶

查找 libarchive 库和包含目录

find_package(LibArchive [<version>] [...])

代码说明

  • find_package(LibArchive [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

Libarchive 是一个多格式归档和压缩库。

导入的目标¶

此模块提供以下 导入目标

LibArchive::LibArchive

在 3.17 版本中添加。

一个封装 libarchive 使用需求的 target，仅在找到 libarchive 时可用。

结果变量¶

此模块定义了以下变量

LibArchive_FOUND

一个布尔值，指示是否找到（请求版本的）libarchive。

LibArchive_VERSION

找到的 libarchive 的 3 部分版本字符串（major.minor.patch）。

版本 3.6 中添加：支持新的 libarchive 版本字符串格式。从 libarchive 版本 3.2 开始，头文件中使用了不同的预处理器宏来定义版本。在 CMake 3.5 及更早版本中，此变量仅对 libarchive 版本 3.1 及更早版本进行设置。在 CMake 3.6 及更新版本中，此变量将适用于所有 libarchive 版本。

LibArchive_INCLUDE_DIRS

使用 libarchive 的包含搜索路径。

LibArchive_LIBRARIES

与 libarchive 链接的库。

示例¶

查找 libarchive 并将其链接到项目 target

find_package(LibArchive)
target_link_libraries(project_target PRIVATE LibArchive::LibArchive)

代码说明

  • find_package(LibArchive) target_link_libraries(project_target PRIVATE LibArchive::LibArchive)：调用 `find_package()` 命令，查找并加载外部依赖包。
