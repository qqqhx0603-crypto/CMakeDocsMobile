TestCXXAcceptsFlag¶

自版本 3.0 起已弃用：此模块不应再使用。它已被 CheckCXXCompilerFlag 模块取代。从 CMake 3.19 开始，CheckCompilerFlag 模块也可用于检查跨多种语言的标志。

此模块提供了一个命令来测试 C++ (CXX) 编译器是否支持特定的标志。

在 CMake 项目中加载此模块，使用

include(TestCXXAcceptsFlag)

代码说明

  • include(TestCXXAcceptsFlag)：调用 `include()` 命令，加载并执行另一个 CMake 脚本。

命令¶

此模块提供以下命令

check_cxx_accepts_flag¶

检查 CXX 编译器是否接受指定的标志

check_cxx_accepts_flag(<flags> <result-variable>)

代码说明

  • check_cxx_accepts_flag(<flags> <result-variable>)：调用 `check_cxx_accepts_flag()` 命令，具体行为由这一行传入的参数决定。

<flags>

一个或多个要测试的编译器标志。对于多个标志，请以空格分隔的字符串提供它们。

<result-variable>

存储结果的内部缓存变量的名称。如果编译器接受标志，则设置为布尔值 true，否则设置为 false。

示例¶

检查 C++ 编译器是否支持特定的标志

include(TestCXXAcceptsFlag)
check_cxx_accepts_flag("-fno-common -fstack-clash-protection" HAVE_FLAGS)

代码说明

  • include(TestCXXAcceptsFlag) check_cxx_accepts_flag("-fno-common -fstack-clash-protection" HAVE_FLAGS)：调用 `include()` 命令，加载并执行另一个 CMake 脚本。

迁移到 CheckCompilerFlag 模块

include(CheckCompilerFlag)
check_compiler_flag(CXX "-fno-common;-fstack-clash-protection" HAVE_FLAGS)

代码说明

  • include(CheckCompilerFlag) check_compiler_flag(CXX "-fno-common;-fstack-clash-protection" HAVE_FLAGS)：调用 `include()` 命令，加载并执行另一个 CMake 脚本。
