FindQt¶

自版本 3.14 起已弃用： 此模块仅在策略 CMP0084 未设置为 NEW 时可用。它仅支持 Qt3 和 Qt4。对于 Qt5 或更高版本，请参阅 cmake-qt(7)。

查找已安装的 Qt3 或 Qt4 版本

find_package(Qt [...])

代码说明

  • find_package(Qt [...])：调用 find_package() 命令，查找并加载外部依赖包。

Qt 是一个跨平台应用程序开发框架，用于创建图形用户界面和应用程序。

仅当项目可以同时使用 Qt3 和 Qt4 版本时才使用此模块。如果需要特定版本，请直接使用 FindQt4 或 FindQt3 模块。

行为

  • 如果找到多个 Qt 版本，用户必须使用变量 DESIRED_QT_VERSION 设置首选的 Qt 主版本。

  • 如果只找到一个 Qt 版本，则会自动设置 DESIRED_QT_VERSION。

  • 设置 DESIRED_QT_VERSION 变量后，将包含相应的 FindQt3 或 FindQt4 模块。

结果变量¶

此模块定义了以下变量

QT4_INSTALLED

TRUE 如果找到 Qt4。

QT3_INSTALLED

TRUE 如果找到 Qt3。

提示¶

QT_REQUIRED

如果此变量在调用 find_package(Qt) 之前设置为 TRUE，CMake 将在未找到 Qt3 或 Qt4 的情况下引发错误。

DESIRED_QT_VERSION

指定要使用的 Qt 主版本。可以是 3、4 或为空，以自动搜索版本。
