FindLTTngUST¶

3.6 版本新增。

查找 LTTng (Linux Trace Toolkit: next generation) 用户空间跟踪库 (LTTng-UST)

find_package(LTTngUST [<version>] [...])

代码说明

  • find_package(LTTngUST [<version>] [...])：调用 find_package() 命令，查找并加载外部依赖包。

导入的目标¶

此模块提供以下 导入目标

LTTng::UST

提供 LTTng-UST 库使用需求的 target。只有在找到 LTTng-UST 时才可用。

结果变量¶

此模块定义了以下变量

LTTngUST_FOUND

布尔值，指示是否找到了（请求版本的）LTTng-UST 库。

LTTngUST_VERSION

版本 4.2 中添加。

LTTng-UST 版本。

LTTNGUST_HAS_TRACEF

TRUE 如果系统中的 LTTng-UST 提供了 tracef() API。

LTTNGUST_HAS_TRACELOG

TRUE 如果系统中的 LTTng-UST 提供了 tracelog() API。

缓存变量¶

以下缓存变量也可以设置

LTTNGUST_INCLUDE_DIRS

LTTng-UST 的包含目录。

LTTNGUST_LIBRARIES

使用 LTTng-UST 所需的库。

已弃用变量¶

以下变量提供用于向后兼容性

LTTNGUST_FOUND

自版本 4.2 起已弃用：使用 LTTngUST_FOUND，其值相同。

布尔值，指示是否找到了（请求版本的）LTTng-UST 库。

LTTNGUST_VERSION_STRING

自版本 4.2 起已弃用：被 LTTngUST_VERSION 取代。

LTTng-UST 版本。

示例¶

查找 LTTng-UST 库并将其链接到项目 target

find_package(LTTugNST)
target_link_libraries(project_target PRIVATE LTTng::UST)

代码说明

  • find_package(LTTugNST)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(project_target PRIVATE LTTng::UST)：调用 target_link_libraries() 命令，为目标链接依赖库。
