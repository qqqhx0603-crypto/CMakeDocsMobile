FindSDL_net¶

查找 SDL_net 库，这是一个跨平台的网络库，用于与 SDL（简单直接媒体层）应用程序一起使用

find_package(SDL_net [<version>] [...])

代码说明

  • find_package(SDL_net [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

注意

此模块专门用于 SDL_net 版本 1。从版本 2.1 开始，SDL_net 在使用 CMake 构建时提供 CMake 包配置文件，应使用 find_package(SDL2_net) 找到它。这些较新的版本提供 导入目标，封装了使用要求。有关更多信息，请参阅官方 SDL 文档。

结果变量¶

此模块定义了以下变量

SDL_net_FOUND

3.3 版本中新增。

布尔值，指示是否找到了（请求版本的）SDL_net 库。

SDL_net_VERSION

版本 4.2 中添加。

包含找到的 SDL_net 版本的易读字符串。

SDL_NET_INCLUDE_DIRS

包含使用 SDL_net 库所需的头文件的包含目录。

SDL_NET_LIBRARIES

需要链接以使用 SDL_net 库的库。

提示¶

此模块接受以下变量

SDLDIR

环境变量，可用于帮助定位安装在自定义位置的 SDL 库。它应指向配置、构建和安装 SDL 库时使用的安装目标：./configure --prefix=$SDLDIR。

已弃用变量¶

以下变量提供用于向后兼容性

SDL_NET_VERSION_STRING

自版本 4.2 起已弃用：使用 SDL_net_VERSION，它具有相同的值。

包含找到的 SDL_net 版本的易读字符串。

SDL_NET_FOUND

自版本 4.2 起已弃用：使用 SDL_net_FOUND，它具有相同的值。

SDLNET_FOUND

自版本 2.8.10 起已弃用：使用 SDL_net_FOUND，它具有相同的值。

SDLNET_INCLUDE_DIR

自版本 2.8.10 起已弃用：使用 SDL_NET_INCLUDE_DIRS，它具有相同的值。

SDLNET_LIBRARY

自版本 2.8.10 起已弃用：使用 SDL_NET_LIBRARIES，它具有相同的值。

示例¶

查找 SDL_net 库并创建一个导入接口目标，以便将其链接到项目目标

find_package(SDL_net)

if(SDL_net_FOUND AND NOT TARGET SDL::SDL_net)
  add_library(SDL::SDL_net INTERFACE IMPORTED)
  set_target_properties(
    SDL::SDL_net
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${SDL_NET_INCLUDE_DIRS}"
      INTERFACE_LINK_LIBRARIES "${SDL_NET_LIBRARIES}"
  )
endif()

target_link_libraries(project_target PRIVATE SDL::SDL_net)

代码说明

  • find_package(SDL_net) if(SDL_net_FOUND AND NOT TARGET SDL::SDL_net) add_library(SDL::SDL_net INTERFACE IMPORTED) set_target_properties( SDL::SDL_net PROPERTIES INTERFACE_INCLUDE_DIRECTORIES "${SDL_NET_INCLUDE_DIRS}" INTERFACE_LINK_LIBRARIES "${SDL_NET_LIBRARIES}" ) endif() target_link_libraries(project_target PRIVATE SDL::SDL_net)：调用 `find_package()` 命令，查找并加载外部依赖包。

在使用 SDL_net 版本 2 时，上游包直接提供 SDL2_net::SDL2_net 导入目标。可以在不使用此模块的情况下在项目中直接使用它

find_package(SDL2_net)
target_link_libraries(project_target PRIVATE SDL2_net::SDL2_net)

代码说明

  • find_package(SDL2_net) target_link_libraries(project_target PRIVATE SDL2_net::SDL2_net)：调用 `find_package()` 命令，查找并加载外部依赖包。
