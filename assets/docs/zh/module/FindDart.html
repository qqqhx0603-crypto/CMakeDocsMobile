FindDart¶

自版本 3.27 起已弃用：此模块仅在策略 CMP0145 未设置为 NEW 时可用。

查找 DART

find_package(DART [...])

代码说明

  • find_package(DART [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

此模块查找 dart 测试软件，并将 DART_ROOT 设置为指向找到它的位置。

结果变量¶

此模块定义了以下变量

Dart_FOUND

3.3 版本中新增。

布尔值，指示是否找到了 DART。

缓存变量¶

以下缓存变量也可以设置

DART_ROOT

Dart 根安装路径。
