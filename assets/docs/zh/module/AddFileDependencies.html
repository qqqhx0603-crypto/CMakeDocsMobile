AddFileDependencies¶

自版本 3.20 起已弃用： 请勿在新的代码中使用此模块。

请改用 set_property() 命令直接附加到 OBJECT_DEPENDS 源文件属性

set_property(SOURCE <source> APPEND PROPERTY OBJECT_DEPENDS <files>...)

代码说明

  • set_property(SOURCE <source> APPEND PROPERTY OBJECT_DEPENDS <files>...)：调用 set_property() 命令，设置目标、目录、源文件等对象的属性。

在 CMake 项目中加载此模块，使用

include(AddFileDependencies)

代码说明

  • include(AddFileDependencies)：调用 include() 命令，加载并执行另一个 CMake 脚本。

命令¶

此模块提供以下命令

add_file_dependencies¶

为源文件添加依赖项

add_file_dependencies(<source> <files>...)

代码说明

  • add_file_dependencies(<source> <files>...)：调用 add_file_dependencies() 命令，具体行为由这一行传入的参数决定。

此命令将给定的 <files> 添加到文件 <source> 的依赖项中。
