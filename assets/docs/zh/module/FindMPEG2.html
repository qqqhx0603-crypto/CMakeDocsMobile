FindMPEG2¶

查找原生 MPEG2 库 (libmpeg2)

find_package(MPEG2 [...])

注意

根据原生 libmpeg2 库的构建和安装方式，它可能依赖于 SDL (Simple DirectMedia Layer) 库。如果找到 SDL，此模块会在使用时将其包含在其使用需求中。

结果变量¶

此模块定义了以下变量

MPEG2_FOUND

布尔值，指示是否找到了 libmpeg2 库。

MPEG2_LIBRARIES

使用 libmpeg2 时需要链接的库。

缓存变量¶

以下缓存变量也可以设置

MPEG2_INCLUDE_DIR

包含 mpeg2.h 和相关头文件以使用 libmpeg2 库的目录。

MPEG2_mpeg2_LIBRARY

libmpeg2 库的路径。

MPEG2_vo_LIBRARY

vo (Video Out) 库的路径。

示例¶

查找 libmpeg2 库并创建一个导入接口目标，以便将其链接到项目目标

find_package(MPEG2)

if(MPEG2_FOUND AND NOT TARGET MPEG2::MPEG2)
  add_library(MPEG2::MPEG2 INTERFACE IMPORTED)
  set_target_properties(
    MPEG2::MPEG2
    PROPERTIES
      INTERFACE_LINK_LIBRARIES "${MPEG2_LIBRARIES}"
      INTERFACE_INCLUDE_DIRECTORIES "${MPEG2_INCLUDE_DIR}"
  )
endif()

target_link_libraries(project_target PRIVATE MPEG2::MPEG2)
