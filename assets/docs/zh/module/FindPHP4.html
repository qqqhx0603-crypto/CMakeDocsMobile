FindPHP4¶

查找 PHP 4 版本，一种通用的脚本语言

find_package(PHP4 [...])

注意

此模块专门用于 PHP 4 版本，该版本已过时且不再受支持。对于现代开发，请使用较新的 PHP 版本。

此模块检查是否已安装 PHP 4，并确定包含目录和 PHP 命令行解释器的位置。

结果变量¶

此模块定义了以下变量

PHP4_FOUND

布尔值，指示是否找到了 PHP 4。

缓存变量¶

以下缓存变量也可以设置

PHP4_INCLUDE_PATH

包含 php.h 和使用 PHP 所需的其他头文件的目录。

PHP4_EXECUTABLE

指向 php 命令行解释器可执行文件的完整路径。

示例¶

查找 PHP

find_package(PHP4)
