FindQt3¶

查找 Qt3，一个跨平台应用程序开发框架，用于创建图形用户界面和应用程序

find_package(Qt3 [<version>] [...])

代码说明

  • find_package(Qt3 [<version>] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

注意

此模块用于 Qt 版本 3。从 Qt 版本 5 开始，Qt 上游也提供了一个导出的配置来查找 Qt。新代码应该遵循 cmake-qt(7)，而不是使用此模块。

结果变量¶

此模块定义了以下变量

Qt3_FOUND

3.3 版本中新增。

布尔值，指示是否找到（请求版本的）Qt3。

QT_FOUND

与 Qt3_FOUND 相同。布尔值，指示是否找到（请求版本的）Qt3。此变量是为了与其他 Qt 查找模块兼容而提供的。

Qt3_VERSION

版本 4.2 中添加。

找到的 Qt3 版本。

QT_LIBRARIES

使用 Qt3 时需要链接的库。

QT_DEFINITIONS

使用 Qt3 编译代码时要使用的编译定义列表。

缓存变量¶

以下缓存变量也可以设置

QT_INCLUDE_DIR

包含 qt.h 和其他 Qt3 头文件的目录。

以下缓存变量也可能被设置，但并非供通用使用

QT_MOC_EXECUTABLE

指向 moc 工具的路径。

QT_UIC_EXECUTABLE

指向 uic 工具的路径。

QT_QT_LIBRARY

指向 Qt3 库的路径。

QT_QTMAIN_LIBRARY

指向 qtmain 库的路径。这仅在 Windows（微软桌面系统） 上需要 Qt3。

提示¶

QT_MT_REQUIRED

要搜索 Qt3 的多线程版本，请在查找 Qt3 之前将此变量设置为 TRUE。

已弃用变量¶

以下变量提供用于向后兼容性

QT_VERSION_STRING

自版本 4.2 起已弃用：使用 Qt3_VERSION，它具有相同的值。

找到的 Qt3 版本。

示例¶

在系统上查找 Qt3

find_package(Qt3)
if(Qt3_FOUND)
  target_link_libraries(foo PRIVATE ${QT_LIBRARIES})
  target_include_directories(foo PRIVATE ${QT_INCLUDE_DIR})
  target_compile_definitions(foo PRIVATE ${QT_DEFINITIONS})
endif()

代码说明

  • find_package(Qt3) if(Qt3_FOUND) target_link_libraries(foo PRIVATE ${QT_LIBRARIES}) target_include_directories(foo PRIVATE ${QT_INCLUDE_DIR}) target_compile_definitions(foo PRIVATE ${QT_DEFINITIONS}) endif()：调用 `find_package()` 命令，查找并加载外部依赖包。

正在查找 Qt3 的多线程版本

set(QT_MT_REQUIRED TRUE)
find_package(Qt3)

代码说明

  • set(QT_MT_REQUIRED TRUE) find_package(Qt3)：调用 `set()` 命令，把变量 `QT_MT_REQUIRED` 设为给定的值。
