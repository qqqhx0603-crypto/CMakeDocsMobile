FindHDF5¶

查找分层数据格式（HDF5），这是一个用于读写自描述阵列数据的库。

find_package(HDF5 [<version>] [COMPONENTS <components>...] [...])

代码说明

  • find_package(HDF5 [<version>] [COMPONENTS <components>...] [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

如果 HDF5 库是使用其基于 CMake 的构建系统构建的，从 HDF5 1.8.15 版本开始，它将提供自己的 CMake 软件包配置文件（hdf5-config.cmake），以便与 find_package() 命令的配置模式（config mode）配合使用。默认情况下，该模块会搜索此文件，如果找到，则根据找到的配置返回结果。

如果未找到上游配置文件，该模块将回退到模块模式（module mode），并调用通常随 HDF5 库一起安装的 HDF5 包装编译器。根据配置的不同，此包装编译器被命名为 h5cc（串行）或 h5pcc（并行）。如果找到，将通过 -show 参数查询该包装器，以确定构建 HDF5 客户端应用程序所需的编译器和链接器标志。串行版和并行版的 HDF5 包装器都会被考虑。优先使用包含其中任意一个的第一个目录。如果同一目录下同时找到两个版本，默认优先使用串行版本。要更改此行为，请将变量 HDF5_PREFER_PARALLEL 设置为 TRUE。

除了查找编译 HDF5 应用程序所需的包含目录和库之外，该模块还尝试查找 HDF5 发行版提供的其他工具，这些工具对于回归测试或开发工作流程非常有用。

组件¶

该模块支持可选组件，可以通过 find_package() 命令指定：

find_package(HDF5 [COMPONENTS <components>...])

代码说明

  • find_package(HDF5 [COMPONENTS <components>...])：调用 `find_package()` 命令，查找并加载外部依赖包。

支持的组件包括

C

查找 HDF5 C 库（C 绑定）。

CXX

查找 HDF5 C++ 库（C++ 绑定）。

Fortran

查找 HDF5 Fortran 库（Fortran 绑定）。

HL

此组件可与其他组件结合使用，以查找提供高级函数（high-level functions）的 C、CXX 和/或 Fortran 版高级（HL）HDF5 库变体。

如果未指定任何组件，该模块默认将搜索 C 组件。

导入的目标¶

此模块提供以下 导入目标

HDF5::HDF5

3.19 版本新增。

封装了所有已找到的 HDF5 绑定库的使用需求的目标（HDF5_LIBRARIES），如果找到 HDF5 及所有必需组件，则可用。

hdf5::hdf5

3.19 版本新增。

封装了 HDF5 C 库使用需求的目标，如果找到 HDF5 库及其 C 组件，则可用。

hdf5::hdf5_cpp

3.19 版本新增。

封装了 HDF5 C 和 C++ 库使用需求的目标，如果找到 HDF5 库及其 C 和 CXX 组件，则可用。

hdf5::hdf5_fortran

3.19 版本新增。

封装了 HDF5 Fortran 库使用需求的目标，如果找到 HDF5 库及其 Fortran 组件，则可用。

hdf5::hdf5_hl

3.19 版本新增。

封装了 HDF5 高级 C 库使用需求的目标，如果找到 HDF5 库及其 C 和 HL 组件，则可用。

hdf5::hdf5_hl_cpp

3.19 版本新增。

高级 C++ 库。

封装了 HDF5 高级 C 和高级 C++ 库使用需求的目标，如果找到 HDF5 库及其 C、CXX 和 HL 组件，则可用。

hdf5::hdf5_hl_fortran

3.19 版本新增。

封装了 HDF5 高级 Fortran 库使用需求的目标，如果找到 HDF5 库及其 Fortran 和 HL 组件，则可用。

hdf5::h5diff

3.19 版本新增。

封装了 h5diff 可执行文件使用需求的已导入可执行目标，如果找到 h5diff，则可用。

结果变量¶

此模块定义了以下变量

HDF5_FOUND

布尔值，指示是否找到了（所请求版本的）HDF5。

HDF5_VERSION

3.3 版本中新增。

找到的 HDF5 库的版本。

HDF5_INCLUDE_DIRS

包含使用 HDF5 所需头文件的目录。

HDF5_DEFINITIONS

使用 HDF5 所需的编译器定义。

HDF5_LIBRARIES

使用 HDF5 时需要链接的所有已请求绑定的库。

HDF5_HL_LIBRARIES

如果启用了 HL 组件，则为所有绑定提供 HDF5 高级 API 所需的库。

HDF5_IS_PARALLEL

布尔值，指示 HDF5 库是否具有并行 IO 支持。

对于每个已启用的语言绑定组件，将定义相应的 HDF5_<LANG>_LIBRARIES 变量，以及可能的 HDF5_<LANG>_DEFINITIONS。如果启用了 HL 组件，还将定义 HDF5_<LANG>_HL_LIBRARIES 变量。

HDF5_C_DEFINITIONS

HDF5 C 绑定所需的编译器定义。

HDF5_CXX_DEFINITIONS

HDF5 C++ 绑定所需的编译器定义。

HDF5_Fortran_DEFINITIONS

HDF5 Fortran 绑定所需的编译器定义。

HDF5_C_INCLUDE_DIRS

HDF5 C 绑定所需的包含目录。

HDF5_CXX_INCLUDE_DIRS

HDF5 C++ 绑定所需的包含目录。

HDF5_Fortran_INCLUDE_DIRS

HDF5 Fortran 绑定所需的包含目录。

HDF5_C_LIBRARIES

HDF5 C 绑定所需的库。

HDF5_CXX_LIBRARIES

HDF5 C++ 绑定所需的库。

HDF5_Fortran_LIBRARIES

HDF5 Fortran 绑定所需的库。

HDF5_C_HL_LIBRARIES

如果启用了 HL 组件，则为高级 C 绑定所需的库。

HDF5_CXX_HL_LIBRARIES

如果启用了 HL 组件，则为高级 C++ 绑定所需的库。

HDF5_Fortran_HL_LIBRARIES

如果启用了 HL 组件，则为高级 Fortran 绑定所需的库。

缓存变量¶

以下缓存变量也可以设置

HDF5_C_COMPILER_EXECUTABLE

HDF5 C 包装编译器的路径。

HDF5_CXX_COMPILER_EXECUTABLE

HDF5 C++ 包装编译器的路径。

HDF5_Fortran_COMPILER_EXECUTABLE

HDF5 Fortran 包装编译器的路径。

HDF5_C_COMPILER_EXECUTABLE_NO_INTERROGATE

3.6 版本新增。

主 C 编译器的路径，该编译器同时也是 HDF5 包装器。此变量仅在模块模式下使用。

HDF5_CXX_COMPILER_EXECUTABLE_NO_INTERROGATE

3.6 版本新增。

主 C++ 编译器的路径，该编译器同时也是 HDF5 包装器。此变量仅在模块模式下使用。

HDF5_Fortran_COMPILER_EXECUTABLE_NO_INTERROGATE

3.6 版本新增。

主 Fortran 编译器的路径，该编译器同时也是 HDF5 包装器。此变量仅在模块模式下使用。

HDF5_DIFF_EXECUTABLE

HDF5 数据集比较工具（h5diff）的路径。

提示¶

在调用 find_package(HDF5) 之前，可以设置以下变量来引导 HDF5 库的搜索：

HDF5_PREFER_PARALLEL

3.4 版本新增。

将其设置为布尔真（true）以优先使用并行 HDF5（默认优先使用串行）。此变量仅在模块模式下使用。

HDF5_FIND_DEBUG

版本 3.9 中添加。

将其设置为布尔真（true）以获取该模块的额外调试输出。

HDF5_NO_FIND_PACKAGE_CONFIG_FILE

版本 3.8 新增。

将其设置为布尔真（true）以跳过查找和使用 CMake 软件包配置文件（hdf5-config.cmake）。

HDF5_USE_STATIC_LIBRARIES

设置为布尔值，以决定是否为 HDF5 及其所有依赖项优先使用静态链接而非动态链接。

版本 3.10 新增: 支持 Windows（微软桌面系统） 上的 HDF5_USE_STATIC_LIBRARIES。

示例¶

示例：查找 HDF5¶

查找 HDF5

find_package(HDF5)

代码说明

  • find_package(HDF5)：调用 `find_package()` 命令，查找并加载外部依赖包。

指定查找 HDF5 的最低要求版本

find_package(HDF5 1.8.15)

代码说明

  • find_package(HDF5 1.8.15)：调用 `find_package()` 命令，查找并加载外部依赖包。

查找 HDF5 并将其设为必需（如果未找到 HDF5，处理将停止并报错）

find_package(HDF5 1.8.15 REQUIRED)

代码说明

  • find_package(HDF5 1.8.15 REQUIRED)：调用 `find_package()` 命令，查找并加载外部依赖包。

搜索静态 HDF5 库

set(HDF5_USE_STATIC_LIBRARIES TRUE)
find_package(HDF5)

代码说明

  • set(HDF5_USE_STATIC_LIBRARIES TRUE) find_package(HDF5)：调用 `set()` 命令，把变量 `HDF5_USE_STATIC_LIBRARIES` 设为给定的值。

指定组件以查找高级 C 和 C++ 函数

find_package(HDF5 COMPONENTS C CXX HL)

代码说明

  • find_package(HDF5 COMPONENTS C CXX HL)：调用 `find_package()` 命令，查找并加载外部依赖包。

示例：使用 HDF5¶

查找 HDF5 并将其链接到项目目标

find_package(HDF5)
target_link_libraries(project_target PRIVATE HDF5::HDF5)

代码说明

  • find_package(HDF5) target_link_libraries(project_target PRIVATE HDF5::HDF5)：调用 `find_package()` 命令，查找并加载外部依赖包。

使用 Fortran HDF5 和 HDF5-HL 函数

find_package(HDF5 COMPONENTS Fortran HL)
target_link_libraries(project_target PRIVATE HDF5::HDF5)

代码说明

  • find_package(HDF5 COMPONENTS Fortran HL) target_link_libraries(project_target PRIVATE HDF5::HDF5)：调用 `find_package()` 命令，查找并加载外部依赖包。
