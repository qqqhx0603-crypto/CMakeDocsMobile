FindKDE3¶

注意

此模块专门针对 KDE 3 版本，该版本已过时且不再维护。如需使用 CMake 开发基于现代 KDE 技术的应用程序，请使用较新版本的 KDE，并参考 KDE 文档。

查找 KDE 3 头文件目录、库以及 KDE 专用的预处理器工具

find_package(KDE3 [...])

代码说明

  • find_package(KDE3 [...])：调用 find_package() 命令，查找并加载外部依赖包。

此模块提供了构建 KDE 3 软件的使用要求，并定义了几个辅助命令以简化在 CMake 中使用 KDE 3 的过程。

结果变量¶

此模块定义了以下变量

KDE3_FOUND

布尔值，表示是否找到了 KDE 3。

KDE3_DEFINITIONS

编译 KDE 3 软件所需的编译器定义。

KDE3_INCLUDE_DIRS

KDE 和 Qt 的头文件目录，用于 target_include_directories() 命令。

KDE3_LIB_DIR

包含已安装的 KDE 3 库的目录，用于 target_link_directories() 命令。

QT_AND_KDECORE_LIBS

包含 Qt 和 kdecore 库的列表，通常在链接 KDE 3 时一起使用。

缓存变量¶

以下缓存变量也可以设置

KDE3_INCLUDE_DIR

包含 KDE 3 头文件的目录。

KDE3_DCOPIDL_EXECUTABLE

dcopidl 可执行文件的路径。

KDE3_DCOPIDL2CPP_EXECUTABLE

dcopidl2cpp 可执行文件的路径。

KDE3_KCFGC_EXECUTABLE

kconfig_compiler 可执行文件的路径。

提示¶

此模块接受以下变量

KDE3_BUILD_TESTS

作为用户可调选项提供。将此变量设置为布尔值 true 可构建 KDE 3 测试用例。

命令¶

此模块提供以下命令以在 CMake 中处理 KDE 3

kde3_automoc¶

为给定的源文件启用 moc 的自动处理

kde3_automoc(<sources>...)

代码说明

  • kde3_automoc(<sources>...)：调用 kde3_automoc() 命令，具体行为由这一行传入的参数决定。

调用此命令可启用自动 moc 文件处理。例如，如果源文件（例如 foo.cpp）包含 include "foo.moc"，则会自动为相应的头文件（foo.h）生成 moc 文件。若要跳过特定源文件的处理，请设置 SKIP_AUTOMOC 源文件属性。

kde3_add_moc_files¶

使用 moc 处理头文件

kde3_add_moc_files(<variable> <headers>...)

代码说明

  • kde3_add_moc_files(<variable> <headers>...)：调用 kde3_add_moc_files() 命令，具体行为由这一行传入的参数决定。

如果不使用 kde3_automoc()，可以使用此命令为一个或多个 <headers> 文件生成 moc 文件。生成的文件的命名格式为 <filename>.moc.cpp，且这些生成源文件的列表将存储在变量 <variable> 中，以供项目目标使用。

kde3_add_dcop_skels¶

生成 KIDL 和 DCOP 骨架文件

kde3_add_dcop_skels(<variable> <dcop-headers>...)

代码说明

  • kde3_add_dcop_skels(<variable> <dcop-headers>...)：调用 kde3_add_dcop_skels() 命令，具体行为由这一行传入的参数决定。

此命令从给定的 DCOP 头文件生成 .kidl 和 DCOP 骨架源文件。生成的源文件列表将存储在变量 <variable> 中，以供项目目标使用。

kde3_add_dcop_stubs¶

生成 DCOP 存根文件

kde3_add_dcop_stubs(<variable> <headers>...)

代码说明

  • kde3_add_dcop_stubs(<variable> <headers>...)：调用 kde3_add_dcop_stubs() 命令，具体行为由这一行传入的参数决定。

使用此命令从一个或多个给定的头文件生成 DCOP 存根。生成的源文件列表将存储在变量 <variable> 中，以供项目目标使用。

kde3_add_ui_files¶

添加 Qt 设计器 UI 文件

kde3_add_ui_files(<variable> <ui-files>...)

代码说明

  • kde3_add_ui_files(<variable> <ui-files>...)：调用 kde3_add_ui_files() 命令，具体行为由这一行传入的参数决定。

此命令从给定的 Qt 设计器 .ui 文件创建实现文件。生成的文件列表将存储在变量 <variable> 中，以供项目目标使用。

kde3_add_kcfg_files¶

添加 KDE kconfig 编译器文件

kde3_add_kcfg_files(<variable> <kcfgc-files>...)

代码说明

  • kde3_add_kcfg_files(<variable> <kcfgc-files>...)：调用 kde3_add_kcfg_files() 命令，具体行为由这一行传入的参数决定。

使用此命令将 KDE kconfig 编译器文件（.kcfgc）添加到应用程序/库中。生成的源文件列表将存储在变量 <variable> 中，以供项目目标使用。

kde3_install_libtool_file¶

创建并安装 libtool 文件

kde3_install_libtool_file(<target>)

代码说明

  • kde3_install_libtool_file(<target>)：调用 kde3_install_libtool_file() 命令，具体行为由这一行传入的参数决定。

此命令为给定的目标 <target> 创建并安装一个基础的 libtool 文件。

kde3_add_executable¶

添加 KDE 可执行文件

kde3_add_executable(<name> <sources>...)

代码说明

  • kde3_add_executable(<name> <sources>...)：调用 kde3_add_executable() 命令，具体行为由这一行传入的参数决定。

此命令在功能上与内置的 add_executable() 命令相同。它最初的目的是支持该模块未来版本中的额外功能。

kde3_add_kpart¶

创建 KDE 插件

kde3_add_kpart(<name> [WITH_PREFIX] <sources>...)

代码说明

  • kde3_add_kpart(<name> [WITH_PREFIX] <sources>...)：调用 kde3_add_kpart() 命令，具体行为由这一行传入的参数决定。

此命令通过一个或多个源文件 <sources> 创建 KDE 插件（KPart、kioslave 等）。它还会创建并安装相应的 libtool .la 文件。

如果提供了 WITH_PREFIX 选项，生成的插件名称将带有 lib 前缀。否则，不添加前缀。

kde3_add_kdeinit_executable¶

创建一个可通过 kdeinit 加载的模块形式的 KDE 应用程序

kde3_add_kdeinit_executable(<name> <sources>...)

代码说明

  • kde3_add_kdeinit_executable(<name> <sources>...)：调用 kde3_add_kdeinit_executable() 命令，具体行为由这一行传入的参数决定。

此命令通过一个或多个源文件 <sources> 创建一个名为 kdeinit_<name> 的库。它还会构建一个链接到该库的小型可执行文件。
