FindFontconfig¶

3.14 版新增。

查找 Fontconfig，一个用于字体配置和定制的库

find_package(Fontconfig [<version>] [...])

导入的目标¶

此模块提供以下 导入目标

Fontconfig::Fontconfig

封装 Fontconfig 使用需求的 target，如果找到 Fontconfig 则可用。

结果变量¶

此模块定义了以下变量

Fontconfig_FOUND

布尔值，指示是否找到（请求版本的）Fontconfig。

Fontconfig_VERSION

找到的 Fontconfig 版本。

Fontconfig_LIBRARIES

使用 Fontconfig 时需要链接的库。

Fontconfig_INCLUDE_DIRS

包含使用 Fontconfig 所需头文件的包含目录。

Fontconfig_COMPILE_OPTIONS

使用 Fontconfig 所需的编译器选项。这些应该传递给 target_compile_options()，当不使用 Fontconfig::Fontconfig 导入 target 时。

示例¶

查找 Fontconfig 并将其链接到项目 target

find_package(Fontconfig)
target_link_libraries(project_target PRIVATE Fontconfig::Fontconfig)
