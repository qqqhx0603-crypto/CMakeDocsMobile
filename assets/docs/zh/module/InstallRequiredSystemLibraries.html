安装所需的系统库¶

此模块搜索编译器提供的系统运行时库，并添加用于安装它们的规则。

在 CMake 项目中加载此模块，使用

include(InstallRequiredSystemLibraries)

在包含此模块之前，可以设置一些可选变量来调整行为

CMAKE_INSTALL_SYSTEM_RUNTIME_LIBS

指定可能未被检测到的其他运行时库。包含后，任何检测到的库都将附加到此变量中。

CMAKE_INSTALL_SYSTEM_RUNTIME_LIBS_SKIP

设置为 TRUE 以跳过调用 install(PROGRAMS) 命令，以便包含者可以指定自己的安装规则，使用 CMAKE_INSTALL_SYSTEM_RUNTIME_LIBS 的值来获取库列表。

CMAKE_INSTALL_DEBUG_LIBRARIES

设置为 TRUE 以在可用时使用 MSVC 工具安装调试运行时库。

CMAKE_INSTALL_DEBUG_LIBRARIES_ONLY

设置为 TRUE 以仅安装 MSVC 工具的调试运行时库，即使发布运行时库也可用。

CMAKE_INSTALL_UCRT_LIBRARIES

3.6 版本新增。

设置为 TRUE 以安装 Windows Universal CRT 库，用于应用程序本地部署（例如，部署到 Windows XP）。这仅在使用 Visual Studio 2015 或更高版本的 MSVC 时才有意义。

3.9 版本新增：可以设置 CMAKE_WINDOWS_KITS_10_DIR 环境变量 为绝对路径，以告诉 CMake 在自定义位置查找 Windows 10 SDK。指定的目录预计包含 Redist/ucrt/DLLs/* 目录。

CMAKE_INSTALL_MFC_LIBRARIES

设置为 TRUE 以安装 MSVC MFC 运行时库。

CMAKE_INSTALL_OPENMP_LIBRARIES

设置为 TRUE 以安装 MSVC OpenMP 运行时库

CMAKE_INSTALL_SYSTEM_RUNTIME_DESTINATION

指定 install(PROGRAMS) 命令的 DESTINATION 选项。如果未指定，则默认在 Windows 上为 bin，在其他平台上为 lib。

CMAKE_INSTALL_SYSTEM_RUNTIME_LIBS_NO_WARNINGS

设置为 TRUE 以禁用有关不存在的必需库文件的警告。（例如，Visual Studio Express 版本可能不提供可再发行文件。）

CMAKE_INSTALL_SYSTEM_RUNTIME_COMPONENT

3.3 版本中新增。

指定 install(PROGRAMS) 命令的 COMPONENT 选项。如果未指定，则不会使用此选项。

3.10 版本新增：支持安装 Intel 编译器运行时。
