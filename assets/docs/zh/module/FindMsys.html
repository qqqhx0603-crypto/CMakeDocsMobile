FindMsys¶

3.21 版本新增。

查找 MSYS，一个在 Microsoft Windows（微软桌面系统） 上原生运行的 POSIX 兼容环境

find_package(Msys [...])

代码说明

  • find_package(Msys [...])：调用 `find_package()` 命令，查找并加载外部依赖包。

注意

此模块主要用于其他 查找模块，以帮助在使用 find_*() 命令（例如 find_program()）时定位程序。在大多数情况下，直接使用这些命令就足够了。只有当已知某个特定程序是通过 MSYS 安装并且可以从 Windows（微软桌面系统） 使用时，才使用此模块。

结果变量¶

此模块定义了以下变量

Msys_FOUND

版本 4.2 中添加。

布尔值，指示是否找到了 MSYS。

MSYS_INSTALL_PATH

MSYS 根安装目录的路径。

示例¶

查找 MSYS 安装并在自定义查找模块中使用其路径

FindFoo.cmake¶
find_package(Msys)
find_program(Foo_EXECUTABLE NAMES foo PATHS ${MSYS_INSTALL_PATH}/usr/bin)

include(FindPackageHandleStandardArgs)
find_package_handle_standard_args(Foo REQUIRED_VARS Foo_EXECUTABLE)

代码说明

  • find_package(Msys) find_program(Foo_EXECUTABLE NAMES foo PATHS ${MSYS_INSTALL_PATH}/usr/bin) include(FindPackageHandleStandardArgs) find_package_handle_standard_args(Foo REQUIRED_VARS Foo_EXECUTABLE)：调用 `find_package()` 命令，查找并加载外部依赖包。
