FindQuickTime¶

查找 QuickTime 多媒体框架，它为视频、音频和交互式媒体提供支持

find_package(QuickTime [...])

代码说明

  • find_package(QuickTime [...])：调用 find_package() 命令，查找并加载外部依赖包。

注意

此模块用于 QuickTime 框架，该框架已被 Apple 弃用且不再支持。在 Apple 系统上，请改用 AVFoundation 和 AVKit。

结果变量¶

此模块定义了以下变量

QuickTime_FOUND

3.3 版本中新增。

布尔值，指示是否找到了 QuickTime。

缓存变量¶

以下缓存变量也可以设置

QUICKTIME_LIBRARY

QuickTime 库的路径。

QUICKTIME_INCLUDE_DIR

包含 QuickTime 头文件的目录。

提示¶

此模块接受以下变量

QUICKTIME_DIR

环境变量，可设置为帮助定位安装在自定义位置的 QuickTime 库。它应指向配置、构建和安装 QuickTime 库时使用的安装目标：./configure --prefix=$QUICKTIME_DIR。

已弃用变量¶

以下变量提供用于向后兼容性

QUICKTIME_FOUND

自版本 4.2 起已弃用：使用 QuickTime_FOUND，其值相同。

布尔值，指示是否找到了 QuickTime。

示例¶

查找 QuickTime 库并创建一个导入接口目标，以便将其链接到项目目标

find_package(QuickTime)

if(QuickTime_FOUND AND NOT TARGET QuickTime::QuickTime)
  add_library(QuickTime::QuickTime INTERFACE IMPORTED)
  set_target_properties(
    QuickTime::QuickTime
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${QUICKTIME_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${QUICKTIME_LIBRARY}"
  )
endif()

target_link_libraries(example PRIVATE QuickTime::QuickTime)

代码说明

  • find_package(QuickTime)：调用 find_package() 命令，查找并加载外部依赖包。

  • if(QuickTime_FOUND AND NOT TARGET QuickTime::QuickTime)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句；条件运算符：NOT 表示逻辑非：反转后续条件的真假值；AND 表示逻辑与：两侧条件都为真时结果为真；TARGET 表示判断给定构建目标是否已定义。

  • add_library(QuickTime::QuickTime INTERFACE IMPORTED)：调用 add_library() 命令，定义一个库目标。

  • set_target_properties(：这一行继续补充当前命令或表达式的内容。

  • QuickTime::QuickTime：这一行继续补充当前命令或表达式的内容。

  • PROPERTIES：这一行继续补充当前命令或表达式的内容。

  • INTERFACE_INCLUDE_DIRECTORIES "${QUICKTIME_INCLUDE_DIR}"：这一行继续补充当前命令或表达式的内容；变量说明：QUICKTIME_INCLUDE_DIR：包含目录、目录相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • INTERFACE_LINK_LIBRARIES "${QUICKTIME_LIBRARY}"：这一行继续补充当前命令或表达式的内容；变量说明：QUICKTIME_LIBRARY：库相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • )：这一行继续补充当前命令或表达式的内容。

  • endif()：这一行结束或切换上一段控制结构的分支。

  • target_link_libraries(example PRIVATE QuickTime::QuickTime)：调用 target_link_libraries() 命令，为目标链接依赖库。
