FindosgWidget¶

查找 OpenSceneGraph 工具包中的 osgWidget NodeKit。

注意

在大多数情况下，建议使用 FindOpenSceneGraph 模块，并将 osgWidget 列为组件。这将自动处理依赖项，例如 OpenThreads 和核心 osg 库

find_package(OpenSceneGraph COMPONENTS osgWidget)

此模块由 FindOpenSceneGraph 内部使用，以查找 osgWidget NodeKit。它不打算在通常使用 find_package() 命令时直接包含。但是，它作为一个独立的模块提供，用于高级用例，在这些用例中需要对检测进行更精细的控制。例如，显式查找 osgWidget 或绕过自动组件检测

find_package(osgWidget)

OpenSceneGraph 和 osgWidget 头文件旨在包含在 C++ 项目源代码中，如下所示：

example.cxx¶
#include <osg/PositionAttitudeTransform>
#include <osgWidget/Widget>
// ...

使用 OpenSceneGraph 工具包时，可能还需要其他库，例如 OpenGL。

结果变量¶

此模块定义了以下变量

osgWidget_FOUND

3.3 版本中新增。

布尔值，指示是否找到了 OpenSceneGraph 工具包的 osgWidget NodeKit。

OSGWIDGET_LIBRARIES

使用 osgWidget 需要链接的库。

OSGWIDGET_LIBRARY

一个结果变量，其值与 OSGWIDGET_LIBRARIES 变量相同。

缓存变量¶

以下缓存变量也可以设置

OSGWIDGET_INCLUDE_DIR

包含使用 osgWidget 所需头文件的包含目录。

OSGWIDGET_LIBRARY_DEBUG

osgWidget 调试库的路径。

提示¶

此模块接受以下变量

OSGDIR

环境变量，可用于帮助定位 OpenSceneGraph 工具包，包括其 osgWidget NodeKit，当安装在自定义位置时。它应指向 OpenSceneGraph 安装前缀，该前缀在配置、构建和安装时使用：./configure --prefix=$OSGDIR。

已弃用变量¶

以下变量提供用于向后兼容性

OSGWIDGET_FOUND

自版本 4.2 起已弃用：使用 osgWidget_FOUND，其值相同。

布尔值，指示是否找到了 OpenSceneGraph 工具包的 osgWidget NodeKit。

示例¶

使用此模块显式查找 osgWidget 并创建一个 导入目标，该目标封装了将其链接到项目目标的使用要求

find_package(osgWidget)

if(osgWidget_FOUND AND NOT TARGET osgWidget::osgWidget)
  add_library(osgWidget::osgWidget INTERFACE IMPORTED)
  set_target_properties(
    osgWidget::osgWidget
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${OSGWIDGET_INCLUDE_DIR}"
      INTERFACE_LINK_LIBRARIES "${OSGWIDGET_LIBRARIES}"
  )
endif()

target_link_libraries(example PRIVATE osgWidget::osgWidget)
