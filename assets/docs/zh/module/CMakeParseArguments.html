CMakeParseArguments¶

自版本 3.5 起已弃用。

该模块曾经实现了 cmake_parse_arguments() 命令，该命令现在由 CMake 本身实现。 现在它是一个空占位符，用于与包含该模块以从 CMake 3.4 及更低版本获取该命令的项目兼容。
