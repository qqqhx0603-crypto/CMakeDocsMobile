FindWget¶

此模块查找用于从 Web 服务器检索内容的命令行工具 wget

find_package(Wget [...])

代码说明

  • find_package(Wget [...])：调用 find_package() 命令，查找并加载外部依赖包。

结果变量¶

此模块定义了以下变量

Wget_FOUND

3.3 版本中新增。

布尔值，指示是否找到了 wget。

缓存变量¶

以下缓存变量也可以设置

WGET_EXECUTABLE

wget 工具的完整路径。

已弃用变量¶

以下变量提供用于向后兼容性

WGET_FOUND

自版本 4.2 起已弃用：使用 Wget_FOUND，其值相同。

布尔值，指示是否找到了 wget。

示例¶

查找 wget 并在进程中执行它

find_package(Wget)
if(Wget_FOUND)
  execute_process(COMMAND ${WGET_EXECUTABLE} -h)
endif()

代码说明

  • find_package(Wget)：调用 find_package() 命令，查找并加载外部依赖包。

  • if(Wget_FOUND)：开始一个 if() 条件判断，根据条件结果选择是否执行后续语句。

  • execute_process(COMMAND ${WGET_EXECUTABLE} -h)：调用 execute_process() 命令，在配置阶段执行外部进程；变量说明：WGET_EXECUTABLE：可执行文件相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • endif()：这一行结束或切换上一段控制结构的分支。

另请参阅¶
