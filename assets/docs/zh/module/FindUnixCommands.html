FindUnixCommands¶

自版本 3.26 起已弃用：请使用 ${CMAKE_COMMAND} -E 子命令代替。

查找 Unix 命令 bash、cp、gzip、mv、rm 和 tar，包括 Cygwin 或 MSYS 中的命令

find_package(UnixCommands [...])

代码说明

  • find_package(UnixCommands [...])：调用 find_package() 命令，查找并加载外部依赖包。

结果变量¶

此模块定义了以下变量

UnixCommands_FOUND

3.3 版本中新增。

布尔值，指示是否找到 Unix 命令。

缓存变量¶

以下缓存变量也可以设置

BASH

bash 命令-line 可执行文件的路径。

CP

cp 命令-line 可执行文件的路径。

GZIP

gzip 命令-line 可执行文件的路径。

MV

mv 命令-line 可执行文件的路径。

RM

rm 命令-line 可执行文件的路径。

TAR

tar 命令-line 可执行文件的路径。
