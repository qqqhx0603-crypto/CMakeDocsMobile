FindOpenCL¶

版本 3.1 中新增。

查找开放计算语言 (OpenCL)

find_package(OpenCL [<version>] [...])

代码说明

  • find_package(OpenCL [<version>] [...])：调用 find_package() 命令，查找并加载外部依赖包。

OpenCL 是一个用于编写在异构平台（如 CPU、GPU 和其他加速器）上执行程序的框架。

版本 3.24 中添加： 检测 OpenCL 3.0。

版本 3.10 中添加： 检测 OpenCL 2.1 和 2.2。

导入的目标¶

此模块提供以下 导入目标

OpenCL::OpenCL

3.7 版本中新增。

目标封装了 OpenCL 使用要求，如果已找到 OpenCL，则可用。

结果变量¶

此模块定义了以下变量

OpenCL_FOUND

布尔值，指示是否找到（请求版本的）OpenCL。

OpenCL_VERSION

版本 4.2 中添加。

找到的最高支持的 OpenCL 版本，格式为 <major>.<minor>（例如 1.2）。

OpenCL_VERSION_MAJOR

OpenCL 实现的主要版本。

OpenCL_VERSION_MINOR

OpenCL 实现的次要版本。

OpenCL_INCLUDE_DIRS

使用 OpenCL 所需的包含目录。

OpenCL_LIBRARIES

链接到 OpenCL 所需的库。

缓存变量¶

以下缓存变量也可以设置

OpenCL_INCLUDE_DIR

OpenCL 包含目录。

OpenCL_LIBRARY

OpenCL 库的路径。

已弃用变量¶

以下变量提供用于向后兼容性

OpenCL_VERSION_STRING

版本 4.2 起已弃用： 使用 OpenCL_VERSION，它具有相同的值。

找到的最高支持的 OpenCL 版本，格式为 <major>.<minor>。

示例¶

查找 OpenCL 并将其链接到项目目标

find_package(OpenCL)
target_link_libraries(project_target PRIVATE OpenCL::OpenCL)

代码说明

  • find_package(OpenCL)：调用 find_package() 命令，查找并加载外部依赖包。

  • target_link_libraries(project_target PRIVATE OpenCL::OpenCL)：调用 target_link_libraries() 命令，为目标链接依赖库。
