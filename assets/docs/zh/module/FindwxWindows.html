FindwxWindows¶

3.0 版本弃用：已被 FindwxWidgets 取代。

查找 wxWidgets（前身为 wxWindows）安装，并确定其包含目录和库的位置，以及库名称。

find_package(wxWindows [...])

代码说明

  • find_package(wxWindows [...])：调用 find_package() 命令，查找并加载外部依赖包。

支持用于单体（monolithic）构建的 wxWidgets 2.6.x 版本，例如那些在 wx/build/msw 目录中使用以下方式编译的版本：

nmake -f makefile.vc BUILD=debug SHARED=0 USE_OPENGL=1 MONOLITHIC=1

代码说明

  • nmake -f makefile.vc BUILD=debug SHARED=0 USE_OPENGL=1 MONOLITHIC=1：这是命令执行后的输出结果，内容为：nmake -f makefile.vc BUILD=debug SHARED=0 USE_OPENGL=1 MONOLITHIC=1；运算符：= 表示赋值或键值关联。

结果变量¶

此模块定义了以下变量

WXWINDOWS_FOUND

布尔值，表示是否找到了 wxWidgets。

WXWINDOWS_LIBRARIES

使用 wxWidgets 时链接所需的库。这包括 wxWidgets 库的路径和任何额外的链接器标志，通常源自 Unix/Linux 系统上 wx-config --libs 的输出。

CMAKE_WXWINDOWS_CXX_FLAGS

使用 wxWidgets 所需的编译器选项（如果有）。在 Linux 上，这对应于 wx-config --cxxflags 的输出。

WXWINDOWS_INCLUDE_DIR

包含 wx/wx.h 和 wx/setup.h 头文件的目录。

WXWINDOWS_LINK_DIRECTORIES

链接目录，对于在类 Unix 平台上设置 rpath 非常有用。

WXWINDOWS_DEFINITIONS

使用 wxWidgets 所需的额外编译定义（如果有）。

提示¶

在调用 find_package(wxWindows) 之前，此模块接受以下变量：

WXWINDOWS_USE_GL

将此变量设置为布尔值 true 以请求 OpenGL（开放图形库） 支持。

HAVE_ISYSTEM

当 C++ 编译器为 GNU (g++) 时，将此变量设置为布尔值 true，以使用 -isystem 替换 -I 编译器选项。

已弃用变量¶

以下变量提供用于向后兼容性

CMAKE_WX_CAN_COMPILE

1.8 版本弃用：已被具有相同值的 WXWINDOWS_FOUND 变量取代。

WXWINDOWS_LIBRARY

1.8 版本弃用：已被具有相同值的 WXWINDOWS_LIBRARIES 变量取代。

CMAKE_WX_CXX_FLAGS

1.8 版本弃用：已被具有相同值的 CMAKE_WXWINDOWS_CXX_FLAGS 变量取代。

WXWINDOWS_INCLUDE_PATH

1.8 版本弃用：已被具有相同值的 WXWINDOWS_INCLUDE_DIR 变量取代。

示例¶

示例：在较旧的 CMake 版本中查找 wxWidgets¶

在较旧的 CMake 版本中，可以通过以下方式找到 wxWidgets (wxWindows)：

find_package(wxWindows)

代码说明

  • find_package(wxWindows)：调用 find_package() 命令，查找并加载外部依赖包。

若要请求 OpenGL（开放图形库） 支持，可以在调用 find_package() 之前设置 WXWINDOWS_USE_GL 变量。

set(WXWINDOWS_USE_GL ON)
find_package(wxWindows)

代码说明

  • set(WXWINDOWS_USE_GL ON)：调用 set() 命令，把变量 WXWINDOWS_USE_GL 设为给定的值；变量说明：WXWINDOWS_USE_GL：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • find_package(wxWindows)：调用 find_package() 命令，查找并加载外部依赖包。

在 CMake 中使用 wxWidgets (wxWindows) 通常是通过包含 Use_wxWindows 模块来完成的，该模块会查找 wxWidgets 并设置相应的库、包含目录和编译器标志。

include(Use_wxWindows)

代码说明

  • include(Use_wxWindows)：调用 include() 命令，加载并执行另一个 CMake 脚本。

示例：自 CMake 3.0 起查找 wxWidgets¶

从 CMake 3.0 开始，可以使用 FindwxWidgets 模块来查找 wxWidgets。

find_package(wxWidgets)

代码说明

  • find_package(wxWidgets)：调用 find_package() 命令，查找并加载外部依赖包。
