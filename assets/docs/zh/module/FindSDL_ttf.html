FindSDL_ttf¶

查找 SDL_ttf 库。该库为 SDL（Simple DirectMedia Layer，简单直接媒体层）应用程序提供使用 TrueType 字体渲染文本的支持。

find_package(SDL_ttf [<version>] [...])

注意

此模块专门用于 SDL_ttf 版本 1。从 2.0.15 版本开始，当使用 CMake 构建时，SDL_ttf 会提供一个 CMake 包配置文件，应使用 find_package(SDL2_ttf) 来查找。同样地，SDL_ttf 版本 3 应使用 find_package(SDL3_ttf) 来查找。这些较新的版本提供了 导入目标 (Imported Targets)，其中封装了使用需求。有关更多信息，请参考 SDL 官方文档。

结果变量¶

此模块定义了以下变量

SDL_ttf_FOUND

3.3 版本中新增。

布尔值，指示是否找到了（所请求版本的）SDL_ttf 库。

SDL_ttf_VERSION

版本 4.2 中添加。

包含所找到的 SDL_ttf 版本号的易读字符串。

SDL_TTF_INCLUDE_DIRS

包含使用 SDL_ttf 库所需头文件的目录。

SDL_TTF_LIBRARIES

使用 SDL_ttf 时需要链接的库。

提示¶

此模块接受以下变量

SDLDIR

环境变量，可用于帮助定位安装在自定义位置的 SDL 库。它应指向配置、构建和安装 SDL 库时使用的安装目标：./configure --prefix=$SDLDIR。

已弃用变量¶

以下变量提供用于向后兼容性

SDL_TTF_VERSION_STRING

4.2 版本弃用：请使用 SDL_ttf_VERSION，其值相同。

包含所找到的 SDL_ttf 版本号的易读字符串。

SDL_TTF_FOUND

4.2 版本弃用：请使用 SDL_ttf_FOUND，其值相同。

SDLTTF_FOUND

2.8.10 版本弃用：已被 SDL_ttf_FOUND 取代，其值相同。

SDLTTF_INCLUDE_DIR

2.8.10 版本弃用：已被 SDL_TTF_INCLUDE_DIRS 取代，其值相同。

SDLTTF_LIBRARY

2.8.10 版本弃用：已被 SDL_TTF_LIBRARIES 取代，其值相同。

示例¶

查找 SDL_ttf 库并创建一个导入的接口目标，以便将其链接到项目目标

find_package(SDL_ttf)

if(SDL_ttf_FOUND AND NOT TARGET SDL::SDL_ttf)
  add_library(SDL::SDL_ttf INTERFACE IMPORTED)
  set_target_properties(
    SDL::SDL_ttf
    PROPERTIES
      INTERFACE_INCLUDE_DIRECTORIES "${SDL_TTF_INCLUDE_DIRS}"
      INTERFACE_LINK_LIBRARIES "${SDL_TTF_LIBRARIES}"
  )
endif()

target_link_libraries(project_target PRIVATE SDL::SDL_ttf)

在使用 SDL_ttf 版本 2 时，上游包直接提供了 SDL2_ttf::SDL2_ttf 导入目标。它可以在项目中直接使用，无需使用此模块。

find_package(SDL2_ttf)
target_link_libraries(project_target PRIVATE SDL2_ttf::SDL2_ttf)

同样地，对于 SDL_ttf 版本 3

find_package(SDL3_ttf)
target_link_libraries(project_target PRIVATE SDL3_ttf::SDL3_ttf)
