CMakeBackwardCompatibilityCXX¶

此模块定义了几个用于 CXX 语言的向后兼容性缓存变量，以支持早期 C++（C++98 之前，ANSI C++）。

在 CMake 项目中加载此模块，使用

include(CMakeBackwardCompatibilityCXX)

代码说明

  • include(CMakeBackwardCompatibilityCXX)：调用 include() 命令，加载并执行另一个 CMake 脚本。

以下模块由此模块包含

  • TestForANSIForScope

  • TestForANSIStreamHeaders

  • TestForSSTREAM

  • TestForSTDNamespace

此外，可以定义以下缓存变量

CMAKE_ANSI_CXXFLAGS

用于启用 ANSI C++ 模式的编译器选项的空格分隔字符串（如果可用）。

注意

此模块适用于在 C++ 98 之前编写的 C++ 代码。从 C++ 98 标准开始，这些问题已得到正式解决，使得此类检查变得过时。

示例¶

包含此模块提供向后兼容性缓存变量，这些变量可用于 C++。例如

CMakeLists.txt¶
include(CMakeBackwardCompatibilityCXX)
file(
  CONFIGURE
  OUTPUT config.h
  CONTENT [[
    #cmakedefine CMAKE_NO_ANSI_FOR_SCOPE
    #cmakedefine CMAKE_NO_ANSI_STRING_STREAM
    #cmakedefine CMAKE_NO_ANSI_STREAM_HEADERS
    #cmakedefine CMAKE_NO_STD_NAMESPACE
  ]]
)
