FindX11¶

在类 Unix 系统上查找 X11 安装。

find_package(X11 [COMPONENTS <components>...] [...])

代码说明

  • find_package(X11 [COMPONENTS <components>...] [...])：调用 find_package() 命令，查找并加载外部依赖包。

结果变量¶

此模块定义了以下变量

X11_FOUND

布尔值，指示是否找到了 X11。

X11_<component>_FOUND

布尔值，指示是否找到 <component>。

X11_INCLUDE_DIR

使用 X11 所需的包含目录。

X11_LIBRARIES

使用 X11 进行链接所需的库。
