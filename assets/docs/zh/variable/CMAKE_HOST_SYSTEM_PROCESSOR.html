CMAKE_HOST_SYSTEM_PROCESSOR¶

变量说明

CMAKE_HOST_SYSTEM_PROCESSOR：CMake 正在运行的 CPU 的名称。

CMake 正在运行的 CPU 的名称。

Windows 平台¶

在 Windows（微软桌面系统） 上，此变量设置为环境变量 PROCESSOR_ARCHITECTURE 的值。

Unix 平台¶

在支持 uname 的系统上，此变量设置为

  • uname -m 在 GNU、Linux、Cygwin、Android 上，或

  • arch 在 OpenBSD 上，或

  • 在其他系统上，

    • uname -p 如果其退出代码非零，否则

    • uname -m。

macOS（苹果桌面系统） 平台¶

默认情况下，使用 uname -m 的值。

在 Apple Silicon 主机上，uname -m 打印的架构可能因 CMake 自身的架构和调用进程树的架构而异。

3.19.2 版本新增：在 Apple Silicon 主机上

  • 可以设置变量 CMAKE_APPLE_SILICON_PROCESSOR 或环境变量 CMAKE_APPLE_SILICON_PROCESSOR 以显式指定主机架构。

  • 如果未设置 CMAKE_OSX_ARCHITECTURES，CMake 会添加显式标志，以告知编译器为主机架构构建，这样工具链就不必根据进程树的架构进行猜测。
