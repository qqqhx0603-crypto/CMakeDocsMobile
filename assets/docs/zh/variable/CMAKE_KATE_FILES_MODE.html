CMAKE_KATE_FILES_MODE¶

在 3.27 版本中新增。

此缓存变量由 Kate 项目生成器使用，用于控制项目文件中 files 条目的模式设置。请参阅 cmake-generators(7)。

可能的值包括 AUTO、SVN、GIT、HG、FOSSIL 和 LIST。

当设置为 LIST 时，CMake 会将 CMake 已知的源文件列表放入项目文件中。当设置为 SVN、GIT、HG 或 FOSSIL 时，CMake 会根据 Subversion、git、Mercurial 或 Fossil 设置生成的项目，然后 Kate 将使用相应的命令行工具来检索项目中的文件列表。如果未设置或设置为 AUTO，CMake 将尝试检测源目录是否是 git 或 svn 的检出目录，并将相应的条目放入项目文件中。
