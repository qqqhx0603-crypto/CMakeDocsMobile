CMAKE_MESSAGE_CONTEXT_SHOW¶

变量说明

CMAKE_MESSAGE_CONTEXT_SHOW：在 3.17 版本中添加。

在 3.17 版本中添加。

将此变量设置为 true 将启用在每个由 message() 命令记录的行旁边显示上下文（请参阅 CMAKE_MESSAGE_CONTEXT 了解如何指定上下文本身）。

此变量是 cmake 命令行的 --log-context 选项的替代方案。命令行选项仅适用于那次 CMake 运行，而将 CMAKE_MESSAGE_CONTEXT_SHOW 设置为 true 作为缓存变量将确保后续的 CMake 运行将继续显示消息上下文。

项目不应设置 CMAKE_MESSAGE_CONTEXT_SHOW。它旨在供用户控制是否在消息中包含上下文。
