CMAKE_LINK_WHAT_YOU_USE_CHECK¶

3.22 版本新增。

在链接器之后由 LINK_WHAT_YOU_USE 命令执行，用于检查不必要地链接的共享库。 此检查当前仅在 ELF 平台上定义，其值为 ldd -u -r。

另请参阅 CMAKE_<LANG>_LINK_WHAT_YOU_USE_FLAG 变量。
