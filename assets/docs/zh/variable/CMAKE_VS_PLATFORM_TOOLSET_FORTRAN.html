CMAKE_VS_PLATFORM_TOOLSET_FORTRAN¶

变量说明

CMAKE_VS_PLATFORM_TOOLSET_FORTRAN：在版本 3.29 中添加。

在版本 3.29 中添加。

Visual Studio（集成开发环境） 项目要使用的 Fortran 编译器。

Visual Studio（集成开发环境） 生成器 支持选择已安装所需 Visual Studio（集成开发环境） 集成功能的 Fortran 编译器。编译器可以通过 CMAKE_GENERATOR_TOOLSET 中的字段 fortran=... 来指定。CMake 会在此变量中提供所选的 Fortran 编译器。

如果未指定该字段，则默认值取决于生成器

  • 在 Visual Studio 18 2026 及更高版本中，默认值为 ifx。

  • 在较旧的 Visual Studio（集成开发环境） 生成器 上，默认值为空，Intel Visual Studio（集成开发环境） 集成将其解释为等同于 ifort。
