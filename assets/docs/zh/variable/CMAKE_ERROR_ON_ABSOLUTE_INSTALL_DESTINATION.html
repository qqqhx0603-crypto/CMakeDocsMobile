CMAKE_ERROR_ON_ABSOLUTE_INSTALL_DESTINATION¶

变量说明

CMAKE_ERROR_ON_ABSOLUTE_INSTALL_DESTINATION：要求 cmake_install.cmake 脚本在遇到带有绝对 INSTALL DESTINATION 的文件时立即报错。

要求 cmake_install.cmake 脚本在遇到带有绝对 INSTALL DESTINATION 的文件时立即报错。

在安装违规文件之前，会发出致命错误。此变量由 CMake 生成的 cmake_install.cmake 脚本使用。如果在运行脚本时将此变量设置为 ON，脚本可能会发出致命错误消息。
