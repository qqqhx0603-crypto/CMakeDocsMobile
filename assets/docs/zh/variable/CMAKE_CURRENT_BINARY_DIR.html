CMAKE_CURRENT_BINARY_DIR¶

当前正在处理的二进制目录的路径。

这是 cmake 当前正在处理的构建目录的完整路径。通过 add_subdirectory() 添加的每个目录都会在构建树中创建一个二进制目录，并且当它被处理时，此变量将被设置。对于就地构建，这是当前正在处理的源目录。

在 cmake -P 脚本模式下运行时，CMake 将变量 CMAKE_BINARY_DIR、CMAKE_SOURCE_DIR、CMAKE_CURRENT_BINARY_DIR 和 CMAKE_CURRENT_SOURCE_DIR 设置为当前工作目录。

修改 CMAKE_CURRENT_BINARY_DIR 的行为未定义。
