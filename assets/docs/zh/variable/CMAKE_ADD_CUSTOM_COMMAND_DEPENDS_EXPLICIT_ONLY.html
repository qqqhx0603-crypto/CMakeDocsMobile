CMAKE_ADD_CUSTOM_COMMAND_DEPENDS_EXPLICIT_ONLY¶

在 3.27 版本中新增。

是否默认启用 DEPENDS_EXPLICIT_ONLY 选项在 add_custom_command() 中。

此变量影响 add_custom_command() 命令的默认行为。 将此变量设置为 ON 等效于在对该命令的所有使用中都使用 DEPENDS_EXPLICIT_ONLY 选项。

另请参阅 CMAKE_OPTIMIZE_DEPENDENCIES。
