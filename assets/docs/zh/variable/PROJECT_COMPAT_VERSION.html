PROJECT_COMPAT_VERSION¶

4.3 版本新增。

最近一次调用 project() 命令时，提供的 COMPAT_VERSION 选项的值（如果存在）。要获取顶级项目的兼容版本，请参阅 CMAKE_PROJECT_COMPAT_VERSION 变量。
