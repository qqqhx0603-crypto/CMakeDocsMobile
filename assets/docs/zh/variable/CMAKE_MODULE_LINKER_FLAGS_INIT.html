CMAKE_MODULE_LINKER_FLAGS_INIT¶

3.7 版本中新增。

用于初始化 CMAKE_MODULE_LINKER_FLAGS 缓存条目的首次构建树配置时的值。该变量旨在由 工具链文件 设置。CMake 可能会根据环境和目标平台在值的前面或后面附加内容。

另请参阅特定配置的变量 CMAKE_MODULE_LINKER_FLAGS_<CONFIG>_INIT。
