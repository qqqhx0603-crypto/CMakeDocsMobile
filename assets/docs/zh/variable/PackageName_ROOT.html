<PackageName>_ROOT¶

变量说明

<PackageName>_ROOT：_ROOT"><PackageName>_ROOT 3.12 版本新增。

3.12 版本新增。

对 find_package(<PackageName>) 的调用将在由 <PackageName>_ROOT CMake 变量指定的路径前缀中搜索，其中 <PackageName> 是传递给 find_package() 调用的（区分大小写）名称，而 _ROOT 是字面量。例如，find_package(Foo) 将搜索在 Foo_ROOT CMake 变量中（如果已设置）指定的路径前缀。请参阅策略 CMP0074。

此变量可以包含单个路径前缀或由 分号分隔的路径前缀列表。

另请参阅 <PackageName>_ROOT 环境变量。

<PACKAGENAME>_ROOT¶

在 3.27 版本中新增。

对 find_package(<PackageName>) 的调用还将搜索由大写 <PACKAGENAME>_ROOT CMake 变量指定的路径前缀。请参阅策略 CMP0144。
