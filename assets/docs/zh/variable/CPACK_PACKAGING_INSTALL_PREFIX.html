CPACK_PACKAGING_INSTALL_PREFIX¶

构建包中使用的前缀。

每个 CPack 生成器都有一个默认值（例如 /usr）。可以通过在 CMakeLists.txt 或 cpack(1) 命令行中设置替代值来覆盖此默认值。示例

set(CPACK_PACKAGING_INSTALL_PREFIX "/opt")

代码说明

  • set(CPACK_PACKAGING_INSTALL_PREFIX "/opt")：调用 `set()` 命令，把变量 `CPACK_PACKAGING_INSTALL_PREFIX` 设为给定的值。

这与 CMAKE_INSTALL_PREFIX 的用途不同，后者用于在不构建包的情况下从构建树进行安装。
