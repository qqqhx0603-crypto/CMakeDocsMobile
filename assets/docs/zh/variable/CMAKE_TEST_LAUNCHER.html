CMAKE_TEST_LAUNCHER¶

在版本 3.29 中添加。

此变量用于初始化在创建可执行目标时，目标属性 TEST_LAUNCHER。它用于指定用于运行测试的启动器，该启动器由 add_test() 命令添加，并运行一个可执行目标。

如果此变量包含一个 分号分隔的列表，则第一个值是命令，其余值是其参数。

此变量可以通过 CMAKE_TEST_LAUNCHER 环境变量进行初始化。
