CMAKE_Fortran_MODDIR_DEFAULT¶

变量说明

CMAKE_Fortran_MODDIR_DEFAULT：Fortran 默认模块输出目录。

Fortran 默认模块输出目录。

大多数 Fortran 编译器会将 .mod 文件写入当前工作目录。对于那些不这样做的情况，此变量将被设置为 .，并在未设置 Fortran_MODULE_DIRECTORY 目标属性时使用。
