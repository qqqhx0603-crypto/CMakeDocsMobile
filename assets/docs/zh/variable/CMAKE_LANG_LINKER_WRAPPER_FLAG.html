CMAKE_<LANG>_LINKER_WRAPPER_FLAG¶

3.13 版本新增。

定义用于将选项传递给链接器工具的编译器驱动程序选项的语法。它将被用于转换链接选项中的 LINKER: 前缀（参见 add_link_options() 和 target_link_options()）。

此变量包含一个 分号分隔的列表 的标记。如果将空格（即 " "）指定为最后一个标记，则标志和 LINKER: 参数将作为单独的参数传递给编译器驱动程序。 CMAKE_<LANG>_LINKER_WRAPPER_FLAG_SEP 变量可以被指定来管理参数的连接。

例如，对于 Clang，我们有

set (CMAKE_C_LINKER_WRAPPER_FLAG "-Xlinker" " ")

代码说明

  • set (CMAKE_C_LINKER_WRAPPER_FLAG "-Xlinker" " ")：调用 `set()` 命令，把变量 `CMAKE_C_LINKER_WRAPPER_FLAG` 设为给定的值。

指定 "LINKER:-z,defs" 将被转换为 -Xlinker -z -Xlinker defs。

对于 GNU GCC

set (CMAKE_C_LINKER_WRAPPER_FLAG "-Wl,")
set (CMAKE_C_LINKER_WRAPPER_FLAG_SEP ",")

代码说明

  • set (CMAKE_C_LINKER_WRAPPER_FLAG "-Wl,") set (CMAKE_C_LINKER_WRAPPER_FLAG_SEP ",")：调用 `set()` 命令，把变量 `CMAKE_C_LINKER_WRAPPER_FLAG` 设为给定的值。

指定 "LINKER:-z,defs" 将被转换为 -Wl,-z,defs。

对于 SunPro

set (CMAKE_C_LINKER_WRAPPER_FLAG "-Qoption" "ld" " ")
set (CMAKE_C_LINKER_WRAPPER_FLAG_SEP ",")

代码说明

  • set (CMAKE_C_LINKER_WRAPPER_FLAG "-Qoption" "ld" " ") set (CMAKE_C_LINKER_WRAPPER_FLAG_SEP ",")：调用 `set()` 命令，把变量 `CMAKE_C_LINKER_WRAPPER_FLAG` 设为给定的值。

指定 "LINKER:-z,defs" 将被转换为 -Qoption ld -z,defs。
