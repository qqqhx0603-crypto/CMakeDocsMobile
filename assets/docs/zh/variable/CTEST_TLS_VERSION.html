CTEST_TLS_VERSION¶

3.30 版本新增。

指定 CTest 的 TLSVersion 设置，在 ctest(1) 仪表盘客户端 脚本或在项目 CMakeLists.txt 代码中，在包含 CTest 模块之前。该值是允许通过 https:// URL 提交到仪表盘的最低 TLS 版本。

该值可以是以下之一：

  • 1.0

  • 1.1

  • 1.2

  • 1.3

如果未设置 CTEST_TLS_VERSION，则使用 CMAKE_TLS_VERSION 变量或 CMAKE_TLS_VERSION 环境变量。
