CMAKE_SYSTEM_NAME¶

变量说明

CMAKE_SYSTEM_NAME：CMake 构建所针对的操作系统名称。有关操作系统版本，请参阅 CMAKE_SYSTEM_VERSION 变量。

CMake 构建所针对的操作系统名称。有关操作系统版本，请参阅 CMAKE_SYSTEM_VERSION 变量。

注意：当以脚本模式运行时，CMAKE_SYSTEM_NAME 默认不会被设置，因为此时不进行任何构建。

宿主机构建的系统名称¶

CMAKE_SYSTEM_NAME 默认设置为与 CMAKE_HOST_SYSTEM_NAME 变量相同的值，以便构建目标为宿主系统。

交叉编译的系统名称¶

在首次配置新的构建树时，可以显式设置 CMAKE_SYSTEM_NAME 以启用 交叉编译。在这种情况下，必须同时显式设置 CMAKE_SYSTEM_VERSION 变量。

平台特定说明

  • MSYS2 的 msys/cmake 包 (/usr/bin/cmake) 仅在 MSYSTEM=MSYS 环境下工作，系统名称为 MSYS。在 MSYSTEM=MINGW64 等其他环境下，请使用其他包，例如 mingw64/mingw-w64-x86_64-cmake (/mingw64/bin/cmake)，该包针对 MSYSTEM=MINGW64，系统名称为 Windows。

  • Cygwin 的 cmake 包 (/usr/bin/cmake) 使用系统名称 CYGWIN。在 Windows（微软桌面系统） 上非 Cygwin 版本的 CMake (例如 $PROGRAMFILES/CMake/bin/cmake) 即使运行在 Cygwin 环境下，系统名称依然使用 Windows。
