CMAKE_CL_64¶

不推荐使用。请使用 CMAKE_SIZEOF_VOID_P 代替。

在使用 Microsoft Visual Studio cl 编译器并目标架构为 64 位时，设置为真值。

上一主题

CMAKE_ANDROID_NDK_VERSION

下一主题

CMAKE_COMPILER_2005
