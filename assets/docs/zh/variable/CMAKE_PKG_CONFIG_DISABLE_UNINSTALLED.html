CMAKE_PKG_CONFIG_DISABLE_UNINSTALLED¶

变量说明

CMAKE_PKG_CONFIG_DISABLE_UNINSTALLED：启用/禁用 cmake_pkg_config() 命令的默认“未安装”搜索行为。当此变量为 false 时，带有“-uninstalled”后缀的包文件比精确的包名称匹配具有更高的优先级。

4.0 版本新增。
