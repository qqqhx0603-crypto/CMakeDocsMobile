CMAKE_<LANG>_USING_LINKER_MODE¶

自版本 4.0 起已弃用：此变量不再使用。存储在 CMAKE_<LANG>_USING_LINKER_<TYPE> 变量中的信息由 CMAKE_<LANG>_LINK_MODE 变量确定。

在版本 3.29 中添加。

此变量控制应如何解释 CMAKE_<LANG>_USING_LINKER_<TYPE> 变量的值。支持的链接器模式值是

FLAG

CMAKE_<LANG>_USING_LINKER_<TYPE> 包含要传递给编译器前端的 分号分隔列表 中的标志。如果未设置 CMAKE_<LANG>_USING_LINKER_MODE，这也是默认行为。

TOOL

CMAKE_<LANG>_USING_LINKER_<TYPE> 包含链接器工具的路径。

警告

变量必须相应设置，以了解 CMake 如何管理链接步骤

  • 当直接使用链接器进行链接步骤时，需要并期望值为 TOOL。

  • 当编译器用作链接步骤的驱动程序时，需要值为 FLAG 或未设置该变量。
