CMAKE_FASTBUILD_CAPTURE_SYSTEM_ENV¶

版本 4.2 中添加。

控制将系统环境变量捕获到 fbuild.bff 中。将其设置为 OFF 会使所有工具（编译器和其他外部进程）的调用变为密封的。

注意

将此变量设置为 OFF 可能会破坏依赖于环境变量（如 INCLUDE 或 LIB）的 MSVC 工具链，除非这些变量在其他地方手动配置。

默认为 ON。
