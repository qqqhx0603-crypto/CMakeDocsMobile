CMAKE_<LANG>_IMPLICIT_INCLUDE_DIRECTORIES¶

变量说明

CMAKE_<LANG>_IMPLICIT_INCLUDE_DIRECTORIES：编译器为头文件隐式搜索的目录。

编译器为头文件隐式搜索的目录。

CMake 不为语言 <LANG> 在编译器命令行中显式指定这些目录。 这可以防止系统包含目录在某些编译器上被视为用户包含目录，这对于 C、CXX 和 CUDA 避免覆盖标准库头文件非常重要。

由于它没有标准库头文件，并且一些编译器不会为模块 .mod 文件搜索它们的隐式包含目录，因此此值不用于 Fortran。
