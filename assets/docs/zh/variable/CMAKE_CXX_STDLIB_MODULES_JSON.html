CMAKE_CXX_STDLIB_MODULES_JSON¶

变量说明

CMAKE_CXX_STDLIB_MODULES_JSON：版本 4.2 中添加。

版本 4.2 中添加。

该变量可用于设置 CMake 理解如何为当前 CXX 编译器构建 import std 目标所需的元数据文件的路径。

只有在编译器无法在没有此类帮助的情况下发现相关的模块元数据文件时，才应使用此变量。

CMAKE_CURRENT_SOURCE_DIR
