CMAKE_SUPPRESS_REGENERATION¶

3.12 版本新增。

如果 CMAKE_SUPPRESS_REGENERATION 为 OFF，这是默认值，那么 CMake 会添加一个特殊目标，所有其他目标都依赖于它，该目标会检查构建系统，并在目标规范源文件发生更改时，可以选择重新运行 CMake 以重新生成构建系统。

如果此变量在顶级 CMakeLists.txt 文件末尾评估为 ON，CMake 将不会将重新生成目标添加到构建系统中，也不会执行任何构建系统检查。
