CMAKE_FIND_LIBRARY_SUFFIXES¶

在查找库时要附加的后缀。

这指定了当 find_library() 命令查找库时，要添加到库名称后的后缀。在 Windows 系统上，这通常是 .lib 以及，取决于编译器，.dll.lib、.dll.a、.a（例如 rustc、GCC 或 Clang），因此当它尝试查找 foo 库时，它将查找 [<prefix>]foo[.dll].lib 和/或 [<prefix>]foo[.dll].a，具体取决于所使用的编译器以及在 CMAKE_FIND_LIBRARY_PREFIXES 中指定的 <prefix>。

CMake 在工具链检查期间通过调用 project() 或 enable_language() 命令自动设置此变量。
