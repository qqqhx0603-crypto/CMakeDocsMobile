CMAKE_FIND_PACKAGE_RESOLVE_SYMLINKS¶

变量说明

CMAKE_FIND_PACKAGE_RESOLVE_SYMLINKS：3.14 版新增。

3.14 版新增。

设置为 TRUE，以告知 find_package() 调用解析 <PackageName>_DIR 的值中的符号链接。

这在软件包搜索路径指向一个代理目录，其中包含指向实际软件包位置的符号链接时很有用。 默认情况下未启用此功能，因为在某些常见用例中也应保留符号链接。
