CMAKE_VS_WINRT_BY_DEFAULT¶

3.13 版本新增。

告知 VS 2010 及以上版本的 Visual Studio 生成器，目标平台默认启用 WinRT 编译，并且需要在省略 /ZW 或 VS_WINRT_COMPONENT 时显式禁用它（与存在这两个选项中的任何一个时启用它相反）

这使得 cmake 配置在 WinRT 方面在不同平台之间保持一致——如果您没有显式启用 WinRT 编译，它将被禁用（通过不启用它或显式禁用它）

注意：即使为项目显式启用，WinRT 编译始终为 C 语言源文件显式禁用。

此变量旨在由 工具链 文件 为此类平台设置。

CMAKE_VS_SDK_SOURCE_DIRECTORIES
