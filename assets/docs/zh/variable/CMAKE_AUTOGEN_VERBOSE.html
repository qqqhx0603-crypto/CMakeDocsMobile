CMAKE_AUTOGEN_VERBOSE¶

变量说明

CMAKE_AUTOGEN_VERBOSE：设置 AUTOMOC、AUTOUIC 和 AUTORCC 的详细程度。一个正整数值或一个真布尔值允许 AUTO* 生成器输出额外的处理信息。

3.13 版本新增。

设置 AUTOMOC、AUTOUIC 和 AUTORCC 的详细程度。一个正整数值或一个真布尔值允许 AUTO* 生成器输出额外的处理信息。

设置 CMAKE_AUTOGEN_VERBOSE 与在生成期间设置 VERBOSE 环境变量具有相同的效果（例如，通过调用 make VERBOSE=1）。额外的详细程度仅限于 AUTO* 生成器。

默认情况下 CMAKE_AUTOGEN_VERBOSE 未设置。
