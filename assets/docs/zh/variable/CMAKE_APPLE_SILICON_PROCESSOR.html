CMAKE_APPLE_SILICON_PROCESSOR¶

添加于 3.19.2 版本。

在运行 macOS（苹果桌面系统） 的 Apple Silicon 主机上，将此变量设置为告知 CMake 为 CMAKE_HOST_SYSTEM_PROCESSOR 使用哪种架构。该值必须是 arm64 或 x86_64。

此变量的值不应由项目代码修改。它旨在由用户作为缓存条目设置，例如通过 -DCMAKE_APPLE_SILICON_PROCESSOR=...。

另请参阅 CMAKE_APPLE_SILICON_PROCESSOR 环境变量。
