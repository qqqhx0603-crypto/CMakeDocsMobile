CMAKE_<LANG>_EXTENSIONS_DEFAULT¶

3.22 版本新增。

编译器默认扩展模式。当 <LANG>_EXTENSIONS 目标属性未设置时，用作默认值 CMAKE_<LANG>_EXTENSIONS (参见 CMP0128)。

此变量为只读。修改它会导致未定义行为。
