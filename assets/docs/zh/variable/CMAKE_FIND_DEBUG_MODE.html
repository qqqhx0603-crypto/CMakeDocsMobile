CMAKE_FIND_DEBUG_MODE¶

变量说明

CMAKE_FIND_DEBUG_MODE：在 3.17 版本中添加。

在 3.17 版本中添加。

将以下命令的额外查找调用信息打印到标准错误输出

  • find_program()

  • find_library()

  • find_file()

  • find_path()

  • find_package()

输出设计用于人工阅读，不适合解析。启用此变量等同于使用 cmake --debug-find，并且增加了仅对部分查找调用启用调试的能力。

set(CMAKE_FIND_DEBUG_MODE TRUE)
find_program(...)
set(CMAKE_FIND_DEBUG_MODE FALSE)

代码说明

  • set(CMAKE_FIND_DEBUG_MODE TRUE)：调用 set() 命令，把变量 CMAKE_FIND_DEBUG_MODE 设为给定的值；变量说明：CMAKE_FIND_DEBUG_MODE：CMake 调试配置相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

  • find_program(...)：调用 find_program() 命令，查找可执行程序。

  • set(CMAKE_FIND_DEBUG_MODE FALSE)：调用 set() 命令，把变量 CMAKE_FIND_DEBUG_MODE 设为给定的值；变量说明：CMAKE_FIND_DEBUG_MODE：CMake 调试配置相关变量，其精确取值和作用域以当前页面或脚本上下文为准。

默认情况下未设置。
