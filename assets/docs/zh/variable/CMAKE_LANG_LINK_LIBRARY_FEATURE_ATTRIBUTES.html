CMAKE_<LANG>_LINK_LIBRARY_<FEATURE>_ATTRIBUTES¶

3.30 版本新增。

此变量定义了使用链接语言 <LANG> 链接时，指定链接库 <FEATURE> 的语义。如果为相同的 <FEATURE> 也定义了 CMAKE_LINK_LIBRARY_<FEATURE>_ATTRIBUTES，则它具有优先权，但除此之外具有类似的效果。有关更多详细信息，请参阅 CMAKE_LINK_LIBRARY_<FEATURE>_ATTRIBUTES。

CMAKE_<LANG>_LINK_GROUP_USING_<FEATURE>_SUPPORTED

下一主题
