CMAKE_<LANG>_HOST_COMPILER_VERSION¶

在版本 3.31 中添加。

当 <LANG> 为 CUDA 或 HIP 且 CMAKE_<LANG>_COMPILER_ID 为 NVIDIA 时，此变量可用。它包含由 nvcc 调用（默认或由 CMAKE_<LANG>_HOST_COMPILER 指定）的主机编译器的版本，格式与 CMAKE_<LANG>_COMPILER_VERSION 相同。
