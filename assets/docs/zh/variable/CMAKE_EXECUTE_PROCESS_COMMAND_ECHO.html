CMAKE_EXECUTE_PROCESS_COMMAND_ECHO¶

变量说明

CMAKE_EXECUTE_PROCESS_COMMAND_ECHO：如果将此变量设置为 STDERR、STDOUT 或 NONE，则在 execute_process() 调用中，命令将被打印到 stderr 或 stdout，或者根本不打印。

版本 3.15 新增。
