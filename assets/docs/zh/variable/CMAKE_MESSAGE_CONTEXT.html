CMAKE_MESSAGE_CONTEXT¶

在 3.17 版本中添加。

当通过 cmake --log-context 命令行选项或 CMAKE_MESSAGE_CONTEXT_SHOW 变量启用时，message() 命令会将 CMAKE_MESSAGE_CONTEXT 列表转换为用点分隔的字符串，并用方括号括起来，然后将其作为前缀添加到日志级别为 NOTICE 及以下的每行消息中。

为了使日志上下文有效工作，项目通常应该 APPEND 和 POP_BACK 项目到 CMAKE_MESSAGE_CONTEXT 的当前值，而不是替换它。项目不应假定源树顶部的消息上下文为空，因为在某些情况下上下文可能已经设置（例如，分层项目）。

警告

有效的上下文名称仅限于可以用作 CMake 变量名称的任何内容。所有以下划线或字符串 cmake_ 开头的名称也保留供 CMake 使用，不应由项目使用。

示例

function(bar)
  list(APPEND CMAKE_MESSAGE_CONTEXT "bar")
  message(VERBOSE "bar VERBOSE message")
endfunction()

function(baz)
  list(APPEND CMAKE_MESSAGE_CONTEXT "baz")
  message(DEBUG "baz DEBUG message")
endfunction()

function(foo)
  list(APPEND CMAKE_MESSAGE_CONTEXT "foo")
  bar()
  message(TRACE "foo TRACE message")
  baz()
endfunction()

list(APPEND CMAKE_MESSAGE_CONTEXT "top")

message(VERBOSE "Before `foo`")
foo()
message(VERBOSE "After `foo`")

list(POP_BACK CMAKE_MESSAGE_CONTEXT)

代码说明

  • function(bar) list(APPEND CMAKE_MESSAGE_CONTEXT "bar") message(VERBOSE "bar VERBOSE message") endfunction() function(baz) list(APPEND CMAKE_MESSAGE_CONTEXT "baz") message(DEBUG "baz DEBUG message") endfunction() function(foo) list(APPEND CMAKE_MESSAGE_CONTEXT "foo") bar() message(TRACE "foo TRACE message") baz() endfunction() list(APPEND CMAKE_MESSAGE_CONTEXT "top") message(VERBOSE "Before `foo`") foo() message(VERBOSE "After `foo`") list(POP_BACK CMAKE_MESSAGE_CONTEXT)：定义一个函数，并为其后的语句建立独立作用域。

结果如下

-- [top] Before `foo`
-- [top.foo.bar] bar VERBOSE message
-- [top.foo] foo TRACE message
-- [top.foo.baz] baz DEBUG message
-- [top] After `foo`

代码说明

  • -- [top] Before `foo` -- [top.foo.bar] bar VERBOSE message -- [top.foo] foo TRACE message -- [top.foo.baz] baz DEBUG message -- [top] After `foo`：这一行给出示例文本或输出，用来说明：-- [top] Before `foo` -- [top.foo.bar] bar VERBOSE message -- [top.foo] foo TRACE message -- [top.foo.baz] baz DEBUG message -- [top] After `foo`
