CMAKE_TRY_COMPILE_NO_PLATFORM_VARIABLES¶

变量说明

CMAKE_TRY_COMPILE_NO_PLATFORM_VARIABLES：在 3.24 版本中添加。

在 3.24 版本中添加。

设置为真值，以告知 try_compile() 命令不要将任何平台变量传递到测试项目中。

try_compile() 命令通常会将一些配置平台和工具链行为的 CMake 变量传递到测试项目。请参阅策略 CMP0137。可以将此变量设置为禁用该行为。
