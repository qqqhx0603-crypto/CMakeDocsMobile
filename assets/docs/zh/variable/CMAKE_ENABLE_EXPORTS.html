CMAKE_ENABLE_EXPORTS¶

变量说明

CMAKE_ENABLE_EXPORTS：自版本 4.3 起已弃用：此变量已被 CMAKE_EXECUTABLE_ENABLE_EXPORTS 和 CMAKE_SHARED_LIBRARY_ENABLE_EXPORTS 变量取代，这些变量自 CMake 3.27 起可用。它仅为与旧 CMake 代码的向后兼容性而提供，不应在新的项目中中使用。

3.4 版本新增。

自版本 4.3 起已弃用：此变量已被 CMAKE_EXECUTABLE_ENABLE_EXPORTS 和 CMAKE_SHARED_LIBRARY_ENABLE_EXPORTS 变量取代，这些变量自 CMake 3.27 起可用。它仅为与旧 CMake 代码的向后兼容性而提供，不应在新的项目中中使用。

指定可执行文件是否导出用于可加载模块的符号。

此变量用于初始化 ENABLE_EXPORTS 目标属性，用于通过对 add_executable() 命令创建的可执行目标。有关详细信息，请参阅属性文档。
