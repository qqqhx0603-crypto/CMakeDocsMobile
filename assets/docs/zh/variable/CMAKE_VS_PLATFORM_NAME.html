CMAKE_VS_PLATFORM_NAME¶

版本 3.1 中新增。

当前生成器使用的 Visual Studio 目标平台名称。

VS 8 及更高版本允许项目文件指定目标平台。CMake 在此变量中提供所选平台的名称。有关详细信息，请参阅 CMAKE_GENERATOR_PLATFORM 变量。

另请参阅 CMAKE_VS_PLATFORM_NAME_DEFAULT 变量。
