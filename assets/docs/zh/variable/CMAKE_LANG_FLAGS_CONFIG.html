CMAKE_<LANG>_FLAGS_<CONFIG>¶

变量说明

CMAKE_<LANG>_FLAGS_<CONFIG>：用于构建 <CONFIG> 配置的语言 <LANG> 的语言范围内的标志。这些标志将被传递给相应配置中编译器的所有调用。这包括驱动编译和驱动链接的调用。

用于构建 <CONFIG> 配置的语言 <LANG> 的语言范围内的标志。这些标志将被传递给相应配置中编译器的所有调用。这包括驱动编译和驱动链接的调用。

此变量中的标志将在 CMAKE_<LANG>_FLAGS 变量中的标志之后传递。在驱动编译的调用中，两个变量的标志都将在诸如 add_compile_options() 和 target_compile_options() 等命令添加的标志之前传递。在驱动链接的调用中，它们将在诸如 add_link_options() 和 target_link_options() 等命令添加的标志之前传递。
