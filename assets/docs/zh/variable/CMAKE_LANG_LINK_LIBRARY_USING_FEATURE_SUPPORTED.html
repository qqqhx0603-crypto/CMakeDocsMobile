CMAKE_<LANG>_LINK_LIBRARY_USING_<FEATURE>_SUPPORTED¶

在 3.24 版本中添加。

如果变量 CMAKE_<LANG>_LINK_LIBRARY_USING_<FEATURE> 定义的 <FEATURE>，对于链接器语言 <LANG> 得到支持，则设置为 TRUE。

注意

在更通用的变量 CMAKE_LINK_LIBRARY_USING_<FEATURE>_SUPPORTED 之前评估此变量。

CMAKE_<LANG>_LINK_LIBRARY_USING_<FEATURE>

下一主题

CMAKE_<LANG>_LINK_WHAT_YOU_USE_FLAG
