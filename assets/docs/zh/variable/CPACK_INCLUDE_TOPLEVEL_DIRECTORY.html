CPACK_INCLUDE_TOPLEVEL_DIRECTORY¶

变量说明

CPACK_INCLUDE_TOPLEVEL_DIRECTORY：一个布尔开关，用于包含/排除顶层目录。

一个布尔开关，用于包含/排除顶层目录。

在准备包时，CPack 会将项目安装在所谓的顶层目录中。其目的是在包中包含（设置为 1 或 ON 或 TRUE）顶层目录，或者不包含（设置为 0 或 OFF 或 FALSE）。

每个 CPack 生成器对此变量都有内置的默认值。例如，归档生成器（ZIP、TGZ、...）包含顶层目录，而 RPM 或 DEB 则不包含。用户可以通过设置此变量来覆盖默认值。

有一个类似的变量 CPACK_COMPONENT_INCLUDE_TOPLEVEL_DIRECTORY，可用于覆盖组件打包情况下的行为，由于历史（现在是向后兼容）原因，该行为可能具有不同的默认值。
