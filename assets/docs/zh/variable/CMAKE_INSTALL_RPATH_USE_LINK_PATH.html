CMAKE_INSTALL_RPATH_USE_LINK_PATH¶

变量说明

CMAKE_INSTALL_RPATH_USE_LINK_PATH：将路径添加到链接器搜索路径和安装后的 rpath。

将路径添加到链接器搜索路径和安装后的 rpath。

CMAKE_INSTALL_RPATH_USE_LINK_PATH 是一个布尔值，如果设置为 True，则会将链接器搜索路径中或包含链接库文件的项目外部任何目录附加到已安装二进制文件的运行时搜索路径 (rpath)。这些目录将附加在 INSTALL_RPATH 目标属性的值之后。

此变量用于为所有目标初始化目标属性 INSTALL_RPATH_USE_LINK_PATH。
