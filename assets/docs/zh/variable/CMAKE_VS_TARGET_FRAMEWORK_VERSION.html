CMAKE_VS_TARGET_FRAMEWORK_VERSION¶

3.22 版本新增。

Visual Studio（集成开发环境） 目标框架版本。

在某些情况下，Visual Studio（集成开发环境） 生成器 可能会在 .csproj 文件中使用 TargetFrameworkVersion 设置的显式值。CMake 将选择的值存储在此变量中。

请参阅 CMAKE_DOTNET_TARGET_FRAMEWORK_VERSION 变量和 DOTNET_TARGET_FRAMEWORK_VERSION 目标属性，以指定项目目标的自定义 TargetFrameworkVersion 值。

另请参阅 CMAKE_VS_TARGET_FRAMEWORK_IDENTIFIER 和 CMAKE_VS_TARGET_FRAMEWORK_TARGETS_VERSION。
