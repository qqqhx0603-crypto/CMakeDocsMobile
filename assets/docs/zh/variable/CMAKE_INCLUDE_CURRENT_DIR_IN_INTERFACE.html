CMAKE_INCLUDE_CURRENT_DIR_IN_INTERFACE¶

自动将当前源码目录和构建目录添加到 INTERFACE_INCLUDE_DIRECTORIES 目标属性。

如果此变量被启用，CMake 会自动为每个共享库目标、静态库目标、模块目标和可执行文件目标，将 CMAKE_CURRENT_SOURCE_DIR 和 CMAKE_CURRENT_BINARY_DIR 添加到 INTERFACE_INCLUDE_DIRECTORIES 目标属性。默认情况下 CMAKE_INCLUDE_CURRENT_DIR_IN_INTERFACE 为 OFF。
