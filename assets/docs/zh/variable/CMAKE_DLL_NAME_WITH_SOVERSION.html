CMAKE_DLL_NAME_WITH_SOVERSION¶

变量说明

CMAKE_DLL_NAME_WITH_SOVERSION：在 3.27 版本中新增。

在 3.27 版本中新增。

此变量用于初始化 Windows（微软桌面系统） 平台共享库目标的 DLL_NAME_WITH_SOVERSION 属性，当设置 WIN32 变量时生效。

有关更多信息，请参阅此目标属性。

请注意，如果使用 CMAKE_PLATFORM_NO_VERSIONED_SONAME 变量全局禁用了版本化文件名，则设置此变量将不会产生任何影响。
