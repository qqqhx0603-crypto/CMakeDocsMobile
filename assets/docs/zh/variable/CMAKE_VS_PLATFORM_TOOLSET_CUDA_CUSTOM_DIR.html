CMAKE_VS_PLATFORM_TOOLSET_CUDA_CUSTOM_DIR¶

3.16 版新增。

指向独立的 NVIDIA CUDA 工具包（例如，从安装程序中提取）。

对于 VS 2010 及更高版本的 Visual Studio 生成器，支持使用独立的（未安装的）NVIDIA CUDA 工具包。可以通过 CMAKE_GENERATOR_TOOLSET 中的字段以 cuda=C:\path\to\cuda 的形式指定路径。给定的目录至少应包含 nvcc 编译器在路径 .\bin 中，并且必须在路径 .\extras\visual_studio_integration\ MSBuildExtensions\ 中提供 Visual Studio 集成文件。可以通过使用 7zip 打开安装程序或复制正在运行的安装程序提取的文件来创建独立的 CUDA 工具包目录。如果没有指定指向独立 CUDA 工具包的路径，则该值可能为空。
