CMAKE_VS_INTEL_Fortran_PROJECT_VERSION¶

当为 Visual Studio 14 2015 或更高版本生成，并且安装了 Intel Fortran 插件时，此变量指定 .vfproj 项目文件格式版本。 此变量供 CMake 内部使用，不应在项目代码中使用。

上一主题

CMAKE_NOT_USING_CONFIG_FLAGS

下一主题

CMAKE_CFG_INTDIR
