CMAKE_<LANG>_STANDARD¶

变体有

  • CMAKE_C_STANDARD

  • CMAKE_CXX_STANDARD

  • CMAKE_CUDA_STANDARD

  • CMAKE_HIP_STANDARD

  • CMAKE_OBJC_STANDARD

  • CMAKE_OBJCXX_STANDARD

当创建目标时，如果设置了<LANG>_STANDARD目标属性的默认值。

有关支持的 CMake 版本，请参阅相应的页面。

有关编译功能和支持的编译器列表，请参阅 cmake-compile-features(7) 手册。
