PROJECT_HOMEPAGE_URL¶

3.12 版本新增。

项目的首页 URL。

这是在当前目录范围或更高范围中最近调用的 project() 命令提供的首页 URL。要获取顶级项目的首页 URL，请参阅 CMAKE_PROJECT_HOMEPAGE_URL 变量。
