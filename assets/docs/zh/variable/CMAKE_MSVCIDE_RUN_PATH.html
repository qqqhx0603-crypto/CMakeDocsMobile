CMAKE_MSVCIDE_RUN_PATH¶

3.10 版本新增。

在使用 add_custom_command() 或 add_custom_target() 时，当使用 Visual Studio（集成开发环境） 生成器 时，应该使用的额外的 PATH 位置。 这允许运行命令和使用 IDE 环境未知的 dll。

如果未显式设置，则该值将由环境变量 CMAKE_MSVCIDE_RUN_PATH 初始化（如果已设置），否则保持为空。

上一主题

CMAKE_MSVC_RUNTIME_LIBRARY

下一主题

CMAKE_NINJA_OUTPUT_PATH_PREFIX
