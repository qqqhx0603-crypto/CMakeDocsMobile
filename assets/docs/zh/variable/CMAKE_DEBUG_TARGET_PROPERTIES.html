CMAKE_DEBUG_TARGET_PROPERTIES¶

启用目标属性的跟踪输出。

该变量可以填充一个属性列表，以便在评估目标属性时生成调试输出。目前，它只能在评估

  • AUTOUIC_OPTIONS

  • COMPILE_DEFINITIONS

  • COMPILE_FEATURES

  • COMPILE_OPTIONS

  • INCLUDE_DIRECTORIES

  • LINK_DIRECTORIES

  • LINK_OPTIONS

  • POSITION_INDEPENDENT_CODE

  • SOURCES

目标属性以及在 COMPATIBLE_INTERFACE_STRING 和其他 COMPATIBLE_INTERFACE_ 属性中列出的任何其他属性。它会输出目标属性中每个条目的来源。默认情况下未设置。
