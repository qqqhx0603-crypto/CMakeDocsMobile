CMAKE_TLS_VERSION¶

变量说明

CMAKE_TLS_VERSION：指定 file(DOWNLOAD) 和 file(UPLOAD) 命令的 TLS_VERSION 选项的默认值。 如果未设置此变量，则命令会检查 CMAKE_TLS_VERSION 环境变量。 如果两者均未设置，则默认值为 TLS 1.2。

3.30 版本新增。

指定 file(DOWNLOAD) 和 file(UPLOAD) 命令的 TLS_VERSION 选项的默认值。 如果未设置此变量，则命令会检查 CMAKE_TLS_VERSION 环境变量。 如果两者均未设置，则默认值为 TLS 1.2。

版本 3.31 中有更改： 默认值为 TLS 1.2。此前，默认情况下没有强制执行最低版本。

该值可以是以下之一：

  • 1.0

  • 1.1

  • 1.2

  • 1.3

此变量还被 ExternalProject 和 FetchContent 模块用于对 file(DOWNLOAD) 和 git clone 进行内部调用。
