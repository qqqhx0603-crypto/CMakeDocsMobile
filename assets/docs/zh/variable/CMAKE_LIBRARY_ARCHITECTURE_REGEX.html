CMAKE_LIBRARY_ARCHITECTURE_REGEX¶

用于匹配可能的目标架构库目录名称的正则表达式。

这用于通过匹配 <arch> 名称，检测 CMAKE_<LANG>_LIBRARY_ARCHITECTURE，从而从隐式链接器搜索路径中获取信息。
