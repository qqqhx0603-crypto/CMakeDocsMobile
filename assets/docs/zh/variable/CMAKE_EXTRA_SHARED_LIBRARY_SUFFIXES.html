CMAKE_EXTRA_SHARED_LIBRARY_SUFFIXES¶

变量说明

CMAKE_EXTRA_SHARED_LIBRARY_SUFFIXES：共享库的附加后缀。

共享库的附加后缀。

除了由 CMAKE_SHARED_LIBRARY_SUFFIX 指定之外，共享库的其他扩展名。CMake 使用此变量来识别目标链接的库分析期间的外部共享库文件。

上一主题

CMAKE_EXECUTABLE_SUFFIX_<LANG>
