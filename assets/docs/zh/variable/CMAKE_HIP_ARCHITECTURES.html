CMAKE_HIP_ARCHITECTURES¶

变量说明

CMAKE_HIP_ARCHITECTURES：用于生成设备代码的 GPU 架构列表。架构名称的解释基于 CMAKE_HIP_PLATFORM。

3.21 版本新增。

用于生成设备代码的 GPU 架构列表。架构名称的解释基于 CMAKE_HIP_PLATFORM。

此变量基于 CMAKE_HIP_PLATFORM 的值进行初始化。

amd

如果可用，则使用 rocm_agent_enumerator 报告的架构，否则使用编译器选择的默认架构。

spirv

默认值为 OFF。HIP 包（例如 chipStar）通过其自身的配置处理目标定位。

此变量用于为所有目标初始化 HIP_ARCHITECTURES 属性。有关更多信息，请参阅目标属性。
