CMAKE_BINARY_DIR¶

构建树的顶层路径。

这是当前 CMake 构建树的顶层完整路径。对于就地构建，这将与 CMAKE_SOURCE_DIR 相同。

在 cmake -P 脚本模式下运行时，CMake 将变量 CMAKE_BINARY_DIR、CMAKE_SOURCE_DIR、CMAKE_CURRENT_BINARY_DIR 和 CMAKE_CURRENT_SOURCE_DIR 设置为当前工作目录。

修改 CMAKE_BINARY_DIR 的行为未定义。
