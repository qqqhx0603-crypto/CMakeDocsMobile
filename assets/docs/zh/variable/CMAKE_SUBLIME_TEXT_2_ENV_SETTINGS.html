CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS¶

变量说明

CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS：此变量包含一个环境变量列表，格式为 var=value 的标记列表。

版本 3.8 新增。

此变量包含一个环境变量列表，格式为 var=value 的标记列表。

示例

set(CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS
   "FOO=FOO1\;FOO2\;FOON"
   "BAR=BAR1\;BAR2\;BARN"
   "BAZ=BAZ1\;BAZ2\;BAZN"
   "FOOBAR=FOOBAR1\;FOOBAR2\;FOOBARN"
   "VALID="
   )

代码说明

  • set(CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS：这一行继续补充当前命令或表达式的内容；变量说明：CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • "FOO=FOO1\;FOO2\;FOON"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "BAR=BAR1\;BAR2\;BARN"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "BAZ=BAZ1\;BAZ2\;BAZN"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "FOOBAR=FOOBAR1\;FOOBAR2\;FOOBARN"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "VALID="：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • )：这一行继续补充当前命令或表达式的内容。

如果变量格式不正确，CMake 将会失败

set(CMAKE_SUBLIME_TEXT_2_ENV_SETTINGS

    )
