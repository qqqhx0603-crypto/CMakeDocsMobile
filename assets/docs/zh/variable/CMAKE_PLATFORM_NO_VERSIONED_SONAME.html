CMAKE_PLATFORM_NO_VERSIONED_SONAME¶

版本 3.1 中新增。

此变量用于全局控制是否应该为共享库使用 VERSION 和 SOVERSION 目标属性。当设置为 true 时，将禁用为每个共享库目标添加版本信息。

默认情况下，此变量仅在 CMake 知道需要它的平台上设置。在其他平台上，将为共享库使用指定的属性。

上一主题

CMAKE_PDB_OUTPUT_DIRECTORY_<CONFIG>

下一主题

CMAKE_POSITION_INDEPENDENT_CODE
