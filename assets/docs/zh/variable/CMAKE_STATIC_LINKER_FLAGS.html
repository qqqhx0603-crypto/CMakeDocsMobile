CMAKE_STATIC_LINKER_FLAGS¶

变量说明

CMAKE_STATIC_LINKER_FLAGS：用于创建静态库的标志。这些标志将在创建静态库时传递给归档工具。

用于创建静态库的标志。这些标志将在创建静态库时传递给归档工具。

另请参阅 CMAKE_STATIC_LINKER_FLAGS_<CONFIG>。

注意

静态库实际上并不链接。它们本质上是目标文件的归档。此变量名称中使用“链接器”一词是为了保持兼容性。
