CMAKE_TASKING_TOOLSET¶

变量说明

CMAKE_TASKING_TOOLSET：在 3.25 版本中新增。

在 3.25 版本中新增。

选择提供编译器的 Tasking 工具集

架构编译器由不同的工具链提供，这些工具链具有不兼容的版本方案。在 工具链文件 中设置此变量，以便 CMake 可以正确检测编译器特性。如果没有指定工具集，则假定为 Standalone。

由于不同的版本方案，编译器版本 (CMAKE_<LANG>_COMPILER_VERSION) 取决于所使用的工具集和架构。如果项目可以使用多个工具集或架构，则在与 CMAKE_<LANG>_COMPILER_VERSION 进行比较时，必须考虑指定的 CMAKE_TASKING_TOOLSET 和自动确定的 CMAKE_<LANG>_COMPILER_ARCHITECTURE_ID。

TriCore

编译器由 TriCore 工具集提供。

SmartCode

编译器由 SmartCode 工具集提供。

Standalone

编译器由独立工具集提供。

注意

对于 TriCore 架构，来自 TriCore 工具集的编译器被选为独立编译器。
