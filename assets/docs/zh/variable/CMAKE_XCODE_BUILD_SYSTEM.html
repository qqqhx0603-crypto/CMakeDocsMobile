CMAKE_XCODE_BUILD_SYSTEM¶

变量说明

CMAKE_XCODE_BUILD_SYSTEM：Xcode 构建系统选择。

3.19 版本新增。

Xcode 构建系统选择。

Xcode 生成器定义此变量，以指示将使用 Xcode 构建系统的哪个变体。该值是 Xcode 的版本，在该版本中，相应的构建系统首次成熟到足以被 CMake 使用。可能的值是

1

最初的 Xcode 构建系统。这是使用 Xcode 11.x 或更早版本时的默认设置，并支持到 Xcode 13.x。

12

Xcode 10 引入的 Xcode “新构建系统”。它在 Xcode 12 中变得足够成熟，可以被 CMake 使用。这是使用 Xcode 12.x 或更高版本时的默认设置。

CMAKE_XCODE_BUILD_SYSTEM 变量是信息性的，不应由项目代码修改。请参阅 工具集和构建系统选择 文档部分以选择 Xcode 构建系统。
