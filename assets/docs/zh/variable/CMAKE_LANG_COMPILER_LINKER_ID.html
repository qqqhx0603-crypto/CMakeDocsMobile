CMAKE_<LANG>_COMPILER_LINKER_ID¶

在版本 3.29 中添加。

链接器标识字符串。

一个简短的字符串，用于唯一标识链接器供应商。可能的值包括

值

名称

AppleClang

Apple Clang

LLD

LLVM LLD

GNU

GNU Binutils - ld 链接器 (也称为 bfd)

GNUgold

GNU Binutils - gold 链接器

MSVC

Microsoft Visual Studio

MOLD

mold: 一种现代链接器，或者在 Apple 平台上是 sold 链接器

AIX

AIX 系统链接器

Solaris

SunOS 系统链接器

此变量不保证对所有链接器或语言都定义。

CMAKE_<LANG>_COMPILER_LINKER_FRONTEND_VARIANT

下一主题

CMAKE_<LANG>_COMPILER_LINKER_VERSION
