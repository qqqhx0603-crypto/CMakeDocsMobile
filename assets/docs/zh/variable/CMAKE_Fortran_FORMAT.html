CMAKE_Fortran_FORMAT¶

变量说明

CMAKE_Fortran_FORMAT：设置为 FIXED 或 FREE 以指示 Fortran 源布局。

设置为 FIXED 或 FREE 以指示 Fortran 源布局。

此变量用于初始化所有目标的 Fortran_FORMAT 属性。有关更多信息，请参阅该目标属性。
