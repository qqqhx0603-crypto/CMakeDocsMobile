CMAKE_TLS_VERIFY¶

为 file(DOWNLOAD) 和 file(UPLOAD) 命令的 TLS_VERIFY 选项指定默认值。 如果未设置此变量，则命令会检查 CMAKE_TLS_VERIFY 环境变量。 如果两者均未设置，则默认值为 on。

版本 3.31 中已更改： 默认值为开启。先前，默认值为关闭。用户可以通过设置 CMAKE_TLS_VERIFY 环境变量为 0 来恢复旧的默认值。

此变量也由 ExternalProject 和 FetchContent 模块用于内部调用 file(DOWNLOAD)。

TLS 验证有助于确保连接到所需的服务器。 在下载已知内容时，还应使用文件哈希来验证它。

set(CMAKE_TLS_VERIFY TRUE)

代码说明

  • set(CMAKE_TLS_VERIFY TRUE)：调用 `set()` 命令，把变量 `CMAKE_TLS_VERIFY` 设为给定的值。
