CMAKE_<LANG>_PVS_STUDIO¶

4.3 版本新增。

当 <LANG>_PVS_STUDIO 目标属性的默认值，当 <LANG> 为 C 或 CXX 时。

此变量用于在创建每个目标时初始化该属性。例如

set(CMAKE_CXX_PVS_STUDIO pvs-studio-analyzer analyze -a "GA\;OP")
add_executable(foo foo.cxx)

代码说明

  • set(CMAKE_CXX_PVS_STUDIO pvs-studio-analyzer analyze -a "GA\;OP") add_executable(foo foo.cxx)：调用 `set()` 命令，把变量 `CMAKE_CXX_PVS_STUDIO` 设为给定的值。
