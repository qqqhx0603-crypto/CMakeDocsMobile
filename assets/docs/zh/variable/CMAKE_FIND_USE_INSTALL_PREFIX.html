CMAKE_FIND_USE_INSTALL_PREFIX¶

在 3.24 版本中添加。

控制以下命令的默认行为，即是否搜索 CMAKE_INSTALL_PREFIX 和 CMAKE_STAGING_PREFIX 变量中的位置。

  • find_program()

  • find_library()

  • find_file()

  • find_path()

  • find_package()

这在交叉编译环境中非常有用。

为了与 CMAKE_FIND_NO_INSTALL_PREFIX 保持向后兼容，查找命令的行为会根据此变量是否存在而改变。

CMAKE_FIND_USE_INSTALL_PREFIX

CMAKE_FIND_NO_INSTALL_PREFIX
