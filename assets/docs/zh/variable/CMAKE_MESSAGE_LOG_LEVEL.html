CMAKE_MESSAGE_LOG_LEVEL¶

变量说明

CMAKE_MESSAGE_LOG_LEVEL：在 3.17 版本中添加。

在 3.17 版本中添加。

当设置时，此变量指定由 message() 命令使用的日志级别。有效值与 --log-level cmake(1) 程序命令行选项相同。如果设置了此变量并且提供了 --log-level 命令行选项，则命令行选项优先。

使用此变量的主要优点是使日志级别在 CMake 运行之间保持不变。将其设置为缓存变量将确保后续 CMake 运行将继续使用所选的日志级别。

项目不应设置此变量，它旨在供用户使用，以便他们可以根据自己的需求控制日志级别。

3.25 版本新增： 请参阅 cmake_language() cmake_language 命令，以获取查询当前消息日志级别的方法。
