MSVC_VERSION¶

如果使用任何 Microsoft Visual C/C++，则为正在使用的版本。如果正在使用模拟 Visual C++ 的编译器，则此变量设置为工具集版本，如 _MSC_VER 预处理器定义所给出的模拟版本。
