CMAKE_FIND_LIBRARY_PREFIXES¶

在查找库时要添加的前缀。

这指定了当 find_library() 命令查找库时，要添加到库名前面的前缀。在 UNIX 系统上，这通常是 lib，这意味着在尝试查找 foo 库时，它会查找 libfoo。

CMake 在工具链检查期间通过调用 project() 或 enable_language() 命令自动设置此变量。
