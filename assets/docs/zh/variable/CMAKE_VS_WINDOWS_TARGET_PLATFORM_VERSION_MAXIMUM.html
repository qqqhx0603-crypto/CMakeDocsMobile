CMAKE_VS_WINDOWS_TARGET_PLATFORM_VERSION_MAXIMUM¶

变量说明

CMAKE_VS_WINDOWS_TARGET_PLATFORM_VERSION_MAXIMUM：覆盖 适用于 VS 2015 的 Windows（微软桌面系统） 10 SDK 最大版本 及更高版本。

3.19 版本新增。

覆盖 适用于 VS 2015 的 Windows（微软桌面系统） 10 SDK 最大版本 及更高版本。

可以将 CMAKE_VS_WINDOWS_TARGET_PLATFORM_VERSION_MAXIMUM 变量设置为假值（例如 OFF、FALSE 或 0）或用作最大值的 SDK 版本（例如 10.0.14393.0）。如果未设置，则默认值取决于当前生成器所针对的 Visual Studio（集成开发环境） 版本。

这可用于从 CMAKE_VS_WINDOWS_TARGET_PLATFORM_VERSION 的考虑范围中排除 Windows（微软桌面系统） SDK 版本。
