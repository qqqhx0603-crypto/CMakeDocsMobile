CMAKE_FASTBUILD_ENV_OVERRIDES¶

变量说明

CMAKE_FASTBUILD_ENV_OVERRIDES：版本 4.2 中添加。

版本 4.2 中添加。

允许覆盖写入 fbuild.bff 的捕获环境中的环境变量。

指定 CMake 样式的键值对列表。这些值将覆盖 FASTBuild 在工具执行期间（例如，编译器、链接器、资源编译器等）使用的环境块中的相应变量。

这对于确保工具依赖于环境变量时的一致行为特别有用（例如，覆盖 PATH 以控制 rc.exe 或 mt.exe 的工具解析）。

示例

set(CMAKE_FASTBUILD_ENV_OVERRIDES
    "PATH=C:/MyTools/bin"
    "TMP=C:/temp"
    "MY_CUSTOM_VAR=some_value"
)

代码说明

  • set(CMAKE_FASTBUILD_ENV_OVERRIDES：这一行继续补充当前命令或表达式的内容；变量说明：CMAKE_FASTBUILD_ENV_OVERRIDES：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • "PATH=C:/MyTools/bin"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "TMP=C:/temp"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • "MY_CUSTOM_VAR=some_value"：这一行继续补充当前命令或表达式的内容；运算符：= 表示赋值或键值关联。

  • )：这一行继续补充当前命令或表达式的内容。

注意

这仅影响 FASTBuild 生成的规则所看到的环境。它不会修改 CMake 本身运行的环境。

默认情况下为空（无覆盖）。
