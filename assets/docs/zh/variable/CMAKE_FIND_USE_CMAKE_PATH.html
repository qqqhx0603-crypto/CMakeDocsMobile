CMAKE_FIND_USE_CMAKE_PATH¶

变量说明

CMAKE_FIND_USE_CMAKE_PATH：3.16 版新增。

3.16 版新增。

控制以下命令在搜索由 CMake 特定缓存变量提供的路径时，默认行为。

  • find_program()

  • find_library()

  • find_file()

  • find_path()

  • find_package()

这在交叉编译环境中非常有用。

默认情况下，此变量未设置，这等同于其值为 TRUE。为上述命令提供的显式选项优先于此变量。

另请参阅 CMAKE_FIND_USE_CMAKE_ENVIRONMENT_PATH, CMAKE_FIND_USE_CMAKE_SYSTEM_PATH, CMAKE_FIND_USE_SYSTEM_ENVIRONMENT_PATH, CMAKE_FIND_USE_SYSTEM_PACKAGE_REGISTRY, CMAKE_FIND_USE_PACKAGE_REGISTRY, 和 CMAKE_FIND_USE_PACKAGE_ROOT_PATH 变量。
