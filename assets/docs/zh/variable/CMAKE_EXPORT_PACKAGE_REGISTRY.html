CMAKE_EXPORT_PACKAGE_REGISTRY¶

版本 3.15 新增。

当 CMP0090 设置为 NEW 时，启用 export(PACKAGE) 命令。

默认情况下，export(PACKAGE) 命令不执行任何操作。在某些情况下，希望写入用户包注册表，因此可以将 CMAKE_EXPORT_PACKAGE_REGISTRY 变量设置为启用它。

如果 CMP0090 未设置为 NEW，则此变量不起作用，而是由 CMAKE_EXPORT_NO_PACKAGE_REGISTRY 变量控制行为。

另请参阅 禁用包注册表。
