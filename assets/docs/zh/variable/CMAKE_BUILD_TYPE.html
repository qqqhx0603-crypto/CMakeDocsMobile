CMAKE_BUILD_TYPE¶

指定单配置生成器（例如 Makefile（构建脚本） 生成器 或 Ninja（构建工具））的构建类型。 典型值包括 Debug、Release、RelWithDebInfo 和 MinSizeRel，但也可以定义自定义构建类型。

当首次创建构建树时，此变量由第一个 project() 或 enable_language() 命令初始化。 如果设置了 CMAKE_BUILD_TYPE 环境变量，则使用其值。 否则，在启用语言时会选择工具链特定的默认值。 默认值通常是空字符串，但这通常不是理想的，并且通常更适合使用其他标准构建类型。

根据情况，此变量的值可能被视为区分大小写或不区分大小写。 请参阅 构建配置，了解关于此和其他相关主题的讨论。

对于多配置生成器，请参阅 CMAKE_CONFIGURATION_TYPES。
