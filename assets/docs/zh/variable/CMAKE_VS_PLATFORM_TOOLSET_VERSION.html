CMAKE_VS_PLATFORM_TOOLSET_VERSION¶

变量说明

CMAKE_VS_PLATFORM_TOOLSET_VERSION：Visual Studio（集成开发环境） 平台工具集版本。

3.12 版本新增。

Visual Studio（集成开发环境） 平台工具集版本。

对于 VS 2017 及更高版本的 Visual Studio（集成开发环境） 生成器，可以选择相同工具集的次要版本。工具集版本号可以通过 CMAKE_GENERATOR_TOOLSET 中的字段指定，形式为 version=14.11。如果未指定，CMake 将选择默认工具集。如果未选择次要版本并使用默认值，则该值可能为空。

如果该值不为空，则它是 MSBuild 在其 Microsoft.VCToolsVersion.*.props 文件名中使用的版本号。

3.19.7 版本新增：VS 16.9 的工具集也可以指定为 14.28.16.9，因为 VS 16.10 使用文件名 Microsoft.VCToolsVersion.14.28.16.9.props。

三组件 MSVC 工具集版本¶

3.19.7 版本新增。

可以将 version= 字段赋予一个三组件工具集版本，例如 14.28.29910，CMake 将将其转换为 MSBuild Microsoft.VCToolsVersion.*.props 文件使用的名称。这对于区分 VS 16.8 的 14.28.29333 工具集和 VS 16.9 的 14.28.29910 工具集很有用。它也与 vcvarsall 的 -vcvars_ver= 行为相匹配。
