CMAKE_VS_NUGET_PACKAGE_RESTORE¶

变量说明

CMAKE_VS_NUGET_PACKAGE_RESTORE：在版本 3.23 中添加。

在版本 3.23 中添加。

在使用 Visual Studio（集成开发环境） 生成器 时，此缓存变量控制着 msbuild 是否应该在构建之前自动尝试恢复 NuGet 包。NuGet 包可以使用 VS_PACKAGE_REFERENCES 属性在目标上定义。如果没有定义任何包引用，则此设置将不起作用。

命令行选项 --resolve-package-references 也可以用来替代地控制全局的解析行为。此选项将优先于缓存变量。

使用 DOTNET_SDK 的目标需要在构建之前运行恢复。禁用此选项可能会导致此类项目的构建失败。

此设置存储为缓存条目。默认值为 ON。

另请参阅 VS_PACKAGE_REFERENCES 属性。
