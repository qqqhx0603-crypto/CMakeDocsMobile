CMAKE_LINK_SEARCH_END_STATIC¶

变量说明

CMAKE_LINK_SEARCH_END_STATIC：结束一个链接行，以便使用静态系统库。

3.4 版本新增。

结束一个链接行，以便使用静态系统库。

一些链接器支持诸如 -Bstatic 和 -Bdynamic 之类的开关，以确定是为 -lXXX 选项使用静态库还是共享库。CMake 使用这些选项来设置库的链接类型，这些库的完整路径未知或（在某些情况下）位于平台的隐式链接目录中。默认情况下，CMake 会在库列表的末尾（如果需要）添加一个选项，以将链接器搜索类型设置回其起始类型。此属性无论其起始状态如何，都会将最终的链接器搜索类型切换为 -Bstatic。

此变量用于初始化目标属性 LINK_SEARCH_END_STATIC，适用于所有目标。如果设置，其值也将由 try_compile() 命令使用。

另请参阅 CMAKE_LINK_SEARCH_START_STATIC。
