CMAKE_ECLIPSE_GENERATE_SOURCE_PROJECT¶

变量说明

CMAKE_ECLIPSE_GENERATE_SOURCE_PROJECT：此缓存变量由 Eclipse 项目生成器使用。请参阅 cmake-generators(7)。

3.6 版本新增。

此缓存变量由 Eclipse 项目生成器使用。请参阅 cmake-generators(7)。

如果将此变量设置为 TRUE，Eclipse 项目生成器将在 CMAKE_SOURCE_DIR 中生成一个 Eclipse 项目。然后可以将此项目用于 Eclipse，例如用于版本控制功能。 CMAKE_ECLIPSE_GENERATE_SOURCE_PROJECT 默认值为 FALSE；因此，不会向源代码目录写入任何内容。
