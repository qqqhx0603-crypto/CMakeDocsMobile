Unix Makefiles（Make 构建脚本）¶

生成标准的 UNIX Makefile（Make 构建脚本） 文件。

在构建树中生成一个 UNIX Makefile（Make 构建脚本） 文件层级。使用任何标准的 UNIX 风格的 make 程序，通过 all 目标来构建项目，通过 install (或 install/strip) 目标来安装项目。

对于项目的每个子目录 sub/dir，都会创建一个 UNIX Makefile（Make 构建脚本） 文件，其中包含以下目标：

all

依赖于子目录所需的所有目标。

安装

运行子目录中的安装步骤（如果存在）。

install/strip

运行子目录中的安装步骤，然后执行 CMAKE_STRIP 命令（如果存在）。

变量 CMAKE_STRIP 将包含平台的 strip 工具，该工具会从生成的二进制文件中移除符号信息。

测试

运行子目录中的测试步骤（如果存在）。

package

运行子目录中的打包步骤（如果存在）。
