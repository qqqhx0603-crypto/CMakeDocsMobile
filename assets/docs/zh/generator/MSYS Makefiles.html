MSYS Makefiles¶

生成用于在 MSYS (Minimal SYStem) shell 下使用 make 的 Makefile。

在 MSYS shell 提示符下并使用 make 作为构建工具时使用此生成器。生成的 Makefile 使用 /bin/sh 作为 shell 来启动构建规则。它们与 Windows 命令提示符不兼容。

要在 Windows 命令提示符下构建，请使用 MinGW Makefiles 生成器。
