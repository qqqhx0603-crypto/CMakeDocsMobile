Kate¶

自 3.27 版本起已弃用： 对 Extra Generators（扩展生成器） 的支持已被弃用，将在未来的 CMake 版本中移除。IDE 可以使用 cmake-file-api(7) 来查看 CMake 生成的项目构建树。

生成 Kate 项目文件。

Kate 的项目文件将在顶级构建目录的顶层目录中创建。要在 Kate 中使用它，必须启用 Project 插件。通过在编辑器中打开 ProjectName.kateproject 文件来在 Kate 中加载项目文件。如果启用了 Kate Build 插件，则 CMake 生成的所有目标都可用于构建。

此“额外”生成器可以指定为

Kate - MinGW Makefile

使用 MinGW Makefiles 生成。

Kate - NMake Makefile

使用 NMake Makefiles 生成。

Kate - Ninja

使用 Ninja 生成。

Kate - Ninja 多配置

使用 Ninja 多配置 生成。

Kate - Unix Makefile

使用 Unix Makefiles 生成。
