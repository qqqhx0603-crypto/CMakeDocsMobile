Sublime Text 2¶

自 3.27 版本起已弃用： 对 Extra Generators 的支持已被弃用，将在未来的 CMake 版本中移除。IDE 可以使用 cmake-file-api(7) 来查看 CMake 生成的项目构建树。

生成 Sublime Text 2 项目文件。

Sublime Text 2 的项目文件将在顶层目录以及每个包含 CMakeLists.txt 文件且该文件包含 project() 调用的子目录中创建。 此外，Makefiles (或 build.ninja 文件) 将生成到构建树中。 适当的 make 程序可以通过默认的 all 目标构建项目。 此外还提供了一个 install 目标。

此“额外”生成器可以指定为

Sublime Text 2 - MinGW Makefiles

使用 MinGW Makefiles 生成。

Sublime Text 2 - NMake Makefiles

使用 NMake Makefiles 生成。

Sublime Text 2 - Ninja

使用 Ninja 生成。

Sublime Text 2 - Unix Makefiles

使用 Unix Makefiles 生成。
