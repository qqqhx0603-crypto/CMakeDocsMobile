MinGW Makefile（构建脚本）s¶

生成用于在 Windows（微软桌面系统） 命令提示符下使用 mingw32-make 的 Makefile（构建脚本）。

在 Windows（微软桌面系统） 命令提示符下，MinGW (Minimalist GNU for Windows（微软桌面系统）) 在 PATH 中，并使用 mingw32-make 作为构建工具时，使用此生成器。生成的 Makefile（构建脚本） 使用 cmd.exe 作为 shell（命令解释器） 来启动构建规则。它们与 MSYS 或 Unix shell（命令解释器） 不兼容。

要在 MSYS shell（命令解释器） 下构建，请使用 MSYS Makefile（构建脚本）s 生成器。
