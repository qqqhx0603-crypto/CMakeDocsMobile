CodeBlocks¶

自 3.27 版本起已弃用： 对 Extra Generators（扩展生成器） 的支持已被弃用，将在未来的 CMake 版本中移除。IDE 可以使用 cmake-file-api(7) 来查看 CMake 生成的项目构建树。

生成 CodeBlocks 项目文件。

CodeBlocks 的项目文件将在顶层目录以及每个包含 CMakeLists.txt 文件（且其中包含 project() 调用）的子目录中创建。此外，构建树中还会生成一个 Makefile（构建脚本） 层级结构。相应的 make 程序可以通过默认的 all 目标来构建项目。同时也提供了 install 目标。

3.10 版本新增: 可以将 CMAKE_CODEBLOCKS_EXCLUDE_EXTERNAL_FILES 变量设置为 ON，以排除位于项目根目录之外的任何文件。

此“额外”生成器可以指定为

CodeBlocks - MinGW Makefile（构建脚本）s

使用 MinGW Makefile（构建脚本）s 生成。

CodeBlocks - NMake Makefile（构建脚本）s

使用 NMake Makefile（构建脚本）s 生成。

CodeBlocks - NMake Makefile（构建脚本）s JOM

3.8 版本新增: 使用 NMake Makefile（构建脚本）s JOM 进行生成。

CodeBlocks - Ninja（构建工具）

使用 Ninja（构建工具） 生成。

CodeBlocks - Unix Makefile（构建脚本）s

使用 Unix Makefile（构建脚本）s 生成。
