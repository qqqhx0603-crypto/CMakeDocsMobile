CFLAGS¶

这是一个 CMake 环境变量。其初始值取自调用进程环境。

为编译 C 文件添加默认编译标志。

CMake 会将此环境变量的值与其工具链的内置默认标志结合使用，以初始化并存储 CMAKE_C_FLAGS 缓存条目。这发生在首次为 C 语言配置构建树时。对于任何配置运行（包括首次运行），如果 CMAKE_C_FLAGS 变量已经定义，则环境变量将被忽略。

另请参阅 CMAKE_C_FLAGS_INIT。
