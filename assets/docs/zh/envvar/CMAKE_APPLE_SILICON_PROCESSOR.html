CMAKE_APPLE_SILICON_PROCESSOR¶

添加于 3.19.2 版本。

这是一个 CMake 环境变量。其初始值取自调用进程环境。

在运行 macOS 的 Apple Silicon 主机上，设置此环境变量以告知 CMake 为 CMAKE_HOST_SYSTEM_PROCESSOR 使用哪种架构。该值必须是 arm64 或 x86_64。

如果已设置，CMAKE_APPLE_SILICON_PROCESSOR 变量将覆盖此环境变量。
