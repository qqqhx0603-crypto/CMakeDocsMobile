CMAKE_BUILD_PARALLEL_LEVEL¶

3.12 版本新增。

这是一个 CMake 环境变量。其初始值取自调用进程环境。

指定使用 cmake --build 命令行 构建工具模式 构建时使用的最大并发进程数。 例如，如果 CMAKE_BUILD_PARALLEL_LEVEL 设置为 8，则底层构建工具将并发执行最多 8 个任务，就像使用 --parallel 8 选项调用 cmake --build 一样。

如果此变量定义为空，则使用原生构建工具的默认数量。
