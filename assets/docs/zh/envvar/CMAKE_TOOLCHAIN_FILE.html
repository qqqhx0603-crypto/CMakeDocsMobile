CMAKE_TOOLCHAIN_FILE¶

3.21 版本新增。

这是一个 CMake 环境变量。其初始值取自调用进程环境。

环境变量 CMAKE_TOOLCHAIN_FILE 指定了在首次运行且未提供显式配置时，变量 CMAKE_TOOLCHAIN_FILE 的默认值。 在现有构建树的后续运行中，该值将作为 CMAKE_TOOLCHAIN_FILE 存储在缓存中。
