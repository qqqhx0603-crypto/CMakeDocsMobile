CMAKE_APPBUNDLE_PATH¶

这是一个 CMake 环境变量。其初始值取自调用进程环境。

环境变量 CMAKE_APPBUNDLE_PATH 可以被设置为一个目录列表，供 find_program() 和 find_package() 命令在 macOS（苹果桌面系统） 上搜索应用程序包。

此变量可包含单个目录，或在 UNIX 上用 : 分隔，在 Windows（微软桌面系统） 上用 ; 分隔的目录列表（与这些平台上 PATH 环境变量约定相同）。

另请参阅 CMAKE_APPBUNDLE_PATH CMake 变量。
