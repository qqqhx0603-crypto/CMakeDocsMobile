CTEST_USE_INSTRUMENTATION¶

Added in version 4.3.

这是一个 CMake 环境变量。其初始值取自调用进程环境。

将此环境变量设置为 1, True 或 ON 启用 插桩 功能，用于 CTest 的 仪表板客户端 模式。

上一主题

CTEST_PROGRESS_OUTPUT

下一主题

CTEST_USE_LAUNCHERS_DEFAULT
