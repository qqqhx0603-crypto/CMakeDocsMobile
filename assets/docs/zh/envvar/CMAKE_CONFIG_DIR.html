CMAKE_CONFIG_DIR¶

在版本 3.31 中添加。

这是一个 CMake 环境变量。其初始值取自调用进程环境。

为 cmake-file-api(7) 和 cmake-instrumentation(7) 查询指定 CMake 用户范围的配置目录。

如果未设置此环境变量，则默认的用户范围配置目录是平台特定的

  • Windows（微软桌面系统）: %LOCALAPPDATA%\CMake

  • macOS（苹果桌面系统）: 如果设置了 $XDG_CONFIG_HOME/CMake，则使用该目录，否则使用 $HOME/Library/Application Support/CMake

  • Linux/Other: 如果设置了 $XDG_CONFIG_HOME/cmake，则使用该目录，否则使用 $HOME/.config/cmake
