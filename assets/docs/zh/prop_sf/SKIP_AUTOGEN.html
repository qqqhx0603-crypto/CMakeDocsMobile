SKIP_AUTOGEN¶

版本 3.8 新增。

从 AUTOMOC、AUTOUIC 和 AUTORCC 处理中排除源文件（针对 Qt 项目）。

有关更精细的排除控制，请参阅 SKIP_AUTOMOC、SKIP_AUTOUIC 和 SKIP_AUTORCC。

示例¶

# ...
set_property(SOURCE file.h PROPERTY SKIP_AUTOGEN ON)
# ...
