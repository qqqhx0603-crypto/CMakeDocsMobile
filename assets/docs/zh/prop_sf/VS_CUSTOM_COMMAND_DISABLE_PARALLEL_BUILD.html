VS_CUSTOM_COMMAND_DISABLE_PARALLEL_BUILD¶

4.0 版本新增。

一个布尔属性，如果在 Visual Studio（集成开发环境） 中通过 add_custom_command() 构建源文件，并且它是自定义命令的 MAIN_DEPENDENCY 输入，则禁用并行构建。请参阅策略 CMP0147。
