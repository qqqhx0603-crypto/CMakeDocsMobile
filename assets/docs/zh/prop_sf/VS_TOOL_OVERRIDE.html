VS_TOOL_OVERRIDE¶

3.7 版本中新增。

在 Visual Studio（集成开发环境） 项目中覆盖源文件的 MSBuild 项目类型。

与 VS_SETTINGS 一起使用，此属性可用于配置自定义 MSBuild 任务的项目。

将项目类型设置为 None 将排除文件不参与构建。

版本 3.31 中更改：此属性现在对所有源文件类型都有效。 之前，它仅适用于 CMake 未知的源类型。
