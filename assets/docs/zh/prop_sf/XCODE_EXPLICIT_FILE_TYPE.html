XCODE_EXPLICIT_FILE_TYPE¶

版本 3.1 中新增。

设置 Xcode 生成器的 explicitFileType 属性，指向源文件。CMake 会根据文件扩展名计算默认值，但也可以使用此属性显式指定。

另请参阅 XCODE_LAST_KNOWN_FILE_TYPE。
