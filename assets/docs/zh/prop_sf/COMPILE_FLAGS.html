COMPILE_FLAGS¶

编译此源文件时要添加的额外标志。

属性 COMPILE_FLAGS 以字符串形式管理，它设置了用于编译此源文件时将添加到编译标志列表中的额外编译器标志。这些标志将在目标范围内的标志之后添加。

使用 COMPILE_DEFINITIONS 传递额外的预处理器定义。

属性 COMPILE_FLAGS 的内容可以使用带有 $<...> 语法的“生成器表达式”。请参阅 cmake-generator-expressions(7) 手册以获取可用的表达式。但是，Xcode 不支持每个配置的每个源设置，因此对于该生成器，依赖于构建配置的表达式不允许使用。

注意

此属性已被 COMPILE_OPTIONS 属性取代。
