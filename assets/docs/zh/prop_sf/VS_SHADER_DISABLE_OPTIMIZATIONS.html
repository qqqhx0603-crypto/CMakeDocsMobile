VS_SHADER_DISABLE_OPTIMIZATIONS¶

3.11 版本新增。

禁用 .hlsl 源代码文件的编译器优化。 这会将 -Od 标志添加到 FxCompiler 工具的命令行中。 将此属性的值指定为 true 以禁用编译器优化。
