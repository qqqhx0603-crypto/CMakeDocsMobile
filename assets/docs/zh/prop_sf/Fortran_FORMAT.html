Fortran_FORMAT¶

设置为 FIXED 或 FREE 以指示 Fortran 源布局。

此属性告知 CMake 给定的 Fortran 源代码文件使用的是固定格式还是自由格式。CMake 会将相应的格式标志传递给编译器。如果目标中的所有源代码文件都使用相同的格式，请考虑使用目标范围内的 Fortran_FORMAT 属性。

注意

对于某些编译器，NAG、PGI 和 Solaris Studio，将此设置为 OFF 将不会产生任何影响。
