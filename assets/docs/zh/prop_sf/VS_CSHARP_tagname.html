VS_CSHARP_<tagname>¶

版本 3.8 新增。

Visual Studio 和 CSharp 源代码文件特定的配置。

告诉 Visual Studio 生成器 在生成的 Visual Studio CSharp 项目中将源代码文件标签 <tagname> 设置为给定值。在其他生成器和语言上会被忽略。此属性可用于定义源代码文件之间的依赖关系或设置任何其他 Visual Studio 特定的参数。

示例¶

用法示例

set_source_files_properties(<filename>
         PROPERTIES
         VS_CSHARP_DependentUpon <other file>
         VS_CSHARP_SubType "Form")

另请参阅¶

  • 用于配置 CSharp/.NET 目标的 CSharpUtilities 模块。
