AUTOUIC_OPTIONS¶

使用 AUTOUIC 时，uic 的附加选项

此属性保存将在构建过程中通过 AUTOUIC 执行 uic 时使用的额外命令行选项，即它等效于 qt4_wrap_ui() 宏的可选 OPTIONS 参数。

默认情况下为空。

在 .ui 源代码文件上设置的选项可能会覆盖在目标上设置的 AUTOUIC_OPTIONS。

示例¶

# ...
set_property(SOURCE widget.ui PROPERTY AUTOUIC_OPTIONS "--no-protection")
# ...

代码说明

  • # ... set_property(SOURCE widget.ui PROPERTY AUTOUIC_OPTIONS "--no-protection") # ...：这是注释，用来说明：... set_property(SOURCE widget.ui PROPERTY AUTOUIC_OPTIONS "--no-protection") # ...。
