OBJECT_DEPENDS¶

编译后的目标文件所依赖的附加文件。

指定一个分号分隔的列表，其中包含任何从该源文件编译的目标文件所依赖的文件的完整路径。在Makefile（Make 构建脚本） 生成器和Ninja生成器上，如果任何命名文件比目标文件更新，则目标文件将被重新编译。Visual Studio（集成开发环境） 生成器和Xcode生成器无法实现这种编译依赖关系。

不需要使用此属性来指定源文件对它包含的生成的头文件的依赖关系。虽然最初引入此属性的目的是为了这个目的，但现在不再需要了。如果生成的头文件是由与源文件相同的目标中的自定义命令创建的，则自动依赖项扫描过程将识别该依赖关系。如果生成的头文件是由另一个目标创建的，则应使用add_dependencies()命令创建目标间的依赖关系（如果由于链接关系而不存在）。
