SYSTEM¶

在 3.25 版本中新增。

此目录属性用于初始化在该目录中创建的非导入目标的 SYSTEM 目标属性。当 add_subdirectory() 和 FetchContent_Declare() 在使用 SYSTEM 选项作为这些命令的参数时，它会被设置为 true。
