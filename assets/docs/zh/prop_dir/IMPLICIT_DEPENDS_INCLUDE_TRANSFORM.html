IMPLICIT_DEPENDS_INCLUDE_TRANSFORM¶

指定目录中依赖项的 #include 行转换。

此属性指定在 C 和 C++ 源代码的隐式依赖项扫描期间转换类似宏的 #include 行的规则。规则列表必须用分号分隔，每个条目的格式为 A_MACRO(%)=value-with-%（% 必须是字面量）。在依赖项扫描期间，#include 行上出现的 A_MACRO(...) 将被给定值替换，并将宏参数替换为 %。例如，条目

MYDIR(%)=<mydir/%>

将把以下形式的行转换为

#include MYDIR(myheader.h)

到

#include <mydir/myheader.h>

允许跟踪依赖项。

此属性适用于目录中所有目标中的源文件。该属性值由其父目录中的值初始化。
