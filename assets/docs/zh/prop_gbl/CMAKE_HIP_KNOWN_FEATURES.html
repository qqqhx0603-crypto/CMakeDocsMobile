CMAKE_HIP_KNOWN_FEATURES¶

3.30 版本新增。

此 CMake 版本所知的 HIP 特性列表。

此全局属性中列出的特性可能对 HIP 编译器可用。如果 HIP 编译器可用该特性，它将列在 CMAKE_HIP_COMPILE_FEATURES 变量中。

此处列出的功能可与 target_compile_features() 命令一起使用。有关编译功能和支持的编译器列表的信息，请参阅 cmake-compile-features(7) 手册。

此 CMake 版本所知的特性是

hip_std_98

编译器模式至少为 HIP/C++ 98。

hip_std_11

编译器模式至少为 HIP/C++ 11。

hip_std_14

编译器模式至少为 HIP/C++ 14。

hip_std_17

编译器模式至少为 HIP/C++ 17。

hip_std_20

编译器模式至少为 HIP/C++ 20。

hip_std_23

编译器模式至少为 HIP/C++ 23。

hip_std_26

3.30 版本新增。

编译器模式至少为 HIP/C++ 26。

注意

如果编译器的默认标准级别至少等于所请求功能的标准级别，CMake 可能会省略 -std= 标志。如果编译器的默认扩展模式与 <LANG>_EXTENSIONS 目标属性不匹配，或者 <LANG>_STANDARD 目标属性已设置，则仍可能添加该标志。
