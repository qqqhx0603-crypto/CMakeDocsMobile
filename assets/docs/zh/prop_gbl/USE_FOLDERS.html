USE_FOLDERS¶

控制是否使用 FOLDER 目标属性来将目标组织到文件夹中。顶级 CMakeLists.txt 文件末尾的 USE_FOLDERS 的值决定了行为。

版本 3.26 中更改： CMake 将此属性视为 ON 默认值。请参阅策略 CMP0143。

并非所有 CMake 生成器都支持记录目标的文件夹详细信息。Xcode 和 Visual Studio 生成器是支持此功能的生成器的示例。 同样，并非所有 IDE 都支持使用文件夹层次结构呈现目标，即使使用的 CMake 生成器提供了必要的信息。
