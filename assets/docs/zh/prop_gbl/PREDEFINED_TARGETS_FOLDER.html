预定义目标文件夹¶

CMake 自动添加的目标的 FOLDER 名称。

如果未设置，CMake 将使用 "CMakePredefinedTargets" 作为此属性的默认值。诸如 INSTALL、PACKAGE 和 RUN_TESTS 之类的目标将被组织到此 FOLDER 中。 另请参阅 FOLDER 目标属性的文档。

PACKAGES_NOT_FOUND

下一主题

PROPAGATE_TOP_LEVEL_INCLUDES_TO_TRY_COMPILE
