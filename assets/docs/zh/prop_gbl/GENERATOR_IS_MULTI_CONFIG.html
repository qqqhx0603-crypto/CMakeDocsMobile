GENERATOR_IS_MULTI_CONFIG¶

版本 3.9 中添加。

只读属性，对于多配置生成器为真。

当使用多配置生成器（例如

  • Ninja Multi-Config

  • Visual Studio 生成器

  • Xcode

多配置生成器使用 CMAKE_CONFIGURATION_TYPES 作为配置集，并忽略 CMAKE_BUILD_TYPE。

FIND_LIBRARY_USE_OPENBSD_VERSIONING
