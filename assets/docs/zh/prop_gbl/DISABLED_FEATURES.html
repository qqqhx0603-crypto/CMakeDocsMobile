DISABLED_FEATURES¶

CMake 运行期间禁用的功能列表。

CMake 运行期间禁用的功能列表。默认情况下，它包含所有未找到的包的名称。这使用 <NAME>_FOUND 变量确定。搜索时指定 QUIET 的包不会被列出。项目可以将自己的功能添加到此列表中。此属性由 FeatureSummary 模块中的宏使用。
