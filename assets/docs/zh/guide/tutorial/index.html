CMake 教程¶

简介¶

本 CMake 教程提供了分步指南，涵盖了 CMake 有助于解决的常见构建系统问题。通过示例项目了解各项主题如何协同工作，将非常有帮助。

步骤¶

教程源代码示例可在 此压缩包 中获取。每个步骤都有其对应的子目录，其中包含可用作起点的代码。教程示例是循序渐进的，因此每个步骤都提供了前一个步骤的完整解决方案。
