AUTORCC_OPTIONS¶

在使用 AUTORCC 时，rcc 的附加选项

此属性保存将在构建过程中通过 AUTORCC 执行 rcc 时使用的附加命令行选项，即它等效于 qt4_add_resources() 宏的可选 OPTIONS 参数。

当创建目标时，如果设置了 CMAKE_AUTORCC_OPTIONS 变量，则此属性将由其值初始化；否则，将初始化为空字符串。

目标上设置的选项可以被 .qrc 源文件上设置的 AUTORCC_OPTIONS 覆盖。

有关将 CMake 与 Qt 结合使用的更多信息，请参阅 cmake-qt(7) 手册。

示例¶

# ...
set_property(TARGET tgt PROPERTY AUTORCC_OPTIONS "--compress;9")
# ...
