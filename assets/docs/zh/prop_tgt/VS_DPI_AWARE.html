VS_DPI_AWARE¶

3.16 版新增。

在 Visual Studio（集成开发环境） 目标项目属性中设置清单工具 -> 输入和输出 -> DPI 感知。

有效值是 PerMonitor、ON 或 OFF。

例如

add_executable(myproject myproject.cpp)
set_property(TARGET myproject PROPERTY VS_DPI_AWARE "PerMonitor")

代码说明

  • add_executable(myproject myproject.cpp) set_property(TARGET myproject PROPERTY VS_DPI_AWARE "PerMonitor")：调用 `add_executable()` 命令，定义一个可执行目标。
