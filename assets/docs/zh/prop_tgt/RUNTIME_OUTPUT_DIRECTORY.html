RUNTIME_OUTPUT_DIRECTORY¶

用于构建 RUNTIME 目标文件的输出目录。

此属性指定应将运行时目标文件构建到哪个目录中。 属性值可以使用 generator expressions。 多配置生成器（Visual Studio（集成开发环境）, Xcode, Ninja（构建工具） Multi-Config）除非使用了生成器表达式，否则会将每个配置的子目录附加到指定的目录。

如果目标创建时设置了该变量，则此属性由 CMAKE_RUNTIME_OUTPUT_DIRECTORY 变量的值初始化。

另请参阅 RUNTIME_OUTPUT_DIRECTORY_<CONFIG> 目标属性。
