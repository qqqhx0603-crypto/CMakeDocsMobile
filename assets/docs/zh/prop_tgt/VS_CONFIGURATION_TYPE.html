VS_CONFIGURATION_TYPE¶

3.6 版本新增。

Visual Studio 项目配置类型。

设置生成的 Visual Studio 项目的 ConfigurationType 属性。 属性值可以使用 生成器 表达式。 如果设置了此属性，它将覆盖基于目标类型（例如 StaticLibrary、Application，...）的默认设置。

支持 Visual Studio 生成器，适用于 VS 2010 及更高版本。
