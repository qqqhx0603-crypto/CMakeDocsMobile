AUTOGEN_BUILD_DIR¶

版本 3.9 中添加。

用于 AUTOMOC、AUTOUIC 和 AUTORCC 为目标生成文件的目录。

该目录按需创建，并自动添加到 ADDITIONAL_CLEAN_FILES 目标属性。

如果未设置或为空，则使用目录 <dir>/<target-name>_autogen，其中 <dir> 是 CMAKE_CURRENT_BINARY_DIR，<target-name> 是 NAME。

默认情况下 AUTOGEN_BUILD_DIR 未设置。

有关将 CMake 与 Qt 结合使用的更多信息，请参阅 cmake-qt(7) 手册。
