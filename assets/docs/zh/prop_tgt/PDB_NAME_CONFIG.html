PDB_NAME_<CONFIG>¶

链接器为可执行文件或共享库目标生成时，每个配置的 MS 调试符号 .pdb 文件的输出名称。

这是 PDB_NAME 的特定配置版本。

4.1 版本新增：PDB_NAME_<CONFIG> 的内容可以使用 生成器 表达式。

注意

此属性不适用于静态库目标，因为不会调用链接器来生成它们，因此它们没有包含调试符号的链接器生成的 .pdb 文件。

链接器生成的程序数据库文件由 /pdb 链接器标志指定，与由 /Fd 编译器标志指定的编译器生成的程序数据库文件不同。 使用 COMPILE_PDB_NAME_<CONFIG> 属性来指定后者。
