AUTOMOC_INCLUDE_DIRECTORIES¶

在 4.1 版本中新增。

指定 AUTOMOC 要显式传递给 Qt 元对象编译器 (moc) 的零个或多个包含目录，而不是自动发现目标的包含目录。

当此属性在目标上设置时，AUTOMOC 只会使用此处列出的目录，并且会忽略任何其他包含路径。

此属性可以包含 生成器 表达式。

最终评估结果中的所有目录路径必须是绝对路径。如果在生成器表达式评估后存在任何非绝对路径，配置将因错误而失败。

另请参阅 CMAKE_AUTOMOC_INCLUDE_DIRECTORIES 变量，该变量可用于在所有目标上初始化此属性。

示例¶

add_library(myQtLib ...)
set_property(TARGET myQtLib PROPERTY AUTOMOC_INCLUDE_DIRECTORIES
  "${CMAKE_CURRENT_SOURCE_DIR}/include/myQtLib"
)
