<LANG>_INCLUDE_WHAT_YOU_USE¶

3.3 版本中新增。

此属性仅在 <LANG> 为 C 或 CXX 时实现。

指定一个 分号分隔的列表，其中包含 include-what-you-use 工具的命令行。 Makefile 生成器 和 Ninja 生成器将运行此工具，并与编译器一起报告，如果该工具报告任何问题，则会发出警告。

当创建目标时，如果设置了 CMAKE_<LANG>_INCLUDE_WHAT_YOU_USE 变量，则此属性将由该变量的值初始化。

版本 3.27 中添加: 此属性支持 生成器表达式。

3.27 版本新增：SKIP_LINTING 可以设置为单个源文件，以将其从 <LANG>_CPPLINT、<LANG>_CLANG_TIDY、<LANG>_CPPCHECK 和 <LANG>_INCLUDE_WHAT_YOU_USE 定义的 linting 工具中排除。 当 SKIP_LINTING 设置为源文件的 true 时，这些工具将不会在该特定文件上运行。
