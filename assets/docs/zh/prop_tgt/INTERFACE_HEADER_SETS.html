INTERFACE_HEADER_SETS¶

在版本 3.23 中添加。

目标 INTERFACE 和 PUBLIC 头文件集合的只读列表（即所有类型为 HEADERS 的文件集合）。这些头文件集合中的文件可以使用 install(TARGETS) 命令和 install(EXPORT) 和 export() 命令进行安装和导出。

可以使用 target_sources() 命令的 FILE_SET 选项，类型为 HEADERS，来定义头文件集合。

另请参阅 HEADER_SETS。
