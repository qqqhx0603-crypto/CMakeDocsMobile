INTERFACE_HEADER_SETS_TO_VERIFY¶

在 3.24 版本中添加。

用于指定目标中哪些 PUBLIC 和 INTERFACE 头文件集合应被验证为接口头文件。

此属性包含一个以分号分隔的头文件集合列表，如果 VERIFY_INTERFACE_HEADER_SETS 设置为 true，则应验证这些集合。如果列表为空，则验证所有 PUBLIC 和 INTERFACE 头文件集合。如果项目不希望验证目标上的任何接口头文件集合，请将 VERIFY_INTERFACE_HEADER_SETS 设置为 false。

另请参阅 PRIVATE_HEADER_SETS_TO_VERIFY。
