COMPILE_FEATURES¶

版本 3.1 中新增。

为该目标启用的编译器特性。

此属性中的特性列表是 CMAKE_C_COMPILE_FEATURES、CMAKE_CUDA_COMPILE_FEATURES 以及 CMAKE_CXX_COMPILE_FEATURES 变量中所列特性的子集。

COMPILE_FEATURES 的内容可以使用语法为 $<...> 的“生成器表达式”。有关可用表达式的信息，请参阅 cmake-generator-expressions(7) 手册。有关编译特性及支持的编译器列表的信息，请参阅 cmake-compile-features(7) 手册。
