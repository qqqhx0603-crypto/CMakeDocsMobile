VS_GLOBAL_ROOTNAMESPACE¶

Visual Studio 项目根命名空间。

设置生成的 Visual Studio 项目的 "RootNamespace" 属性。 只有设置此属性时才会生成该属性。

上一主题

VS_GLOBAL_PROJECT_TYPES

下一主题

VS_GLOBAL_<variable>
