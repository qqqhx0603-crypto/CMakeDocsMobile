POST_INSTALL_SCRIPT¶

已弃用的安装支持。

PRE_INSTALL_SCRIPT 和 POST_INSTALL_SCRIPT 属性是指定在安装目标之前和之后运行的 CMake 脚本的旧方法。它们仅在使用旧的 INSTALL_TARGETS 命令安装目标时使用。请使用 install() 命令代替。
