PRIVATE_HEADER_SETS_TO_VERIFY¶

4.3 版本新增。

用于指定目标中哪些 PUBLIC 和 PRIVATE 头文件集合应被验证为私有头文件。

此属性包含一个以分号分隔的头文件集合列表，这些集合应该在 VERIFY_PRIVATE_HEADER_SETS 设置为 true 时进行验证。如果列表为空，则所有 PUBLIC 和 PRIVATE 头文件集合都会被验证。如果项目不想对目标验证任何私有头文件集合，请将 VERIFY_PRIVATE_HEADER_SETS 设置为 false。

另请参阅 INTERFACE_HEADER_SETS_TO_VERIFY。
