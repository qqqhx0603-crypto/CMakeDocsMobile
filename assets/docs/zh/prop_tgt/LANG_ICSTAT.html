<LANG>_ICSTAT¶

在 4.1 版本中新增。

此属性仅当 <LANG> 为 C 或 CXX 时才支持。

指定一个 分号分隔的列表，其中包含用于 icstat 静态分析工具的命令行。 Makefile（Make 构建脚本） 生成器 和 Ninja（构建工具） 生成器 将与编译器一起运行 icstat 并报告任何问题。 如果 icstat 工具返回非零值，则构建将失败。

当创建目标时，如果设置了该变量，此属性将由 CMAKE_<LANG>_ICSTAT 变量的值初始化。 它还支持 生成器 表达式。

可以通过设置 SKIP_LINTING 源文件属性，来为单个源文件抑制此 lint。
