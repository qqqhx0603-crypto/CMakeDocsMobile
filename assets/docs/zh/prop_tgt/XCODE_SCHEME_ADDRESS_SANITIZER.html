XCODE_SCHEME_ADDRESS_SANITIZER¶

3.13 版本新增。

是否在生成的 Xcode 方案的“诊断”部分启用 Address Sanitizer。

此属性由变量 CMAKE_XCODE_SCHEME_ADDRESS_SANITIZER 的值进行初始化，如果该变量在创建目标时已设置。

请参阅 XCODE_GENERATE_SCHEME 目标属性文档，以查看所有与 Xcode 方案相关的属性。
