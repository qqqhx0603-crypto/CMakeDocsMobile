IMPORTED_LINK_INTERFACE_LIBRARIES_<CONFIG>¶

<CONFIG> 特定版本的 IMPORTED_LINK_INTERFACE_LIBRARIES。

配置名称对应于导入目标的项目的配置名称。如果设置了此属性，则会完全覆盖该命名配置的通用属性。

如果目标同时具有非空的 INTERFACE_LINK_LIBRARIES 属性，则此属性将被忽略。

此属性已弃用。请改用 INTERFACE_LINK_LIBRARIES。
