MACOSX_BUNDLE¶

在 macOS（苹果桌面系统） 或 iOS 上将可执行文件构建为应用程序包。

当此属性设置为 TRUE 时，在 macOS（苹果桌面系统） 或 iOS 上构建的可执行文件将被创建为应用程序包。这使其成为可以从 Finder 启动的 GUI 可执行文件。有关创建应用程序包的 Info.plist 文件的信息，请参阅 MACOSX_BUNDLE_INFO_PLIST 目标属性。如果变量 CMAKE_MACOSX_BUNDLE 在创建目标时被设置，则此属性将由该变量的值初始化。
