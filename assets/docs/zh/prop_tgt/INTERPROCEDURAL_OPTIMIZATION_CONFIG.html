INTERPROCEDURAL_OPTIMIZATION_<CONFIG>¶

目标的特定配置间的过程间优化。

这是 INTERPROCEDURAL_OPTIMIZATION 的特定配置版本。如果设置，此属性将覆盖命名配置的通用属性。

当创建目标时，如果设置了 CMAKE_INTERPROCEDURAL_OPTIMIZATION_<CONFIG> 变量，则此属性将被初始化。

另请参阅¶

  • 使用 CheckIPOSupported 模块来检查编译器是否支持过程间优化，然后再启用此目标属性。
