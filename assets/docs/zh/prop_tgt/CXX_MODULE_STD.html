CXX_MODULE_STD¶

3.30 版本新增。

CXX_MODULE_STD 是一个布尔值，指定目标是否可以使用 import std; 导入其 C++ 源代码。

注意

只有当通过 CMAKE_EXPERIMENTAL_CXX_IMPORT_STD 门控启用了对 import std; 的实验性支持时，此设置才有意义。

当此属性显式设置为 ON 时，CMake 将添加对提供 C++ 标准库模块的目标的依赖项，该模块适用于目标所应用的 C++ 标准。 此目标仅适用于当前构建，不会出现在目标的导出接口中。 在使用时，这些目标将根据需要重新应用。

注意

类似于 CXX_SCAN_FOR_MODULES 的引入，此属性默认情况下**不**为使用 cxx_std_23 的目标添加 import std 支持，除非明确请求，以保留使用 C++23 而不使用 import std 的项目的现有行为。 一旦该功能得到更广泛的应用，预计未来会有一项策略来改变默认行为。

此属性的值与禁用扫描的目标无关（请参阅 CXX_SCAN_FOR_MODULES）。 此外，此属性仅适用于使用 C++23 (cxx_std_23) 或更高版本的目标。

该属性支持 生成器 表达式，但是不允许依赖于配置、消耗目标或链接器语言的表达式。 目标是否使用 import std 不应依赖于此类内容，因为它应该是目标源代码的静态属性。

导出 C++ 模块源代码的目标将导出此属性的解析值。
