INTERFACE_PRECOMPILE_HEADERS¶

3.16 版新增。

列出需要预编译到使用该目标的消耗目标中的接口头文件。

目标可以通过填充此属性来发布头文件，供消耗目标进行预编译。target_precompile_headers() 命令会将传递给 PUBLIC 和 INTERFACE 关键字的值填充到此属性中。项目也可以直接获取和设置该属性。有关此属性在已安装或已导出目标中的正确使用指南，请参阅 target_precompile_headers() 中的讨论。

INTERFACE_PRECOMPILE_HEADERS 的内容可以使用语法为 $<...> 的“生成器表达式”。请参阅 cmake-generator-expressions(7) 手册以了解可用表达式。有关定义构建系统属性的更多信息，请参阅 cmake-buildsystem(7) 手册。
