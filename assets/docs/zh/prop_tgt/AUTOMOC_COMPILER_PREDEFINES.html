AUTOMOC_COMPILER_PREDEFINES¶

3.10 版本新增。

一个布尔值，由 AUTOMOC 使用来确定是否应该生成编译器预定义文件 moc_predefs.h。

CMake 会根据在 CMAKE_CXX_COMPILER_PREDEFINES_COMMAND 中定义的命令的输出，生成一个 moc_predefs.h 文件，当

  • AUTOMOC 启用时，

  • AUTOMOC_COMPILER_PREDEFINES 启用时，

  • CMAKE_CXX_COMPILER_PREDEFINES_COMMAND 不为空，并且

  • Qt 版本大于或等于 5.8 时。

生成的 moc_predefs.h 文件位于 AUTOGEN_BUILD_DIR 中，并作为 --include 选项的参数传递给 moc。

默认情况下，AUTOMOC_COMPILER_PREDEFINES 从 CMAKE_AUTOMOC_COMPILER_PREDEFINES 初始化，而 CMAKE_AUTOMOC_COMPILER_PREDEFINES 默认值为 ON。

有关将 CMake 与 Qt 结合使用的更多信息，请参阅 cmake-qt(7) 手册。
