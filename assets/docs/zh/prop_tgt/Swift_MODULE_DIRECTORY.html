Swift_MODULE_DIRECTORY¶

版本 3.15 新增。

指定目标提供的 Swift 模块的输出目录。

如果目标包含 Swift 源代码文件，则此属性指定模块将被放置的目录。当未设置此属性时，模块将被放置在与目标源代码目录对应的构建目录中。如果变量 CMAKE_Swift_MODULE_DIRECTORY 在创建目标时设置，则其值将用于初始化此属性。

版本 4.0 中添加： 属性值可以使用 生成器 表达式。多配置生成器 (Ninja（构建工具） Multi-Config) 除非使用生成器表达式，否则会将每个配置的子目录附加到指定的目录。

警告

Xcode 生成器会忽略此属性。
