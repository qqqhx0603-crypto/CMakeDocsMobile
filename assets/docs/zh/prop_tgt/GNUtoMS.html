GNUtoMS¶

将 GNU 导入库（.dll.a）转换为 MS 格式（.lib）。

当使用 GNU 工具在 Windows 上（MinGW/MSYS）链接一个导出符号的共享库或可执行文件，并且安装了 Visual Studio 时，将导入库（.dll.a）从 GNU 格式转换为 MS 格式（.lib）。这两个导入库都将由 install(TARGETS) 安装，并由 install(EXPORT) 和 export() 导出，以便使用 GNU 或 MS 兼容工具进行链接。

如果变量 CMAKE_GNUtoMS 在创建目标时设置，其值将用于初始化此属性。必须在启用语言的第一个命令（例如 project() 或 enable_language()）之前设置该变量。CMake 在使用 GNU 工具在 Windows 上配置时，会自动将该变量作为选项提供给用户。
