ANDROID_ARCH¶

3.4 版本新增。

在 使用 NVIDIA Nsight Tegra（NVIDIA Tegra 调试扩展） Visual Studio（集成开发环境） Edition（NVIDIA Tegra 的 Visual Studio（集成开发环境） 扩展） 为 Android 交叉编译 时，此属性设置 Android 目标架构。

这是一个字符串属性，可以设置为以下值之一：

  • armv7-a: "ARMv7-A (armv7-a)"

  • armv7-a-hard: "ARMv7-A，硬浮点 ABI (armv7-a)"

  • arm64-v8a: "ARMv8-A，64 位 (arm64-v8a)"

  • x86: "x86 (x86)"

  • x86_64: "x86_64 (x86_64)"

当创建目标时，如果设置了 CMAKE_ANDROID_ARCH 变量，则此属性将由该变量的值初始化。
