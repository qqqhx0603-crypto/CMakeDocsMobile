VS_DOTNET_REFERENCEPROP_<refname>_TAG_<tagname>¶

3.10 版本新增。

为 .NET 引用 <refname> 定义一个 XML 属性 <tagname>。

可以为通过目标属性 VS_DOTNET_REFERENCES、VS_DOTNET_REFERENCE_<refname> 定义的 .NET 引用以及通过 target_link_libraries() 建立的其他 C# 目标的工程引用设置引用属性。

此属性仅适用于 C# 目标和 Visual Studio（集成开发环境） 生成器 2010 及更高版本。
