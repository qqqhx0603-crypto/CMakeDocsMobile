XCODE_GENERATE_SCHEME¶

版本 3.15 新增。

如果启用，Xcode 生成器将生成方案（scheme）文件。这些文件对于从命令行调用分析（analyze）、归档（archive）、为测试构建（build-for-testing）和测试（test）操作非常有用。

如果该属性在创建目标时已设置，则它将由变量 CMAKE_XCODE_GENERATE_SCHEME 的值进行初始化。

以下目标属性会覆盖每个方案文件中“诊断”（Diagnostic）选项卡上对应设置的默认值。其中每一个属性在创建目标时都会由各自的 CMAKE_ 变量进行初始化。

  • XCODE_SCHEME_ADDRESS_SANITIZER

  • XCODE_SCHEME_ADDRESS_SANITIZER_USE_AFTER_RETURN

  • XCODE_SCHEME_DISABLE_MAIN_THREAD_CHECKER

  • XCODE_SCHEME_DYNAMIC_LIBRARY_LOADS

  • XCODE_SCHEME_DYNAMIC_LINKER_API_USAGE

  • XCODE_SCHEME_GUARD_MALLOC

  • XCODE_SCHEME_MAIN_THREAD_CHECKER_STOP

  • XCODE_SCHEME_MALLOC_GUARD_EDGES

  • XCODE_SCHEME_MALLOC_SCRIBBLE

  • XCODE_SCHEME_MALLOC_STACK

  • XCODE_SCHEME_THREAD_SANITIZER

  • XCODE_SCHEME_THREAD_SANITIZER_STOP

  • XCODE_SCHEME_UNDEFINED_BEHAVIOUR_SANITIZER

  • XCODE_SCHEME_UNDEFINED_BEHAVIOUR_SANITIZER_STOP

  • XCODE_SCHEME_ENABLE_GPU_API_VALIDATION

  • XCODE_SCHEME_ENABLE_GPU_SHADER_VALIDATION

  • XCODE_SCHEME_ZOMBIE_OBJECTS

以下目标属性将应用于“信息”（Info）、“参数”（Arguments）和“选项”（Options）选项卡

  • XCODE_SCHEME_ARGUMENTS

  • XCODE_SCHEME_DEBUG_AS_ROOT

  • XCODE_SCHEME_DEBUG_DOCUMENT_VERSIONING

  • XCODE_SCHEME_ENABLE_GPU_FRAME_CAPTURE_MODE

  • XCODE_SCHEME_ENVIRONMENT

  • XCODE_SCHEME_EXECUTABLE

  • XCODE_SCHEME_LAUNCH_CONFIGURATION

  • XCODE_SCHEME_LAUNCH_MODE

  • XCODE_SCHEME_TEST_CONFIGURATION

  • XCODE_SCHEME_LLDB_INIT_FILE

  • XCODE_SCHEME_WORKING_DIRECTORY
