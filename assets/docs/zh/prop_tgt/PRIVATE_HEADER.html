PRIVATE_HEADER¶

在 FRAMEWORK 共享库目标中指定私有头文件。

带有 FRAMEWORK 属性的共享库目标在 macOS、iOS 上生成框架，在其他平台上生成普通的共享库。可以将此属性设置为放置在框架文件夹内的 PrivateHeaders 目录中的头文件列表。在非 Apple 平台上，可以使用 PRIVATE_HEADER 选项与 install(TARGETS) 命令一起安装这些头文件。
