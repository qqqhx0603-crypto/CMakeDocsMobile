RUNTIME_OUTPUT_NAME¶

运行时目标文件的输出名称。

此属性指定运行时目标文件的基本名称。它会覆盖 OUTPUT_NAME 和 OUTPUT_NAME_<CONFIG> 属性。

另请参阅 RUNTIME_OUTPUT_NAME_<CONFIG> 目标属性。

上一主题

RUNTIME_OUTPUT_DIRECTORY_<CONFIG>

下一主题

RUNTIME_OUTPUT_NAME_<CONFIG>
