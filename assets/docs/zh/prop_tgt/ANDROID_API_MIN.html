ANDROID_API_MIN¶

版本 3.2 中新增。

设置 Android 最小 API 版本（例如 9）。版本号必须是正十进制整数。如果目标创建时设置了 CMAKE_ANDROID_API_MIN 变量，则此属性将由该变量的值初始化。使用此 API 版本的原生代码构建。
