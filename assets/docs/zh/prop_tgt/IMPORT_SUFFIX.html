IMPORT_SUFFIX¶

导入库名称后的后缀。

类似于目标属性 SUFFIX，但用于导入库（通常对应于一个 DLL）而不是常规库。 可以设置为覆盖导入库名称后缀（例如 .lib）的目标属性。
