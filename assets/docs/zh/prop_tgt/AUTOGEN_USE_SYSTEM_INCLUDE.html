AUTOGEN_USE_SYSTEM_INCLUDE¶

在 3.27 版本中新增。

AUTOGEN_USE_SYSTEM_INCLUDE 是一个布尔属性，可以设置为目标，以指示 autogen 目标包含目录应作为系统包含目录或普通包含目录添加到目标中。

如果未设置此属性，则 autogen 目标包含目录默认作为系统包含目录添加。请参阅策略 CMP0151。

有关将 CMake 与 Qt 结合使用的更多信息，请参阅 cmake-qt(7) 手册。

当创建目标时，如果设置了 CMAKE_AUTOGEN_USE_SYSTEM_INCLUDE 变量，则此属性将由该变量初始化。
