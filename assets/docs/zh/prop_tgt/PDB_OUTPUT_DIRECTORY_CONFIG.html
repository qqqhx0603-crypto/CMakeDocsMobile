PDB_OUTPUT_DIRECTORY_<CONFIG>¶

链接器为可执行文件或共享库目标生成时，每个配置的 MS 调试符号 .pdb 文件的输出目录。

这是 PDB_OUTPUT_DIRECTORY 的每个配置版本，但多配置生成器（Visual Studio（集成开发环境） 生成器，Xcode）不会将每个配置的子目录附加到指定的目录。当创建目标时，如果设置了该变量，此属性将由 CMAKE_PDB_OUTPUT_DIRECTORY_<CONFIG> 变量的值初始化。

3.12 版本新增：PDB_OUTPUT_DIRECTORY_<CONFIG> 的内容可以使用 生成器 表达式。

注意

此属性不适用于静态库目标，因为不会调用链接器来生成它们，因此它们没有包含调试符号的链接器生成的 .pdb 文件。

链接器生成的程序数据库文件由 /pdb 链接器标志指定，与编译器生成的程序数据库文件不同，后者由 /Fd 编译器标志指定。使用 COMPILE_PDB_OUTPUT_DIRECTORY_<CONFIG> 属性来指定后者。
