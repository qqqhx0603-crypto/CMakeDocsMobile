XCODE_ATTRIBUTE_<一个属性>¶

直接设置 Xcode 目标属性。

告诉 Xcode 生成器在生成的 Xcode 项目中将 <一个属性> 设置为给定值。在其他生成器上被忽略。

这提供了对生成的 Xcode 项目文件的低级控制。 它是指定 CMake 否则无法控制的设置的最后手段。 虽然这可以覆盖 CMake 通常自行生成的设置，但这样做会绕过 CMake 的项目模型，并可能导致问题。

请参阅 CMAKE_XCODE_ATTRIBUTE_<一个属性> 变量，以设置目录树中所有目标的属性。

XCODE_ATTRIBUTE_<一个属性> 的内容可以使用 $<...> 语法的“生成器表达式”。 请参阅 cmake-generator-expressions(7) 手册以获取可用的表达式。 请参阅 cmake-buildsystem(7) 手册，了解有关定义构建系统属性的更多信息。
