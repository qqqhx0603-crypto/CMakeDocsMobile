COMPATIBLE_INTERFACE_NUMBER_MIN¶

其链接接口中的最小值将被使用的属性。

COMPATIBLE_INTERFACE_NUMBER_MIN 属性可以包含一个该目标的属性列表，在生成时（generate time），当在所有已链接依赖项的 INTERFACE 变体属性中进行评估时，可以读取这些属性的最小值。例如，如果属性 FOO 出现在列表中，那么对于每个依赖项，其所有依赖关系中的 INTERFACE_FOO 属性内容将相互比较，并与依赖者中的 FOO 属性进行比较。在生成时读取 FOO 属性时，将返回其中的最小值。如果该属性未设置，则会被忽略。

请注意，对于每个依赖项，此属性中指定的属性集不得与任何其他 兼容接口属性 中指定的属性集相交。
