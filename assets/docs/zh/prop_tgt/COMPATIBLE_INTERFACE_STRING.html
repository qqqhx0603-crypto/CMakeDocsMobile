COMPATIBLE_INTERFACE_STRING¶

必须与其链接接口字符串兼容的属性

属性 COMPATIBLE_INTERFACE_STRING 可以包含此目标的属性列表，这些属性在所有已链接依赖项的 INTERFACE 属性变体中评估为字符串时必须相同。 例如，如果属性 FOO 出现在列表中，则对于每个依赖项，其所有依赖项中的 INTERFACE_FOO 属性内容必须相互相等，并且与依赖项中的 FOO 属性相等。 如果未设置该属性，则将其忽略。

请注意，对于每个依赖项，此属性中指定的属性集不得与任何其他 兼容接口属性 中指定的属性集相交。
