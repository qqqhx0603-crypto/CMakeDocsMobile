ANDROID_STL_TYPE¶

3.4 版本新增。

在 使用 NVIDIA Nsight Tegra（NVIDIA Tegra 调试扩展） Visual Studio（集成开发环境） Edition（NVIDIA Tegra 的 Visual Studio（集成开发环境） 扩展） 为 Android 交叉编译 时，此属性指定项目对 STL 支持的类型。这是一个字符串属性，可以设置为以下值之一：

none

无 C++ 支持

system

最小 C++，无 STL

gabi++_static

GAbi++ 静态

gabi++_shared

GAbi++ 共享

gnustl_static

GNU libstdc++ 静态

gnustl_shared

GNU libstdc++ 共享

stlport_static

STLport 静态

stlport_shared

STLport 共享

当创建目标时，如果设置了 CMAKE_ANDROID_STL_TYPE 变量，则此属性将由该变量的值初始化。
