FASTBUILD_CACHING¶

版本 4.2 中添加。

一个目标属性，用于控制生成的 fbuild.bff 文件中该目标是否启用缓存。

如果设置为 OFF，则 FASTBuild 生成器将禁用此目标的缓存功能。这对于已知缓存不可靠或不值得缓存的目标非常有用。

示例

set_property(TARGET my_target PROPERTY FASTBUILD_CACHING OFF)

代码说明

  • set_property(TARGET my_target PROPERTY FASTBUILD_CACHING OFF)：调用 `set_property()` 命令，设置目标、目录、源文件等对象的属性。

默认为 ON。
