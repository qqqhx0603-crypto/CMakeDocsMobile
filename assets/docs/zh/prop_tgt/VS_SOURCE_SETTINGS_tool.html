VS_SOURCE_SETTINGS_<tool>¶

在 3.18 版本中新增。

为使用 <tool> 的所有未构建文件设置任何项元数据。

接受 Key=Value 对的列表。告诉 Visual Studio 生成器 将 Key 设置为 Value 作为使用 <tool> 的所有未构建文件的项元数据。

例如

set_property(TARGET main PROPERTY VS_SOURCE_SETTINGS_FXCompile "Key=Value" "Key2=Value2")

将为使用 FXCompile 的所有未构建文件将 Key 设置为 Value 和 Key2 设置为 Value2。

生成器 表达式 受支持。

上一主题

VS_SOLUTION_DEPLOY

下一主题

VS_USE_DEBUG_LIBRARIES
