FOLDER¶

对于使用文件夹层次结构呈现目标的 IDE，此属性指定将目标放置在哪个文件夹下。要嵌套文件夹，请使用诸如 FOLDER 这样的值，例如 GUI/Dialogs，使用 / 字符分隔文件夹级别。没有 FOLDER 属性的目标将显示为顶级实体。具有相同 FOLDER 属性值的目标将作为兄弟节点出现在同一个文件夹中。

只有一些 CMake 生成器会尊重 FOLDER 属性（例如 Xcode 或任何 Visual Studio（集成开发环境） 生成器）。那些不尊重的生成器会简单地忽略它。

当创建目标时，如果设置了变量 CMAKE_FOLDER，则此属性将由该变量的值初始化。

全局属性 USE_FOLDERS 必须设置为 true，否则将忽略 FOLDER 属性。
