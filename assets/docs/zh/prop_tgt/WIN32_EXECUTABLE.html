WIN32_EXECUTABLE¶

在 Windows（微软桌面系统） 上使用 WinMain 入口点构建可执行文件。

当此属性设置为 true 时，在 Windows（微软桌面系统） 上链接的可执行文件将创建带有 WinMain() 入口点，而不是仅使用 main()。 这使其成为 GUI 可执行文件而不是控制台应用程序。 请参阅 CMAKE_MFC_FLAG 变量文档，以配置 WinMain 可执行文件对 Microsoft Foundation Classes (MFC) 的使用。 如果在创建目标时设置了该变量，则此属性将由 CMAKE_WIN32_EXECUTABLE 变量的值初始化。

此属性支持 生成器 表达式，除非目标是托管的（包含 C# 代码）。
