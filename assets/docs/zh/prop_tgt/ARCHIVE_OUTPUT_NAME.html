ARCHIVE_OUTPUT_NAME¶

ARCHIVE 目标文件的输出名称。

此属性指定归档目标文件的基本名称。它会覆盖 OUTPUT_NAME 和 OUTPUT_NAME_<CONFIG> 属性。

注意

在 macOS 上，对于链接器导入文件 (例如 .tbd 文件，详见 ENABLE_EXPORTS 属性) ，此属性将被忽略，当

  • FRAMEWORK 被设置时，因为框架布局无法更改。

  • Xcode 生成器被使用时，因为 Xcode 工具的限制和约束。

在这两种情况下，链接器导入文件将以与共享库相同的名称生成。

另请参阅 ARCHIVE_OUTPUT_NAME_<CONFIG> 目标属性。
