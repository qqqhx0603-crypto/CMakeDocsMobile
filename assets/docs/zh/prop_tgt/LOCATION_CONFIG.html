LOCATION_<CONFIG>¶

只读属性，提供目标在磁盘上的位置。

一个只读属性，指示目标的主要文件在配置 <CONFIG> 中在磁盘上的位置。该属性仅为库和可执行目标定义。导入的目标可能提供与导入项目不同的配置集。默认情况下，CMake 会查找完全匹配项，否则使用任意可用的配置。使用 MAP_IMPORTED_CONFIG_<CONFIG> 属性来显式映射导入的配置。

不要设置影响目标位置的属性，在读取此属性之后。这些包括名称与 (RUNTIME|LIBRARY|ARCHIVE)_OUTPUT_(NAME|DIRECTORY)(_<CONFIG>)?、(IMPLIB_)?(PREFIX|SUFFIX) 或 LINKER_LANGUAGE 匹配的属性。未能遵循此规则不会被诊断，并且会导致目标的的位置未定义。
