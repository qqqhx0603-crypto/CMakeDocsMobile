CUDA_OPTIX_COMPILATION¶

在 3.27 版本中新增。

将 CUDA 源代码编译为 .optixir 文件，而不是在 对象库 中编译为 .obj 文件。

例如

add_library(myoptix OBJECT a.cu b.cu)
set_property(TARGET myoptix PROPERTY CUDA_OPTIX_COMPILATION ON)

代码说明

  • add_library(myoptix OBJECT a.cu b.cu)：调用 add_library() 命令，定义一个库目标。

  • set_property(TARGET myoptix PROPERTY CUDA_OPTIX_COMPILATION ON)：调用 set_property() 命令，设置目标、目录、源文件等对象的属性。
