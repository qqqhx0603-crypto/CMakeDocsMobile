CXX_MODULE_SETS¶

版本 3.28 新增。

目标目标的 PRIVATE 和 PUBLIC C++ 模块集合的只读列表（即所有类型为 CXX_MODULES 的文件集合）。这些文件集合中列出的文件在 IDE 集成目的上被视为源文件。

可以使用 target_sources() 命令的 FILE_SET 选项，类型为 CXX_MODULES 来定义 C++ 模块集合。

另请参阅 CXX_MODULE_SET_<NAME>、CXX_MODULE_SET 和 INTERFACE_CXX_MODULE_SETS。
