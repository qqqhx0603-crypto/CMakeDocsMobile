XCODE_EMBED_<type>_REMOVE_HEADERS_ON_COPY¶

在 3.20 版本中添加。

仅由 Xcode 生成器使用的布尔属性。它指定是否从使用 XCODE_EMBED_<type> 属性嵌入的所有框架中移除头文件。

支持的 `<type>` 值包括：

FRAMEWORKS

如果未定义 XCODE_EMBED_FRAMEWORKS_REMOVE_HEADERS_ON_COPY 属性，则默认情况下在复制时不会移除头文件。

APP_EXTENSIONS

3.21 版本新增。

如果未定义 XCODE_EMBED_APP_EXTENSIONS_REMOVE_HEADERS_ON_COPY 属性，则默认情况下在复制时会移除头文件。

EXTENSIONKIT_EXTENSIONS

3.26 版新增。

如果未定义 XCODE_EMBED_APP_EXTENSIONS_REMOVE_HEADERS_ON_COPY 属性，则默认情况下在复制时会移除头文件。

PLUGINS

在版本 3.23 中添加。
