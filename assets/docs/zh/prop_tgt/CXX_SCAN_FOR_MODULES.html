CXX_SCAN_FOR_MODULES¶

版本 3.28 新增。

CXX_SCAN_FOR_MODULES 是一个布尔值，指定 CMake 是否会在目标中扫描 C++ 源代码以查找模块依赖项。 另请参阅 CXX_SCAN_FOR_MODULES，它用于设置每个源文件的设置，如果设置了该设置，则会覆盖目标范围内的设置。

当创建目标时，如果设置了变量 CMAKE_CXX_SCAN_FOR_MODULES，则此属性将由该变量的值初始化。

当此属性设置为 ON 或未设置时，CMake 将在构建时扫描目标的 CXX 源代码，并根据需要将模块依赖项信息添加到编译行中。 当此属性设置为 OFF 时，CMake 将不会在构建时扫描目标的 CXX 源代码。

请注意，仅当目标为 C++20 或更高版本启用时，才会执行扫描。 始终会对目标中属于文件集类型为 CXX_MODULES 的源代码进行模块扫描。
