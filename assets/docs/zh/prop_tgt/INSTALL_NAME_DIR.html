INSTALL_NAME_DIR¶

在 Apple 平台上安装目标时使用的目录名称。

INSTALL_NAME_DIR 是一个字符串，指定在 Apple 平台上安装目标时共享库的 "install_name" 字段的目录部分。如果未设置，则使用的默认目录由 MACOSX_RPATH 确定。如果设置了 BUILD_WITH_INSTALL_NAME_DIR 属性，则该属性将在构建树中被使用。策略 CMP0068 和 CMP0042 也相关。

当创建目标时，如果设置了变量 CMAKE_INSTALL_NAME_DIR，则此属性将由该变量的值初始化。

此属性支持 生成器 表达式。特别是，可以使用 $<INSTALL_PREFIX> 生成器表达式来相对于安装前缀设置目录。

在支持运行时路径（RPATH）的平台上，请参阅 INSTALL_RPATH 目标属性。在 Windows 上，TARGET_RUNTIME_DLLS 生成器表达式相关。
