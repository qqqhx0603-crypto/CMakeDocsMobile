INTERFACE_SOURCES¶

版本 3.1 中新增。

要编译进使用方目标（consuming targets）的接口源文件列表。

目标可以填充此属性，以发布供使用方目标编译的源文件。target_sources() 命令会使用传递给 PUBLIC 和 INTERFACE 关键字的值来填充此属性。项目也可以直接获取和设置此属性。

当使用 target_link_libraries() 指定目标依赖项时，CMake 会从所有目标依赖项中读取此属性，以确定使用方目标的源文件。

INTERFACE_SOURCES 的内容可以使用语法为 $<...> 的“生成器表达式”。有关可用表达式的信息，请参阅 cmake-generator-expressions(7) 手册。有关定义构建系统属性的更多信息，请参阅 cmake-buildsystem(7) 手册。
