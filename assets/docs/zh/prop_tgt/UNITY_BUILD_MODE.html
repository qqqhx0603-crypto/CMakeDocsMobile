UNITY_BUILD_MODE¶

在 3.18 版本中新增。

CMake 提供了不同的算法来选择哪些源文件组合在一起形成一个bucket。 此属性决定了选择方式，它具有以下可接受的值：

BATCH

当处于此模式时，CMake 确定哪些文件组合在一起。 UNITY_BUILD_BATCH_SIZE 属性控制每个 unity 源文件可以组合的源文件的上限。

用法示例

add_library(example_library
            source1.cxx
            source2.cxx
            source3.cxx
            source4.cxx)

set_target_properties(example_library PROPERTIES
                      UNITY_BUILD_MODE BATCH
                      UNITY_BUILD_BATCH_SIZE 2
                      )

代码说明

  • add_library(example_library source1.cxx source2.cxx source3.cxx source4.cxx) set_target_properties(example_library PROPERTIES UNITY_BUILD_MODE BATCH UNITY_BUILD_BATCH_SIZE 2 )：调用 `add_library()` 命令，定义一个库目标。

GROUP

当处于此模式时，每个目标都会显式指定如何分组源文件。 具有相同 UNITY_GROUP 值的每个源文件将被组合在一起。 任何没有此属性的源文件都将单独编译。 UNITY_BUILD_BATCH_SIZE 属性在使用此模式时将被忽略。

用法示例

add_library(example_library
            source1.cxx
            source2.cxx
            source3.cxx
            source4.cxx)

set_target_properties(example_library PROPERTIES
                      UNITY_BUILD_MODE GROUP
                      )

set_source_files_properties(source1.cxx source2.cxx source3.cxx
                            PROPERTIES UNITY_GROUP "bucket1"
                            )
set_source_files_properties(source4.cxx
                            PROPERTIES UNITY_GROUP "bucket2"
                            )

代码说明

  • add_library(example_library source1.cxx source2.cxx source3.cxx source4.cxx) set_target_properties(example_library PROPERTIES UNITY_BUILD_MODE GROUP ) set_source_files_properties(source1.cxx source2.cxx source3.cxx PROPERTIES UNITY_GROUP "bucket1" ) set_source_files_properties(source4.cxx PROPERTIES UNITY_GROUP "bucket2" )：调用 `add_library()` 命令，定义一个库目标。

如果没有显式指定 UNITY_BUILD_MODE，CMake 将默认设置为 BATCH。
