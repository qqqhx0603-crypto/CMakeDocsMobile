VS_DEBUGGER_ENVIRONMENT¶

3.13 版本新增。

为 Visual Studio（集成开发环境） C++ 目标设置本地调试器环境。 属性值可以使用 生成器 表达式。 这在 Visual Studio（集成开发环境） 项目文件中的 <LocalDebuggerEnvironment> 中定义。 如果在创建目标时设置了变量 CMAKE_VS_DEBUGGER_ENVIRONMENT，则此属性将由该变量的值初始化。

此属性仅适用于 Visual Studio（集成开发环境） 生成器；在其他生成器上将被忽略。
