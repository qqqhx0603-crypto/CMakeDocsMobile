HEADER_SETS¶

在版本 3.23 中添加。

目标 PRIVATE 和 PUBLIC 头文件集合的只读列表（即所有类型为 HEADERS 的文件集合）。这些文件集合中列出的文件在 IDE 集成目的上被视为源文件。这些文件还将其 HEADER_FILE_ONLY 属性设置为 TRUE。

可以使用 target_sources() 命令的 FILE_SET 选项，类型为 HEADERS 来定义头文件集合。

另请参阅 HEADER_SET_<NAME>、HEADER_SET 和 INTERFACE_HEADER_SETS。
