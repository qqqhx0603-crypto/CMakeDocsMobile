IMPORTED_GLOBAL¶

3.11 版本新增。

指示一个 IMPORTED target 是否全局可见。

此属性的布尔值对于使用 IMPORTED GLOBAL 选项创建的 add_executable() 或 add_library() 的目标为 True。对于在项目内部构建的目标，它始终为 False。

对于使用 IMPORTED 选项创建的 add_executable() 或 add_library() 但没有附加选项 GLOBAL 的目标，此属性也为 False。但是，将此类本地 IMPORTED 目标的此属性设置为 True 会将该目标提升到全局范围。 只能在创建该 IMPORTED 目标所在的相同目录中进行此提升。

注意

一旦导入目标被设置为全局，就无法更改回非全局状态。 因此，如果项目设置此属性，则它可能仅提供 True 值。 如果项目尝试将属性设置为非 True 值，即使该值为 False，CMake 也会发出错误。

注意

在将 IMPORTED target 从 LOCAL 提升到 GLOBAL 之前创建的本地 ALIAS target（别名目标）s，将保留其初始范围（请参阅 ALIAS_GLOBAL 目标属性）。
