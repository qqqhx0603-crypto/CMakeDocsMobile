VS_GLOBAL_PROJECT_TYPES¶

Visual Studio（集成开发环境） 项目类型。

可以设置为 Visual Studio（集成开发环境） 识别的一个或多个 UUID，以指示项目类型。此值会逐字复制到生成的项目文件中。例如，对于托管 C++ 单元测试项目

{3AC096D0-A1C2-E12C-1390-A8335801FDAB};{8BC9CEB8-8B4A-11D0-8D11-00A0C91BC942}

代码说明

  • {3AC096D0-A1C2-E12C-1390-A8335801FDAB};{8BC9CEB8-8B4A-11D0-8D11-00A0C91BC942}：这一行给出示例文本或输出，用来说明：{3AC096D0-A1C2-E12C-1390-A8335801FDAB};{8BC9CEB8-8B4A-11D0-8D11-00A0C91BC942}

UUID 以分号分隔。

VS_GLOBAL_KEYWORD

下一主题

VS_GLOBAL_ROOTNAMESPACE
