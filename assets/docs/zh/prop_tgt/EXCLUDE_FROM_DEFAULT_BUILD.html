EXCLUDE_FROM_DEFAULT_BUILD¶

从解决方案构建配置中排除目标。

此属性仅由 Visual Studio（集成开发环境） 生成器 使用。当设置为 TRUE 时，目标将在运行“构建解决方案”命令时从构建中排除。

此属性具有每个配置的版本： EXCLUDE_FROM_DEFAULT_BUILD_<CONFIG>。

注意

解决方案构建配置不考虑项目依赖关系。如果一个目标被排除，即使配置中包含依赖于它的另一个目标，它也不会被构建。此行为与 CMake 生成的 ALL_BUILD 目标和 EXCLUDE_FROM_ALL 属性不同。
