EXPORT_FIND_PACKAGE_NAME¶

注意

实验性。受 CMAKE_EXPERIMENTAL_EXPORT_PACKAGE_DEPENDENCIES 控制。

控制在导出 find_dependency() 调用时与依赖目标关联的包名称，用于 install(PACKAGE_INFO)、export(PACKAGE_INFO)、install(EXPORT) 或 export(EXPORT)。这可用于为由 CMake 构建和导出的包或由 FetchContent 提供的包分配包名称。

此属性由 CMAKE_EXPORT_FIND_PACKAGE_NAME 初始化。
