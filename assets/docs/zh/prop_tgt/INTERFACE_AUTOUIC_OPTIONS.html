INTERFACE_AUTOUIC_OPTIONS¶

新增于 3.0 版本。

传递给 uic 的接口选项列表。

目标可以填充此属性以发布在调用 uic 时所需选项。 消耗目标可以将条目添加到他们自己的 AUTOUIC_OPTIONS 属性中，例如 $<TARGET_PROPERTY:foo,INTERFACE_AUTOUIC_OPTIONS>，以使用目标 foo 接口中指定的 uic 选项。 target_link_libraries() 命令会自动执行此操作。

此属性支持生成器表达式。 请参阅 cmake-generator-expressions(7) 手册以获取可用的表达式。
