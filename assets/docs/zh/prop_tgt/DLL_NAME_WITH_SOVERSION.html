DLL_NAME_WITH_SOVERSION¶

在 3.27 版本中新增。

此属性控制是否将 SOVERSION 目标属性添加到为 Windows 平台生成的 DLL 文件名中，当 WIN32 变量被设置时选择该平台。

所列属性的值将作为 -<SOVERSION> 附加到共享库目标的运行时组件的基本名称。

请注意，如果使用 CMAKE_PLATFORM_NO_VERSIONED_SONAME 变量全局禁用了版本化的文件名，则设置此属性将不会产生任何影响。
