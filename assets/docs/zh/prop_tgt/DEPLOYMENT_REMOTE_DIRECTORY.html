DEPLOYMENT_REMOTE_DIRECTORY¶

3.6 版本新增。

设置由 Visual Studio（集成开发环境） 生成器 生成的 .vcproj 文件中的 WinCE 项目的 RemoteDirectory 在 DeploymentTool 中，以及 RemoteExecutable 在 DebuggerTool 中。当您想在远程 WinCE 设备上进行调试时，这很有用。例如

set_property(TARGET ${TARGET} PROPERTY
  DEPLOYMENT_REMOTE_DIRECTORY "\\FlashStorage")

代码说明

  • set_property(TARGET ${TARGET} PROPERTY DEPLOYMENT_REMOTE_DIRECTORY "\\FlashStorage")：调用 `set_property()` 命令，设置目标、目录、源文件等对象的属性。

生成

<DeploymentTool RemoteDirectory="\FlashStorage" ... />
<DebuggerTool RemoteExecutable="\FlashStorage\target_file" ... />

代码说明

  • <DeploymentTool RemoteDirectory="\FlashStorage" ... /> <DebuggerTool RemoteExecutable="\FlashStorage\target_file" ... />：这一行给出示例文本或输出，用来说明：<DeploymentTool RemoteDirectory="\FlashStorage" ... /> <DebuggerTool RemoteExecutable="\FlashStorage\target_file" ... />
