BUILD_WITH_INSTALL_RPATH¶

BUILD_WITH_INSTALL_RPATH 是一个布尔值，指定是否在构建树中将目标与 INSTALL_RPATH 链接。这优先于 SKIP_BUILD_RPATH，并避免在安装前重新链接的需要。

当创建目标时，如果设置了 CMAKE_BUILD_WITH_INSTALL_RPATH 变量，则此属性将由该变量的值初始化。

如果策略 CMP0068 不为 NEW，则此属性还控制在 macOS（苹果桌面系统） 构建树中使用 INSTALL_NAME_DIR。无论哪种方式，BUILD_WITH_INSTALL_NAME_DIR 目标属性优先。
