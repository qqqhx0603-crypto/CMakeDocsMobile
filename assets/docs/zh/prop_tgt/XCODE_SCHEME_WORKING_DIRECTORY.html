XCODE_SCHEME_WORKING_DIRECTORY¶

在 3.17 版本中添加。

指定生成的 Xcode scheme 中 *Run* 和 *Profile* 操作的 工作目录。如果值包含生成器表达式，则会对这些表达式进行求值。

此属性由变量 CMAKE_XCODE_SCHEME_WORKING_DIRECTORY 的值初始化，如果创建目标时设置了该变量。

请参阅 XCODE_GENERATE_SCHEME 目标属性文档，以查看所有与 Xcode 方案相关的属性。

另请参阅 DEBUGGER_WORKING_DIRECTORY。
