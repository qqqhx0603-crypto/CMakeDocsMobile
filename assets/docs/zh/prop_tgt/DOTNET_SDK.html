DOTNET_SDK¶

在版本 3.23 中添加。

指定 C# 项目的 .NET SDK。例如：Microsoft.NET.Sdk。

此属性告知 Visual Studio（集成开发环境） 生成器 (VS 2019 及更高版本) 使用指定的 SDK 生成 .NET SDK 样式的项目。此属性仅对这些生成器有意义，并且仅在 C# 目标中有效。即使它们是托管的（例如，使用 COMMON_LANGUAGE_RUNTIME），它也会被 C++ 项目忽略。

必须设置一个非空字符串才能生成 .NET SDK 样式的项目。CMake 不会对该属性的值进行任何验证。

可以使用 CMAKE_DOTNET_SDK 变量为所有目标初始化此属性。

注意

此版本的 CMake 中的 Visual Studio（集成开发环境） 生成器 尚未学会支持 add_custom_command() 在 .NET SDK 样式项目中。目前，将自定义命令附加到设置了 DOTNET_SDK 属性的目标是一个错误。
