ANDROID_API¶

版本 3.1 中新增。

在使用 使用 NVIDIA Nsight Tegra（NVIDIA Tegra 调试扩展） Visual Studio（集成开发环境） Edition（NVIDIA Tegra 的 Visual Studio（集成开发环境） 扩展） 为 Android 交叉编译 时，此属性设置 Android 目标 API 版本（例如 15）。版本号必须是正十进制整数。如果目标创建时设置了 CMAKE_ANDROID_API 变量，则此属性将由该变量的值初始化。

上一主题

ANDROID_ANT_ADDITIONAL_OPTIONS

下一主题

ANDROID_API_MIN
