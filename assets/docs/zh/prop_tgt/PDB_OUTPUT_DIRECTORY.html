PDB_OUTPUT_DIRECTORY¶

链接器为可执行文件或共享库目标生成的文件 .pdb 调试符号的输出目录。

此属性指定链接器将 MS 调试符号放置到的目录。

3.12 版本新增：属性值可以使用 generator 表达式。多配置生成器会将每个配置的子目录附加到指定的目录，除非使用了生成器表达式。

当创建目标时，如果设置了该变量，此属性将由 CMAKE_PDB_OUTPUT_DIRECTORY 变量的值初始化。

注意

此属性不适用于静态库目标，因为不会调用链接器来生成它们，因此它们没有包含调试符号的链接器生成的 .pdb 文件。

链接器生成的程序数据库文件由 /pdb 链接器标志指定，与由 /Fd 编译器标志指定的编译器生成的程序数据库文件不同。使用 COMPILE_PDB_OUTPUT_DIRECTORY 属性来指定后者。

上一主题

PDB_NAME_<CONFIG>

下一主题

PDB_OUTPUT_DIRECTORY_<CONFIG>
