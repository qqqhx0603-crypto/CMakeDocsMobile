ALIAS_GLOBAL¶

在 3.18 版本中新增。

只读属性，指示 ALIAS 目标 是否全局可见。

此属性的布尔值为 TRUE，表示指向使用 GLOBAL 选项创建的 IMPORTED 目标 的别名，这些选项用于 add_executable() 或 add_library()，否则为 FALSE。对于在项目内部构建的目标，该属性未定义。

注意

通过更改 IMPORTED_GLOBAL 目标属性的值，将 IMPORTED 目标 从 LOCAL 提升到 GLOBAL 作用域，不会改变本地别名的作用域。
