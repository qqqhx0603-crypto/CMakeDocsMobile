OUTPUT_NAME¶

目标文件的输出名称。

这设置了为可执行文件或库目标创建的输出文件的基本名称。如果未设置，则在生成期间默认使用逻辑目标名称。在配置期间默认情况下未设置该值。

OUTPUT_NAME 和下面列出的变体的内容可以使用 生成器 表达式。

另请参阅变体

  • OUTPUT_NAME_<CONFIG>

  • ARCHIVE_OUTPUT_NAME_<CONFIG>

  • ARCHIVE_OUTPUT_NAME

  • LIBRARY_OUTPUT_NAME_<CONFIG>

  • LIBRARY_OUTPUT_NAME

  • RUNTIME_OUTPUT_NAME_<CONFIG>
