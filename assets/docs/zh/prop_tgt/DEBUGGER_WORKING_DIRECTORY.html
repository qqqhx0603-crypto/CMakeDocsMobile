DEBUGGER_WORKING_DIRECTORY¶

4.0 版本新增。

为目标设置本地调试器工作目录，特别是调试器启动的进程。

属性值可以使用 生成器 表达式。当创建目标时，如果设置了变量 CMAKE_DEBUGGER_WORKING_DIRECTORY，则此属性将由其值初始化。

如果同时设置了 VS_DEBUGGER_WORKING_DIRECTORY 属性，在使用以下 Visual Studio（集成开发环境） 生成器 时，它将优先于 DEBUGGER_WORKING_DIRECTORY。

类似地，如果设置了 XCODE_SCHEME_WORKING_DIRECTORY，在使用 Xcode 生成器时，它将覆盖 DEBUGGER_WORKING_DIRECTORY。
