COMMON_LANGUAGE_RUNTIME¶

3.12 版本新增。

通过设置此目标属性，目标将被配置为使用 C++/CLI 支持进行构建。

Visual Studio（集成开发环境） 生成器 根据 COMMON_LANGUAGE_RUNTIME 目标属性的值定义 clr 参数

未设置 (默认)

原生 C++。

"" (已设置但为空)

使用 .NET Framework 的混合非托管/托管 C++。

netcore

3.26 版新增。

使用 .NET Core 的混合非托管/托管 C++。

这需要 VS 2019 的 v142 工具集或更高版本。

pure

托管 C++。

safe

托管 C++。

此属性仅在 Visual Studio（集成开发环境） 生成器 中针对 VS 2010 及更高版本进行评估。

要能够使用 VS 2017 及更高版本构建托管 C++ 目标，必须安装组件 C++/CLI 支持，这可能默认情况下未完成。

另请参阅 IMPORTED_COMMON_LANGUAGE_RUNTIME
