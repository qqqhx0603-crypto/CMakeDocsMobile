ISPC_HEADER_DIRECTORY¶

3.19 版本新增。

指定目标提供的 ISPC 头文件的相对输出目录。

如果目标包含 ISPC 源代码文件，则此项指定生成的文件头将放置在哪个目录中。相对路径相对于 CMAKE_CURRENT_BINARY_DIR 的值来处理。当此属性未设置时，头文件将被放置在生成器定义的构建目录中。如果变量 CMAKE_ISPC_HEADER_DIRECTORY 在创建目标时设置，其值将用于初始化此属性。
