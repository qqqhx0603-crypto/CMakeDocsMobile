<LANG>_COMPILER_LAUNCHER¶

3.4 版本新增。

此属性仅在 <LANG> 为 C、CXX、Fortran、HIP、ISPC、OBJC、OBJCXX 或 CUDA 时实现。

指定一个 分号分隔的列表，其中包含用于编译器启动工具的命令行。 Makefile 生成器 和 Ninja 生成器将运行此工具，并将编译器及其参数传递给该工具。 一些示例工具包括 distcc 和 ccache。

当创建目标时，如果设置了 CMAKE_<LANG>_COMPILER_LAUNCHER 变量，则此属性将由该变量的值初始化。

3.25 版本新增： 属性值可以使用 生成器 表达式。
