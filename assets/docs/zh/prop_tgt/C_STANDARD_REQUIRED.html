C_STANDARD_REQUIRED¶

版本 3.1 中新增。

布尔值，描述 C_STANDARD 属性的值是否为必需。

如果此属性设置为 ON，则 C_STANDARD 目标属性被视为必需。如果此属性设置为 OFF 或未设置，则 C_STANDARD 目标属性被视为可选，如果请求的标准不可用，则可能会“降级”到之前的标准。对于没有 C 标准级别的编译器，例如 Visual Studio 16.7（微软集成开发环境）（微软集成开发环境） 之前的 Microsoft Visual C++，此属性无效。

有关编译功能和支持的编译器列表，请参阅 cmake-compile-features(7) 手册。

当创建目标时，如果设置了 CMAKE_C_STANDARD_REQUIRED 变量，则此属性将由该变量的值初始化。
