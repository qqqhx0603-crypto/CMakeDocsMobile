ISPC_INSTRUCTION_SETS¶

3.19 版本新增。

要生成代码的指令集架构列表。

当创建目标时，如果设置了 CMAKE_ISPC_INSTRUCTION_SETS 变量，则此属性将由该变量的值初始化。

在为多个指令集生成代码时，必须使用 ISPC_INSTRUCTION_SETS 目标属性，以便 CMake 可以跟踪将生成哪些目标文件。

示例¶

set_property(TARGET tgt PROPERTY ISPC_INSTRUCTION_SETS avx2-i32x4 avx512skx-i32x835)

为 avx2 和 avx512skx 目标架构生成代码。
