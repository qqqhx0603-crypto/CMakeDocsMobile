VS_JUST_MY_CODE_DEBUGGING¶

版本 3.15 新增。

启用使用 Visual Studio（集成开发环境） 调试器的“仅我的代码”功能。

支持于 Visual Studio（集成开发环境） 生成器 (VS 2010 及更高版本), Makefile（构建脚本） 生成器 以及 Ninja（构建工具） 生成器。

当创建目标时，如果设置了 CMAKE_VS_JUST_MY_CODE_DEBUGGING 变量，则此属性会由该变量初始化。
