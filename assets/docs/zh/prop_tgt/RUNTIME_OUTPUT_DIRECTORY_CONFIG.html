RUNTIME_OUTPUT_DIRECTORY_<CONFIG>¶

用于 RUNTIME 目标文件的每配置输出目录。

这是 RUNTIME_OUTPUT_DIRECTORY 目标属性的每配置版本，但多配置生成器（Visual Studio（集成开发环境） 生成器，Xcode）不会将每配置子目录附加到指定的目录。当创建目标时，如果设置了该变量，此属性将由 CMAKE_RUNTIME_OUTPUT_DIRECTORY_<CONFIG> 变量的值初始化。

RUNTIME_OUTPUT_DIRECTORY_<CONFIG> 的内容可以使用 生成器 表达式。

上一主题

RUNTIME_OUTPUT_DIRECTORY

下一主题

RUNTIME_OUTPUT_NAME
