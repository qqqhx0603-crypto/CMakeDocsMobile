EXPORT_NO_SYSTEM¶

在 3.25 版本中新增。

此属性影响 install(EXPORT) 和 export() 命令在分别安装或导出目标时行为。当 EXPORT_NO_SYSTEM 设置为 true 时，这些命令会生成一个导入的目标，其 SYSTEM 属性设置为 false。

请参阅 NO_SYSTEM_FROM_IMPORTED 目标属性，以在使用包含目录的目标上设置此行为，而不是提供它们的那个目标。
