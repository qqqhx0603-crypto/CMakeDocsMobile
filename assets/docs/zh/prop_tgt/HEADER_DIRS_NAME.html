HEADER_DIRS_<NAME>¶

在版本 3.23 中添加。

以分号分隔的目标 <NAME> 头文件集合的基本目录列表，该集合的类型为 HEADERS。该属性支持 生成器 表达式。

此属性通常仅由 target_sources(FILE_SET) 设置，而不是直接操作。

请参阅 HEADER_DIRS 获取默认头文件集合的基本目录列表。请参阅 HEADER_SETS 获取所有头文件集合的文件集名称。
