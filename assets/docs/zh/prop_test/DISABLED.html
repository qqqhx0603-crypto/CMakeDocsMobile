已禁用¶

版本 3.9 中添加。

如果设置为 True，则测试将被跳过，其状态将为“未运行”。一个 DISABLED 测试不会计入测试总数，其完成状态将向 CDash 报告为 Disabled。

一个 DISABLED 测试不参与测试固定装置依赖关系解析。如果一个 DISABLED 测试在其 FIXTURES_REQUIRED 属性中定义了固定装置要求，则它不会导致为这些固定装置添加设置或清理测试到测试集中。

如果一个设置了 FIXTURES_SETUP 属性的测试被 DISABLED，则固定装置的行为将如同该设置测试通过，并且任何需要该固定装置的测试用例仍将运行。
