TIMEOUT¶

允许此测试运行的秒数。

如果设置了此属性，将限制测试的运行时间不超过指定的秒数。如果超过该时间，测试进程将被终止，ctest 将移至下一个测试。此设置优先于 CTEST_TEST_TIMEOUT。

显式的 0 值表示测试没有超时限制，除非需要遵守 ctest --stop-time 的规定。

另请参阅 TIMEOUT_AFTER_MATCH 和 TIMEOUT_SIGNAL_NAME。
