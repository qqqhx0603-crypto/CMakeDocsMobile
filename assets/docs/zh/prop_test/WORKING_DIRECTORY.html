WORKING_DIRECTORY¶

测试可执行文件将被调用的目录。

如果未设置此项，则测试将以与测试创建相关的二进制目录作为工作目录运行（即，CMAKE_CURRENT_BINARY_DIR，即调用 add_test() 的位置）。
