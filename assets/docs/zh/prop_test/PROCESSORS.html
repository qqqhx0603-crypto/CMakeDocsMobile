处理器¶

用于指定此测试需要多少个处理器槽。如果未设置，默认值为 1 个处理器。

表示此测试将需要多少个处理器。这通常用于 MPI 测试，应与 ctest_test() 的 PARALLEL_LEVEL 选项结合使用。

这还将用于在 ctest(1) 的命令行输出中的标签和子项目摘要中显示加权测试计时结果。测试运行的挂钟时间将乘以此属性，以更好地了解 CTest 为该测试分配了多少 CPU 资源。

另请参阅 PROCESSOR_AFFINITY 测试属性。
