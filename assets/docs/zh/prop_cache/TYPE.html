类型¶

GUI 中条目的控件类型。

缓存条目值始终是字符串，但 CMake GUI 会呈现控件来帮助用户设置值。GUI 将此属性用作提示来确定控件类型。有效的 TYPE 值是

BOOL

布尔 ON/OFF 值。

PATH

指向目录的路径。

FILEPATH

指向文件的路径。

STRING

通用字符串值。

INTERNAL

完全不在 GUI 中显示。

STATIC

由 CMake 管理的值，请勿更改。

UNINITIALIZED

类型尚未指定。

通常，缓存条目的 TYPE 应该由创建它的命令设置（set(), option(), find_library() 等）。
