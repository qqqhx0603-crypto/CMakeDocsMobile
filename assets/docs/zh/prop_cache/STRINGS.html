字符串¶

列举 GUI 选择的可能的 STRING 条目值。

对于类型为 STRING 的缓存条目，这将列举一组值。CMake GUI 可能会使用这些值来提供一个选择控件，而不是通用的字符串输入字段。这仅仅是为了方便。CMake 不会强制值与所列出的值之一匹配。
