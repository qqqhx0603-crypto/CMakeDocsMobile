mark_as_advanced¶

将 CMake 缓存变量标记为高级变量。

mark_as_advanced([CLEAR|FORCE] <var1> ...)

代码说明

  • mark_as_advanced([CLEAR|FORCE] <var1> ...)：调用 `mark_as_advanced()` 命令，具体行为由这一行传入的参数决定。

设置命名缓存变量的高级/非高级状态。

高级变量不会在任何 CMake GUI 中显示，除非开启了 show advanced 选项。在脚本模式下，高级/非高级状态没有影响。

如果给出了关键字 CLEAR，则高级变量将更改回非高级状态。如果给出了关键字 FORCE，则变量将被标记为高级。如果既没有指定 FORCE 也没有指定 CLEAR，则新值将被标记为高级，但如果变量已经具有高级/非高级状态，则不会更改它。

版本 3.17 中的更改：传递给此命令且尚未在缓存中的变量将被忽略。请参阅策略 CMP0102。
