continue¶

版本 3.2 中新增。

继续到包含的 foreach 或 while 循环的顶部。

continue()

代码说明

  • continue()：调用 continue() 命令，具体行为由这一行传入的参数决定。

continue() 命令允许 cmake 脚本中止当前 foreach() 或 while() 循环的剩余部分，并从下一次迭代的顶部开始。

另请参阅 break() 命令。
