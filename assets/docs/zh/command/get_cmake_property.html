get_cmake_property¶

获取 CMake 实例的全局属性。

get_cmake_property(<variable> <property>)

代码说明

  • get_cmake_property(<variable> <property>)：调用 get_cmake_property() 命令，具体行为由这一行传入的参数决定。

从 CMake 实例获取一个全局属性。 <property> 的值存储在指定的 <variable> 中。 如果未找到该属性，<variable> 将被设置为 NOTFOUND。 请参阅 cmake-properties(7) 手册以获取可用的属性。

除了全局属性之外，此命令（出于历史原因）还支持 VARIABLES 和 MACROS 目录属性。 它还支持一个特殊的 COMPONENTS 全局属性，该属性列出了传递给 install() 命令的组件。
