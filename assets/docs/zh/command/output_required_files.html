output_required_files¶

自版本 3.0 起已弃用。请参阅 CMake 策略 CMP0032。

近似的 C 预处理器依赖项扫描。

此命令仅存在是因为旧版本的 CMake 提供了它。CMake 使用更高级的扫描器自动处理预处理器依赖项扫描。

output_required_files(srcfile outputfile)

输出所有由指定 srcfile 所需的源文件列表。此列表写入 outputfile。这类似于为 srcfile 编写依赖项，但它会从 .h 文件跳转到 .cxx、.c 和 .cpp 文件（如果可能）。
