load_command¶

自版本 3.0 起已弃用。请参阅 CMake 策略 CMP0031。

将命令加载到正在运行的 CMake 中。

load_command(COMMAND_NAME <loc1> [loc2 ...])

代码说明

  • load_command(COMMAND_NAME <loc1> [loc2 ...])：调用 `load_command()` 命令，具体行为由这一行传入的参数决定。

CMake 会在给定的位置搜索名为 cmCOMMAND_NAME 的库。如果找到，则将其加载为模块，并将该命令添加到可用的 CMake 命令集中。通常，try_compile() 在此命令之前使用以编译该模块。如果命令成功加载，则名为 CMAKE_LOADED_COMMAND_<COMMAND_NAME> 的变量将被设置为加载的模块的完整路径。否则，该变量将不会被设置。
