set_directory_properties¶

设置当前目录和子目录的属性。

set_directory_properties(PROPERTIES <prop1> <value1> [<prop2> <value2>] ...)

以键值对的形式设置当前目录及其子目录的属性。

另请参阅 set_property(DIRECTORY) 命令。

请参阅 目录属性 以获取 CMake 了解的属性列表及其各自的文档，以了解每个属性的行为。
