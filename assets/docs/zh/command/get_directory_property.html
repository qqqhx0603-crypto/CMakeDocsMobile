get_directory_property¶

获取 DIRECTORY 范围的属性。

get_directory_property(<variable> [DIRECTORY <dir>] <prop-name>)

将目录范围的属性存储到名为 <variable> 的变量中。

DIRECTORY 参数指定从哪个目录获取属性值，而不是当前目录。相对路径被视为相对于当前源目录。CMake 必须已经知道该目录，要么是通过调用 add_subdirectory() 添加的，要么是顶级目录。

3.19 版本新增： <dir> 可以引用二进制目录。

如果未为指定目录范围定义该属性，则返回一个空字符串。对于 INHERITED 属性，如果未在指定目录范围内找到该属性，则搜索将链接到父范围，如 define_property() 命令中所述。

get_directory_property(<variable> [DIRECTORY <dir>]
                       DEFINITION <var-name>)

从目录获取变量定义。这种形式对于从另一个目录获取变量定义很有用。
