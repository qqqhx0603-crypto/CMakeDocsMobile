option¶

提供一个布尔选项，用户可以选择性地选择它。

option(<variable> "<help_text>" [value])

代码说明

  • option(<variable> "<help_text>" [value])：调用 option() 命令，声明一个布尔型配置选项。

如果没有提供初始 <value>，则布尔 OFF 是默认值。如果 <variable> 已经设置为普通变量或缓存变量，则该命令不执行任何操作（请参阅策略 CMP0077）。

在 CMake 项目模式下，会创建一个具有选项值的布尔缓存变量。在 CMake 脚本模式下，会设置一个具有选项值的布尔变量。

另请参阅¶

  • 使用 CMakeDependentOption 模块来指定依赖于其他选项值或一组条件的布尔选项。
