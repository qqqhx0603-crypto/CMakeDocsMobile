install_programs¶

自 3.0 版本弃用：请使用 install(PROGRAMS) 命令代替。

此命令已被 install() 命令取代。为了与旧的 CMake 代码兼容而提供。 FILES 形式直接被 install() 命令的 PROGRAMS 形式取代。正则表达式形式可以使用 file() 命令的 GLOB 形式更清晰地表达。

install_programs(<dir> file1 file2 [file3 ...])
install_programs(<dir> FILES file1 [file2 ...])

代码说明

  • install_programs(<dir> file1 file2 [file3 ...]) install_programs(<dir> FILES file1 [file2 ...])：调用 `install_programs()` 命令，具体行为由这一行传入的参数决定。

创建规则，将列出的程序安装到给定的目录中。使用 FILES 参数来保证命令的文件列表形式即使只有一个参数也会被使用。

install_programs(<dir> regexp)

代码说明

  • install_programs(<dir> regexp)：调用 `install_programs()` 命令，具体行为由这一行传入的参数决定。

在第二种形式中，当前源目录中与正则表达式匹配的任何程序都将被安装。

此命令用于安装不由 cmake 构建的程序，例如 shell（命令解释器） 脚本。请参阅 install() 命令的 TARGETS 形式，以创建 cmake 构建目标的安装规则。

目录 <dir> 相对于安装前缀，安装前缀存储在变量 CMAKE_INSTALL_PREFIX 中。
