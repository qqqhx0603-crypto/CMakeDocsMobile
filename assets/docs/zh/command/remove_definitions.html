remove_definitions¶

移除由 add_compile_definitions() 或 add_definitions() 添加的编译定义。

remove_definitions([<definitions>...])

代码说明

  • remove_definitions([<definitions>...])：调用 remove_definitions() 命令，具体行为由这一行传入的参数决定。

参数为

<definitions>...

零个或多个编译定义。

此命令也可用于移除由 add_definitions() 添加的任何标志，但它主要用于移除使用 -D 或 /D 传递的预处理器定义。

示例¶

在以下示例中，当前目录作用域的目标将仅具有 BAZ 和 QUUX 编译定义

add_compile_definitions(FOO BAR BAZ -DQUUX)

# ...

remove_definitions(-DFOO -DBAR)

代码说明

  • add_compile_definitions(FOO BAR BAZ -DQUUX)：调用 add_compile_definitions() 命令，为当前目录或目标补充预处理宏定义。

  • # ...：这是 CMake 注释。此处省略了与当前要点无关的代码。

  • remove_definitions(-DFOO -DBAR)：调用 remove_definitions() 命令，具体行为由这一行传入的参数决定。
