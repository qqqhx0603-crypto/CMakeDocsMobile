include_regular_expression¶

设置用于依赖项检查的正则表达式。

include_regular_expression(regex_match [regex_complain])

代码说明

  • include_regular_expression(regex_match [regex_complain])：调用 include_regular_expression() 命令，具体行为由这一行传入的参数决定。

设置在依赖项检查中使用的正则表达式。只有匹配 regex_match 的文件才会被追踪为依赖项。只有匹配 regex_complain 的文件在无法找到时会生成警告（标准头文件路径不会被搜索）。

对于 regex_match，默认值为 "^.*$"（匹配所有内容）。对于 regex_complain，默认值为 "^$"（仅匹配空字符串）。
