link_libraries¶

链接后续添加的所有目标。

link_libraries([item1 [item2 [...]]]
               [[debug|optimized|general] <item>] ...)

代码说明

  • link_libraries([item1 [item2 [...]]] [[debug|optimized|general] <item>] ...)：调用 `link_libraries()` 命令，具体行为由这一行传入的参数决定。

指定在当前目录或其子目录中，通过诸如 add_executable() 或 add_library() 等命令创建的任何目标时要使用的库或标志。 请参阅 target_link_libraries() 命令以了解参数的含义。

注意

尽可能应优先使用 target_link_libraries() 命令。 库依赖项会自动链接，因此很少需要对整个目录指定链接库。
