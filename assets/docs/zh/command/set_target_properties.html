set_target_properties¶

目标可以拥有影响其构建方式的属性。

set_target_properties(<targets> ...
                      PROPERTIES <prop1> <value1>
                      [<prop2> <value2>] ...)

代码说明

  • set_target_properties(<targets> ... PROPERTIES <prop1> <value1> [<prop2> <value2>] ...)：调用 `set_target_properties()` 命令，批量设置目标属性。

设置目标上的属性。该命令的语法是列出您想要更改的所有目标，然后提供您想要设置的值。您可以使用任何属性值对，稍后可以使用 get_property() 或 get_target_property() 命令提取它。

别名目标 不支持设置目标属性。
