write_file¶

自版本 3.0 起已弃用：请使用 file(WRITE) 命令代替。

write_file(filename "message to write"... [APPEND])

代码说明

  • write_file(filename "message to write"... [APPEND])：调用 `write_file()` 命令，具体行为由这一行传入的参数决定。

第一个参数是文件名，其余参数是要写入的消息。如果指定了参数 APPEND，则消息将被追加。

注意 1：file(WRITE) 和 file(APPEND) 执行的功能与此命令完全相同，但增加了更多功能。

注意 2：在使用 write_file 时，生成的文件不能用作 CMake 的输入（CONFIGURE_FILE、源文件...），因为它会导致无限循环。如果您想生成 CMake 的输入文件，请使用 configure_file()。
