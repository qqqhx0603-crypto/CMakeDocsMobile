block¶

在 3.25 版本中新增。

在一个专用的变量和/或策略作用域内评估一组命令。

block([SCOPE_FOR [POLICIES] [VARIABLES]] [PROPAGATE <var-name>...])
  <commands>
endblock()

代码说明

  • block([SCOPE_FOR [POLICIES] [VARIABLES]] [PROPAGATE <var-name>...])：调用 block() 命令，具体行为由这一行传入的参数决定。

  • <commands>：这一行继续补充当前命令或表达式的内容。

  • endblock()：这一行结束或切换上一段控制结构的分支。

block() 和对应的 endblock() 之间的所有命令都会被记录下来，而不会立即执行。一旦 endblock() 被评估，记录的命令列表就会在请求的作用域内执行，随后 block() 命令创建的作用域将被移除。

SCOPE_FOR

指定必须创建哪些作用域。

POLICIES

创建一个新的策略作用域。这等同于 cmake_policy(PUSH)，并在离开块作用域时自动执行 cmake_policy(POP)。

VARIABLES

创建一个新的变量作用域。

如果未指定 SCOPE_FOR，则等同于

block(SCOPE_FOR VARIABLES POLICIES)

代码说明

  • block(SCOPE_FOR VARIABLES POLICIES)：调用 block() 命令，具体行为由这一行传入的参数决定。

PROPAGATE

当 block() 命令创建变量作用域时，该选项用于在父作用域中设置或取消设置指定的变量。这等同于 set(PARENT_SCOPE) 或 unset(PARENT_SCOPE) 命令。

set(var1 "INIT1")
set(var2 "INIT2")
set(var3 "INIT3")

block(PROPAGATE var1 var2)
  set(var1 "VALUE1")
  unset(var2)
  set(var3 "VALUE3")
endblock()

# Now var1 holds VALUE1, var2 is unset, and var3 holds the initial value INIT3

代码说明

  • set(var1 "INIT1")：调用 set() 命令，把变量 var1 设为给定的值；变量说明：var1：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • set(var2 "INIT2")：调用 set() 命令，把变量 var2 设为给定的值；变量说明：var2：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • set(var3 "INIT3")：调用 set() 命令，把变量 var3 设为给定的值；变量说明：var3：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • block(PROPAGATE var1 var2)：调用 block() 命令，具体行为由这一行传入的参数决定。

  • set(var1 "VALUE1")：调用 set() 命令，把变量 var1 设为给定的值；变量说明：var1：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • unset(var2)：调用 unset() 命令，具体行为由这一行传入的参数决定；变量说明：var2：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • set(var3 "VALUE3")：调用 set() 命令，把变量 var3 设为给定的值；变量说明：var3：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • endblock()：这一行结束或切换上一段控制结构的分支。

  • # Now var1 holds VALUE1, var2 is unset, and var3 holds the initial value INIT3：这是 CMake 注释。这是代码作者保留的说明，描述当前步骤的目的、输入输出或使用限制；英文原文保留在代码中。

此选项仅允许在创建变量作用域时使用。在其他情况下将会报错。

当 block() 位于 foreach() 或 while() 命令内部时，可以在块内使用 break() 和 continue() 命令。

while(TRUE)
  block()
     ...
     # the break() command will terminate the while() command
     break()
  endblock()
endwhile()

代码说明

  • while(TRUE)：调用 while() 命令，开始 while 循环。

  • block()：调用 block() 命令，具体行为由这一行传入的参数决定。

  • ...：这一行继续补充当前命令或表达式的内容。

  • # the break() command will terminate the while() command：这是 CMake 注释。这是代码作者保留的说明，描述当前步骤的目的、输入输出或使用限制；英文原文保留在代码中。

  • break()：调用 break() 命令，具体行为由这一行传入的参数决定。

  • endblock()：这一行结束或切换上一段控制结构的分支。

  • endwhile()：这一行结束或切换上一段控制结构的分支。
