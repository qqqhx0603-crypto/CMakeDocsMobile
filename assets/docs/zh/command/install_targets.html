install_targets¶

自 3.0 版本弃用： 请使用 install(TARGETS) 命令代替。

此命令已被 install() 命令取代。为了兼容旧的 CMake 代码，仍然提供此命令。

install_targets(<dir> [RUNTIME_DIRECTORY dir] target target)

代码说明

  • install_targets(<dir> [RUNTIME_DIRECTORY dir] target target)：调用 install_targets() 命令，具体行为由这一行传入的参数决定。

创建规则，将列出的目标安装到给定的目录中。目录 <dir> 相对于安装前缀，安装前缀存储在变量 CMAKE_INSTALL_PREFIX 中。如果指定了 RUNTIME_DIRECTORY，那么在具有特殊运行时文件（Windows（微软桌面系统） DLL（动态链接库））的系统上，文件将被复制到该目录。
