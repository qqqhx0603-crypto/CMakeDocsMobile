subdir_depends¶

自版本 3.0 起已弃用。请参阅 CMake 策略 CMP0029。

什么也不做。

subdir_depends(subdir dep1 dep2 ...)

代码说明

  • subdir_depends(subdir dep1 dep2 ...)：调用 `subdir_depends()` 命令，具体行为由这一行传入的参数决定。

不执行任何操作。 此命令曾经帮助项目正确地排序并行构建。 此功能现在是自动的。
