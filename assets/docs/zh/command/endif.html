endif¶

结束 if 块中的命令列表。

endif([<condition>])

代码说明

  • endif([<condition>])：调用 endif() 命令，具体行为由这一行传入的参数决定。

请参阅 if() 命令。

可选的 <condition> 参数仅为向后兼容性而支持。如果使用，它必须与打开的 if 子句的参数完全相同。
