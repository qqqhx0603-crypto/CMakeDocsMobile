CMP0014¶

警告

此策略的 OLD 行为已在 CMake 4.0 版本中移除。此策略必须通过调用 cmake_minimum_required() 或 cmake_policy() 来设置为 NEW。

输入目录必须包含 CMakeLists.txt。

在 CMake 2.8 之前的版本中，会静默地忽略由 add_subdirectory() 或 subdirs() 引用目录中缺失的 CMakeLists.txt 文件，将其视为存在但为空。在 CMake 2.8.0 及更高版本中，此 cmake_policy() 决定了是否将这种情况视为错误。此策略的 OLD 行为是静默地忽略该问题。此策略的 NEW 行为是报告错误。

此策略是在 CMake 2.8.0 版本中引入的。在 CMake 4.0 版本移除之前，可以通过 cmake_policy() 或 cmake_minimum_required() 进行设置。如果未设置，CMake 会发出警告，并使用 OLD 行为。
