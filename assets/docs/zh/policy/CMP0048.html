CMP0048¶

警告

此策略的 OLD 行为已在 CMake 4.0 版本中移除。此策略必须通过调用 cmake_minimum_required() 或 cmake_policy() 来设置为 NEW。

project() 命令管理 VERSION 变量。

CMake 3.0 版本引入了 project() 命令的 VERSION 选项，用于指定项目版本和名称。为了保持 PROJECT_VERSION 及相关变量与 PROJECT_NAME 变量的一致性，当 project() 未提供 VERSION 时，必须将 VERSION 变量设为空字符串。然而，这可能会改变现有项目的行为，因为这些项目可能已经自行设置了 VERSION 变量，而 project() 现在可能会清除它们。此策略用于控制此类项目的兼容性行为。

此策略的 OLD 行为是不改动 VERSION 变量。此策略的 NEW 行为是按照 project() 命令的文档设置 VERSION。

此策略是在 CMake 版本 3.0 中引入的。在 CMake 版本 4.0 中移除之前，可以通过 cmake_policy() 或 cmake_minimum_required() 设置。如果未设置，CMake 会发出警告并使用 OLD 行为。
