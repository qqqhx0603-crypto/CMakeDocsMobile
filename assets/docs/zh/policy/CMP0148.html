CMP0148¶

在 3.27 版本中新增。

模块 FindPythonInterp 和 FindPythonLibs 已被移除。

这些模块自 CMake 3.12 起已被弃用。CMake 3.27 及更高版本更倾向于不提供这些模块。此策略为尚未迁移的项目提供兼容性。

使用模块 FindPythonInterp 和/或 FindPythonLibs 的项目应更新为使用其替代方案

  • FindPython3

  • FindPython2

  • FindPython

此策略的 OLD 行为是让 find_package(PythonInterp) 和 find_package(PythonLibs) 加载已弃用的模块。 NEW 行为是使模块的使用失败，就像它们不存在一样。

此策略在 CMake 版本 3.27 中引入。它可以通过 cmake_policy() 或 cmake_minimum_required() 来设置。如果未设置，CMake 会发出警告并使用 OLD 行为。

注意

策略的 OLD 行为根据定义已被弃用，并可能在未来版本的 CMake 中被移除。
