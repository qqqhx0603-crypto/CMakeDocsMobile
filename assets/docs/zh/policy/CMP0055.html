CMP0055¶

警告

此策略的 OLD 行为已在 CMake 4.0 版本中移除。此策略必须通过调用 cmake_minimum_required() 或 cmake_policy() 来设置为 NEW。

版本 3.2 中新增。

对 break() 命令进行严格检查。

CMake 3.1 及更早版本允许在循环上下文之外调用 break() 命令，并且忽略任何给定的参数。 这是一种未定义行为。

此策略的 OLD 行为是允许将 break() 放置在循环上下文之外，并忽略任何参数。 此策略的 NEW 行为是在发现放置错误的 break 或任何参数时发出错误。

此策略是在 CMake 3.2 版本中引入的。 在 CMake 4.0 版本中移除之前，可以通过 cmake_policy() 或 cmake_minimum_required() 进行设置。 如果未设置，CMake 会发出警告，并使用 OLD 行为。
