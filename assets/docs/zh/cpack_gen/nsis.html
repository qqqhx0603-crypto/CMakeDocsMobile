CPack NSIS 生成器¶

CPack Nullsoft 可脚本化安装系统 (NSIS) 生成器的特定选项。

3.22 版本变更: NSIS 生成器需要 NSIS 3.03 或更高版本。

CPack NSIS 生成器的专用变量¶

以下变量专门用于基于 Windows（微软桌面系统） Nullsoft 可脚本化安装系统构建的图形化安装程序。

CPACK_NSIS_INSTALL_ROOT¶

NSIS 安装程序向最终用户提供的默认安装目录位于此根目录下。提供给最终用户的完整目录为：${CPACK_NSIS_INSTALL_ROOT}/${CPACK_PACKAGE_INSTALL_DIRECTORY}

CPACK_NSIS_MUI_ICON¶

图标文件名。用作生成的安装程序主图标的 *.ico 文件的名称。

CPACK_NSIS_MUI_UNIICON¶

图标文件名。用作生成的卸载程序主图标的 *.ico 文件的名称。

CPACK_NSIS_INSTALLER_MUI_ICON_CODE¶

未记录。

CPACK_NSIS_MUI_WELCOMEFINISHPAGE_BITMAP¶

3.5 版本新增。

用作 NSIS MUI_WELCOMEFINISHPAGE_BITMAP 的位图文件名。

CPACK_NSIS_MUI_UNWELCOMEFINISHPAGE_BITMAP¶

3.5 版本新增。

用作 NSIS MUI_UNWELCOMEFINISHPAGE_BITMAP 的位图文件名。

CPACK_NSIS_EXTRA_PREINSTALL_COMMANDS¶

将添加到安装部分（Section）开头、在目标系统上安装树可用之前的额外 NSIS 命令。

CPACK_NSIS_EXTRA_INSTALL_COMMANDS¶

将添加到安装部分（Section）末尾、在目标系统上安装树可用之后的额外 NSIS 命令。

CPACK_NSIS_EXTRA_UNINSTALL_COMMANDS¶

将添加到卸载部分（Section）、在从目标系统删除安装树之前的额外 NSIS 命令。

CPACK_NSIS_COMPRESSOR¶

传递给 NSIS SetCompressor 命令的参数。

CPACK_NSIS_ENABLE_UNINSTALL_BEFORE_INSTALL¶

询问是否先卸载以前的版本。如果设置为 ON，安装程序将查找以前安装的版本；如果找到，将询问用户在继续安装之前是否将其卸载。

CPACK_NSIS_MODIFY_PATH¶

修改 PATH 环境变量的开关。如果设置为 ON，安装程序中将出现一个额外的页面，允许用户选择是否应将程序目录添加到系统 PATH 变量中。

CPACK_NSIS_DISPLAY_NAME¶

出现在 Windows（微软桌面系统） 控制面板 中 应用和功能 的显示名称字符串。

CPACK_NSIS_PACKAGE_NAME¶

显示在安装程序顶部的标题。

CPACK_NSIS_INSTALLED_ICON_NAME¶

包含安装程序图标的可执行文件的路径。

CPACK_NSIS_HELP_LINK¶

提供应用程序安装帮助的网站 URL。

CPACK_NSIS_URL_INFO_ABOUT¶

提供有关应用程序更多信息的网站 URL。

CPACK_NSIS_CONTACT¶

关于安装过程的问题和评论的联系信息。

CPACK_NSIS_<compName>_INSTALL_DIRECTORY¶

3.7 版本中新增。

指定组件 <compName> 的自定义安装目录，而不是 $INSTDIR。

CPACK_NSIS_CREATE_ICONS_EXTRA¶

用于创建 开始菜单 快捷方式的额外 NSIS 命令。

CPACK_NSIS_DELETE_ICONS_EXTRA¶

用于卸载 开始菜单 快捷方式的额外 NSIS 命令。

CPACK_NSIS_EXECUTABLES_DIRECTORY¶

创建 NSIS 开始菜单 链接时，默认认为它们位于 bin 目录，除非设置了此变量。例如，如果可执行文件在 exec 目录中，则将其设置为 exec。

CPACK_NSIS_MUI_FINISHPAGE_RUN¶

指定一个可执行文件，以便在 NSIS 安装程序的完成页面上添加运行选项。

CPACK_NSIS_MENU_LINKS¶

指定 [application] 菜单中的链接。这应包含一系列 link link name 对。链接可以是 URL 或相对于安装前缀的路径。例如：

set(CPACK_NSIS_MENU_LINKS
  "doc/cmake-@CMake_VERSION_MAJOR@.@CMake_VERSION_MINOR@/cmake.html"
  "CMake Help" "https://cmake.com.cn" "CMake Web Site")

代码说明

  • set(CPACK_NSIS_MENU_LINKS：这一行继续补充当前命令或表达式的内容；变量说明：CPACK_NSIS_MENU_LINKS：当前示例使用的变量，其值由前面的命令、调用参数或运行环境提供。

  • "doc/cmake-@CMake_VERSION_MAJOR@.@CMake_VERSION_MINOR@/cmake.html"：这一行继续补充当前命令或表达式的内容。

  • "CMake Help" "https://cmake.com.cn" "CMake Web Site")：这一行继续补充当前命令或表达式的内容。

CPACK_NSIS_UNINSTALL_NAME¶

在 3.17 版本中添加。

指定用于卸载该版本的程序名称。默认为 Uninstall。

CPACK_NSIS_WELCOME_TITLE¶

在 3.17 版本中添加。

在欢迎页面顶部显示的标题。

CPACK_NSIS_WELCOME_TITLE_3LINES¶

在 3.17 版本中添加。

将欢迎页面上的标题显示为 3 行而不是 2 行。

CPACK_NSIS_FINISH_TITLE¶

在 3.17 版本中添加。

在完成页面顶部显示的标题。

CPACK_NSIS_FINISH_TITLE_3LINES¶

在 3.17 版本中添加。

将完成页面上的标题显示为 3 行而不是 2 行。

CPACK_NSIS_MUI_HEADERIMAGE¶

在 3.17 版本中添加。

显示在安装程序页面标题栏上的图像。

CPACK_NSIS_MANIFEST_DPI_AWARE¶

在 3.18 版本中新增。

如果设置，声明安装程序是支持 DPI 感知的。

CPACK_NSIS_BRANDING_TEXT¶

在 3.20 版本中添加。

如果设置，更新安装窗口底部的文本。要将字符串设置为空白，请使用空格 (" ")。

CPACK_NSIS_BRANDING_TEXT_TRIM_POSITION¶

在 3.20 版本中添加。

如果设置，将控件的大小调整为品牌文本字符串的大小。此变量允许的值为 LEFT、CENTER 或 RIGHT。如果未指定，默认行为是 LEFT。

CPACK_NSIS_EXECUTABLE¶

3.21 版本新增。

如果设置，指定 NSIS 可执行文件的名称。默认为 makensis。

CPACK_NSIS_IGNORE_LICENSE_PAGE¶

3.22 版本新增。

如果设置，在安装过程中不显示包含许可证的页面。

CPACK_NSIS_EXECUTABLE_PRE_ARGUMENTS¶

在 3.25 版本中新增。

此变量是一个分号分隔的列表，包含要在执行 nsis 脚本前预置的参数。如果参数不以 / 或 - 开头，它会自动为相应参数添加一个。将要运行的命令是

makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...

代码说明

  • makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...：这一行给出示例文本或输出，用来说明：makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...

其中 <preArgs>... 由 CPACK_NSIS_EXECUTABLE_PRE_ARGUMENTS 构成，<postArgs>... 由 CPACK_NSIS_EXECUTABLE_POST_ARGUMENTS 构成。

CPACK_NSIS_EXECUTABLE_POST_ARGUMENTS¶

在 3.25 版本中新增。

此变量是一个分号分隔的列表，包含要在执行 nsis 脚本后追加的参数。如果参数不以 / 或 - 开头，它会自动为相应参数添加一个。将要运行的命令是

makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...

代码说明

  • makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...：这一行给出示例文本或输出，用来说明：makensis.exe <preArgs>... "nsisFileName.nsi" <postArgs>...

其中 <preArgs>... 由 CPACK_NSIS_EXECUTABLE_PRE_ARGUMENTS 构成，<postArgs>... 由 CPACK_NSIS_EXECUTABLE_POST_ARGUMENTS 构成。

CPACK_NSIS_CRC_CHECK¶

版本 4.2 中添加。

指定安装程序在允许安装之前是否对自身执行 CRC 校验。此变量允许的值为 on、off 和 force。如果未指定，默认行为是 on。
