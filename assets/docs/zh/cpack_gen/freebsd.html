CPack FreeBSD 生成器¶

3.10 版本新增。

内置（二进制）CPack FreeBSD (pkg) 生成器（仅限 Unix）

影响 CPack FreeBSD (pkg) 生成器的变量¶

  • 3.18 版本新增： CPACK_ARCHIVE_THREADS

CPack FreeBSD (pkg) 生成器专用变量¶

CPack FreeBSD 生成器可用于创建 pkg(8) 软件包 —— 这些软件包可用于 FreeBSD、DragonflyBSD、NetBSD、OpenBSD，也可以根据所安装的软件包管理工具用于 Linux 或 OSX —— 它使用 CPack。

CPack FreeBSD 生成器是一个 CPack 生成器，并使用 CPack 所使用的 CPACK_XXX 变量。它尝试重用可能已经为 CPack DEB 生成器指定的 Debian 软件包信息。如果 Debian 未指定，它也会尝试重用 RPM 包装信息。

CPack FreeBSD 生成器应在任何安装了 libpkg 的主机上工作。它生成的软件包特定于主机架构和 ABI。

CPack FreeBSD 生成器通过 CPACK_FREEBSD_XXX 变量设置软件包元数据。与 CPack Deb 生成器不同，CPack FreeBSD 生成器不支持组件化软件包；它将所有通过 CMake 创建的软件制品打包为一个单一的软件包。

所有变量都可以在 CPackConfig 文件或 CMakeLists.txt 中专门为 FreeBSD 包装进行设置，但大多数变量都有使用通用设置（例如 CMAKE_PROJECT_NAME）或在合理时使用 Debian 特定变量（例如上游项目主页通常不会因包装风格而改变）的默认值。当没有可回退的 Debian 信息，但 RPM 包装拥有该信息时，会回退到 RPM 信息（例如软件包许可证）。

CPACK_FREEBSD_PACKAGE_NAME¶

设置软件包名称（在软件包清单中，也会影响输出文件名）。

强制项:

是

默认值：

  • CPACK_PACKAGE_NAME（始终由 CPack 本身根据 CMAKE_PROJECT_NAME 设置）。

CPACK_FREEBSD_PACKAGE_COMMENT¶

设置软件包注释。这是 pkg(8) 在标准 “pkg info” 输出中显示的简短描述。

强制项:

是

默认值：

  • CPACK_PACKAGE_DESCRIPTION_SUMMARY（如果未明确设置，始终由 CPack 本身设置）。

CPACK_FREEBSD_PACKAGE_DESCRIPTION¶

设置软件包描述。这是软件包的详细描述，在使用 “pkg info” 并以特定软件包作为参数时显示。

强制项:

是

默认值：

  • CPACK_DEBIAN_PACKAGE_DESCRIPTION（如果已为 Debian 包装设置，则用作回退）。

  • CPACK_PACKAGE_DESCRIPTION_SUMMARY（如果未明确设置，始终由 CPack 本身设置）。

  • PROJECT_DESCRIPTION（可以使用 project() 的 DESCRIPTION 参数设置）。

CPACK_FREEBSD_PACKAGE_WWW¶

该软件包的网站 URL，最好是（如果适用）可以获取原始源代码以及可以找到任何额外上游文档或信息的站点。

强制项:

是

默认值：

  • CPACK_PACKAGE_HOMEPAGE_URL，或者如果未设置，

  • CPACK_DEBIAN_PACKAGE_HOMEPAGE（如果已为 Debian 包装设置，则用作回退）。

3.12 版本新增： CPACK_PACKAGE_HOMEPAGE_URL 变量。

CPACK_FREEBSD_PACKAGE_LICENSE¶

适用于此软件包的许可证。必须是 pkg 识别为有效许可证标识符的一个或多个许可证标识符（例如 “GPLv2”）。

强制项:

是

默认值：

  • CPACK_RPM_PACKAGE_LICENSE

CPACK_FREEBSD_PACKAGE_LICENSE_LOGIC¶

此变量仅在有多个许可证时才有意义。默认值为 “single”，仅适用于单个许可证。其他可接受的值由 pkg 决定 —— 即 “dual” 或 “multi” —— 分别表示许可证的选择（OR）或同时（AND）应用。

强制项:

否

默认值：

single

CPACK_FREEBSD_PACKAGE_MAINTAINER¶

该软件包的 FreeBSD 维护者（例如 kde@freebsd.org）。

强制项:

是

默认值：

none

CPACK_FREEBSD_PACKAGE_ORIGIN¶

该软件包的来源（ports 标签）；对于通过 CPack 在 ports 系统之外构建的软件包，其重要性较低。默认值作为权宜之计，将软件包置于 misc/ 下。

强制项:

是

默认值：

misc/<软件包名称>

CPACK_FREEBSD_PACKAGE_CATEGORIES¶

该软件包所属的 ports 分类（如果它是从 ports 构建的）。如果未设置，则根据软件包来源确定单个分类。

强制项:

是

默认值：

派生自 ORIGIN

CPACK_FREEBSD_PACKAGE_DEPS¶

应添加为软件包依赖项的来源列表。它们的形式为 <分类>/<软件包名称>，例如 x11/libkonq。无需提供版本信息（这不包含在清单中）。

强制项:

否

默认值：

empty
